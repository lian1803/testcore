## 3. 독창적 채널 전략 (이 셀러에만 통하는 것)

### 채널 A: 기존 팔로워 1,041명 "첫 구매 이벤트" 집중 공략

**왜 이 채널인가?**
- 일반 체험단은 외부 유입이지만, 이 셀러는 이미 팔로워 확보 = 광고비 0원으로 초기 리뷰 확보 가능
- 식품/밀키트는 첫 구매 장벽 낮음(시식 개념) = 할인 쿠폰으로 즉시 전환 가능

**구체적 실행 계획:**
1. **타겟:** 팔로워 1,041명 전체
2. **메시지:** 스마트스토어 알림톡 발송 — "팔로워 감사 이벤트: 첫 구매 50% 할인 + 리뷰 작성 시 다음 구매 30% 추가 할인"
3. **목표:** 1,041명 중 10% 전환(추정: 식품 카테고리 쿠폰 이벤트 평균 전환율 8-15%) = 104명 구매 → 리뷰 작성 유도 50% = **52개 리뷰 확보**
4. **비용:** 할인 원가만 (광고비 0원)
5. **실행 주체:** 박시우(이벤트 배너 제작) + 이도현(쿠폰 설정 및 발송)

### 채널 B: 네이버 블로그 "밀키트 비교 콘텐츠" SEO 공략

**왜 이 채널인가?**
- "쭈꾸미볶음 추천", "닭갈비 밀키트 비교"는 구매 의도 높은 롱테일 키워드 = 검색 유입 전환율 5-10%(추정: 네이버 블로그 식품 후기 평균)
- 리뷰 0개 스토어는 네이버 쇼핑 검색 진입 불가 → 블로그 우회 유입으로 초기 트래픽 확보

**구체적 실행 계획:**
1. **타겟 키워드:** "쭈꾸미볶음 맛집 vs 밀키트 비교", "닭갈비 밀키트 추천 순위"
2. **콘텐츠 형식:** 블로그 포스팅 3개 (각 2,000자 이상, 이미지 10장 이상)
   - 포스팅 1: "쭈꾸미볶음 밀키트 5종 직접 먹어본 솔직 후기"
   - 포스팅 2: "닭갈비 밀키트 추천 — 양념 맛 비교 실험"
   - 포스팅 3: "1인 가구 밀키트 best 3 — 조리 시간 5분 이내"
3. **SEO 전략:** 제목에 키워드 포함 + 본문 내 키워드 밀도 2-3% + 이미지 alt 태그 설정
4. **목표:** 3개월 내 각 포스팅 월 조회수 500회 달성(추정: 롱테일 키워드 초기 진입 시 월 300-800회) → 총 1,500회 × 클릭률 10%(추정) × 전환율 5%(추정) = **월 7-8건 구매**
5. **실행 주체:** 박시우(블로그 포스팅 작성 및 이미지 제작)
