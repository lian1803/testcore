## 주의사항

### 1. 원가 산출 정확성 검증 필요
- 이도현이 제출한 견적서에서 **스탠다드 원가가 77만원 → 103만원으로 정정**되었으나, 최종 청구가 120만원 → 170만원 조정이 필요합니다.
- **프리미엄 원가도 156만원 → 178만원 재산출** 필요. 청구가 250만원 → 280만원 조정 권장.
- **액션:** 이도현에게 전체 패키지 견적서 재검토 지시 (마진율 35-40% 유지 확인)

### 2. 리뷰 확보 전략 리스크
- 체험단 30건 진행 시 **네이버 리뷰 정책 위반 가능성** 존재 (일시적 대량 리뷰 = 부정 리뷰 의심)
- **액션:** 체험단 진행 시 **2주 간격 분산 진행** 필요 (1차 15건 → 2주 후 2차 15건)

### 3. 3개 셀러 모두 리뷰 0개 = 동일한 페인포인트
- 동일한 솔루션 제안으로 차별화 부족 우려
- **액션:** 윤채원에게 셀러별 맞춤 전략 추가 요청 (sellerlivepick = 밀키트 재구매 전략 / hr-medical = 캠핑 시즌 타겟팅 / carminespace = 고가 제품 신뢰도 전략)

### 4. 발송 시간 조정 필요
- 한소율이 제안한 "평일 오전 10-11시"는 3개 셀러 모두 동일 시간대 발송 = 응대 가능 리소스 부족
- **액션:** 발송 시간 분산 (10:30 / 11:00 / 14:00)

### 5. A/B 테스트 일정 불명확
- 강하린이 제시한 "W2 상세페이지 헤드라인 A/B 테스트 14일 진행"은 **계약 후 실행 단계**인데, 현재는 **계약 전 리드 단계**
- **액션:** 강하린에게 계약 전/후 타임라인 명확히 구분 요청
