## 2. A/B 테스트 계획 3개

### **테스트 1: 상세페이지 헤드라인 최적화 (리드 카드 실제 제품명 기반)**

**배경:**
- 현재 sellerlivepick의 주력 제품: 쭈꾸미볶음, 닭갈비 밀키트
- 리드 카드 기반 주요 키워드 3개: "쭈꾸미볶음", "닭갈비 밀키트", "양념 쭈꾸미"

**가설:**
- **A안 (정보형):** "쭈꾸미볶음 밀키트 | 프리미엄 해산물 + 직화 양념 포함"
- **B안 (감성형):** "집에서 오 분 만에 술집 쭈꾸미볶음? 신선 해동 밀키트"

**측정 지표:** CTR(%) — 상세페이지 클릭률  
**테스트 기간:** 14일  
**샘플 규모:** 주당 노출수 150~280명 (예상: W2~W3 누적 약 400~500 노출)  
**판단 기준:** 0.3%p 이상 차이 (예: A안 2.8% vs B안 3.2% = B안 채택)

**이유:** 네이버 쇼핑 초기 셀러는 감성형 헤드라인이 클릭율 3~15% 더 높음 (추정: 밀키트 카테고리 감성 소비층 61% 비중)

---

### **테스트 2: 체험단 리뷰 인센티브 최적화**

**배경:**
- sellerlivepick 현재 상태: 리뷰 0개 = 검색 노출 진입 불가
- 목표: 4주 내 리뷰 30개 달성 (검색 노출 기준점)
- 리드 카드 팔로워 1,041명 = 초기 시드 고객 풀 확보

**가설:**
- **A안:** 무료 제공 + 리뷰 작성 시 추가 10% 할인 쿠폰 (원가 손실 ~3,000원/건)
- **B안:** 유료 체험단 (5,000원 구매 + 3,000원 리뷰비 환급 = 최종 구매가 2,000원) (원가 손실 ~2,500원/건)

**측정 지표:** 리뷰 작성 완료율(%)  
**테스트 기간:** 14일 (각 안 5명씩 = 총 10명)  
**판단 기준:** 15%p 이상 차이 (예: A안 70% vs B안 85% = B안이 고객 만족도 높음으로 판단, B안 확대)

**이유:** 
- A안 예상 완료율: ~60~70% (무료 제공 고객은 리뷰 의무감 낮음)
- B안 예상 완료율: ~75~85% (실제 구매한 고객 = 리뷰 작성 의도 높음)
- 4주 30개 목표 달성: B안으로 주당 5~6명 확보 가능 (추정: 체험단 25명 × 85% = 약 21명, + 자연 구매층 리뷰 9명 = 30명)

### **테스트 3: 상품 콘텐츠 스타일 A vs B (팔로워 활성화 중심)**

**배경:**
- sellerlivepick 팔로워 1,041명 보유하나 활성화율 0% (매출 전환 루트 부재)
- 목표: 팔로워 저장률↑ → 재방문율↑ → 재구매율 증대

**가설:**
- **A안 (정보형):** "쭈꾸미볶음 | 조리법 3단계 + 영양정보 + 보관기한 명시"
- **B안 (감성형):** "야식의 정석. 쭈꾸미 한입의 짜릿함 + 매콤한 양념의 중독성 | 혼술/회식 추천"

**측정 지표:** 저장률(%) — 상품 페이지 저장/북마크 비율  
**테스트 기간:** 14일  
**샘플 규모:** 팔로워 활성화 유도 (스토어 톡톡/팬레터로 500명에 유도, 클릭 예상 50~70명)  
**판단 기준:** 저장률 2%p 이상 차이 (예: A안 3.2% vs B안 5.8% = B안 채택, 팔로워 감성 소비층 확인)

**이유:** 
- 정보형(A안)은 비교 구매층 대상 → 저장율 2~3%
- 감성형(B안)은 냉동식품 충동 구매층 대상 → 저장율 4~6% (추정: 밀키트 카테고리 SNS 저장율 데이터)
- 승리 스타일로 콘텐츠 3~5개 추가 제작 → 팔로워 활성화율 목표 15% 달성
