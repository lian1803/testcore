# team/ — 팀이 만든 결과물/프로젝트

이 폴더는 **팀의 작업 산출물**이다. 실행 코드가 아니다.

## 들어가는 것
- `CLAUDE.md`, `PRD.md`, `런칭준비.md` — 프로젝트 기획/지시서
- 웹 코드, SaaS 앱 소스 — 팀이 만드는 실제 제품
- 디자인 파일, PDF, 문서

## 들어가지 않는 것
- Python 에이전트 실행 로직 → `company/teams/` 에
- `run_*.py` 스크립트 → `company/` 루트에

## 예시
- `[진행중] AI_세금신고_도우미/` — 활성 프로젝트
- `LLM-비용-서킷브레이커팀/` — 완성된 SaaS
- `디자인팀/` — 디자인팀 산출물

## 비활성 프로젝트 표기
- `[진행중] ...` — 현재 활성
- `[중단] ...` — 중단 (daily_auto 스킵)
- `[나중에] ...` — 보류 (daily_auto 스킵)
