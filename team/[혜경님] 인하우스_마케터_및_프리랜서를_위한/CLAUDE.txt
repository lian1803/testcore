# Lian Dash — Claude Code 구현 지시서 v1.0

## 시장 리서치 (CDO/마케팅팀 참고용)

### 타겟 사용자
- **누구:** 25~40대 디지털 마케팅 경력 2~7년의 인하우스 마케터(중소기업 1~5인 팀) 및 프리랜서 마케터(월 3~10개 클라이언트 관리)
- **행동 패턴:** 하루 평균 7개 이상 툴 사용(GA4, 메타 비즈니스, 네이버SA, 카카오모먼트 각각 별도 접속). 마케터 60%가 AI 툴 매일 사용 중이나 멀티플랫폼 데이터 통합에 어려움. 디지털 마케팅 커뮤니티(페이스북 그룹, 오픈카톡, 링크드인) 활발히 활동. 유튜브·블로그로 마케팅 트렌드 학습.
- **핵심 Pain:**
  1. **데이터 분산 피로** — GA4·메타·네이버SA·카카오 각각 별도 확인으로 하루 1~2시간 낭비. 마케터 81%가 AI 도입 후에도 ROI 분석 지연 경험
  2. **한국 플랫폼 공백** — 글로벌 통합 툴(Databox, Whatagraph 등)이 네이버SA·카카오모먼트 미지원. 국내 마케터는 사실상 수동 취합
  3. **AI 인사이트 부재** — 데이터는 있으나 "이번 주 뭘 고쳐야 하는지" 자동으로 알려주는 툴 없음. 수동 분석 의존으로 마케팅 ROI 10~20% 개선 여지 방치

### 시장 트렌드 (서윤·태호 종합)
- **AI-Native SaaS 폭증** — 마케팅 기술 AI 시장 CAGR 38%, 2025년 $470억 → 2028년 $1,070억. "AI가 분석해주는" 툴로 수요 이동 중
- **GA4 피로감** — "GA4 너무 복잡하다" 불만 급증, 대안 툴 탐색 수요 증가. 진입장벽 낮추는 계기
- **멀티플랫폼 통합 수요** — 마케터 73%가 3개 이상 채널 사용, 데이터 사일로가 업계 최대 고통
- **한국 로컬 플랫폼 Blind Spot** — Mixpanel·Heap 등 글로벌 툴이 네이버SA·카카오 미지원. 국내 MSP 광고주 대상 실질적 공백
- **클라이언트 워크스페이스 표준화** — 프리랜서·에이전시 시장에서 클라이언트 격리 SaaS 구조가 필수 기능으로 자리잡는 중
- **액션 가능한 인사이트** — 단순 데이터 연결이 아닌 "이번 주 광고 최적화점 3개" 형식의 구체적 제안이 차별점

### 검증 결과 (하은)
- ✅ **검증된 수치:** 글로벌 SaaS 시장 2026년 $3,000억~$3,150억(CAGR 18~19%), AI 마케팅 기술 CAGR 38% — 복수 출처 교차 확인
- ✅ **검증된 수치:** 마케터 60% AI 툴 매일 사용, 마케터 73% 3개 이상 채널 운영
- ⚠️ **주의 — 미검증 추정치:** 아시아 SOM $10~20억, 타겟 페르소나 월 예산 $100~500, 프리랜서 MRR $5k~50k — 근거 출처 링크 깨짐. 가격 전략 수립 시 자체 인터뷰 데이터로 보완 필요
- ⚠️ **주의 — 경쟁사 한국 대응:** Databox의 한국 플랫폼 깊이 부족은 "추정" 수준. 경쟁사가 한국 시장 미진출인 이유가 "기술 장벽"일 가능성 높음 — 차별점인 동시에 리스크
- 🚨 **핵심 리스크:** $29 스타터 플랜은 API 비용·AI 추론 비용·서버 비용 반영 시 손실 가능성 높음. $49 이상 재검토 권장
- 🚨 **가장 위험한 가정:** "네이버SA·카카오모먼트 API 안정적 연동 가능 + AI 인사이트 개발·유지보수 역량 충분" — 이 가정 붕괴 시 핵심 차별점 소멸

## 구현 주의사항 (준혁 조건부 GO 조건 반영)

> **이 섹션은 Claude Code가 구현 시작 전 반드시 확인해야 할 제약사항이다.**

### 🔴 MVP 범위 제한 (절대 준수)
- **1차 MVP 채널:** GA4 + 메타(FB+IG) + 네이버SA **3채널만**
- **카카오모먼트:** API 정책 검증 미완료 — 2차 릴리즈. 코드 구조만 확장 가능하게 설계
- **구글 애즈:** 3채널 안정화 후 추가 — 2차 릴리즈
- **멀티 워크스페이스:** 스타터 플랜 단일 워크스페이스만. DB 스키마는 멀티 대비로 설계
- **자체 AI/ML 모델:** 절대 금지. OpenAI GPT-4o API 외부 호출로만 구현
- **팀 협업·멤버 초대·권한 관리:** 2차 릴리즈
- **커스텀 위젯 편집기·슬랙·노션 연동·화이트라벨:** 2차 릴리즈

### 🟡 기술 검증 우선순위
- 네이버SA API 연동은 PoC 완료 전까지 Mock 데이터로 대체 가능하게 구현
- 각 채널 연동 모듈을 독립 서비스로 분리 — API 정책 변경 시 해당 모듈만 수정 가능하도록
- AI 인사이트는 월 10회 제한(스타터) — 비용 통제 필수

### 🟡 수익성 보호
- Stripe 결제 없이는 14일 이후 대시보드 접근 차단 로직 필수
- API 호출 비용 모니터링을 위한 사용량 로깅 필수 (채널별 API 호출 수, OpenAI 토큰 수)
