# LIANCP 쇼케이스 랜딩페이지 — 완전체 (showcase.html 초월 버전)

## 🎯 전략 분석 (서준 → 전체 팀)

### showcase.html 분석 결과
- **강점**: Silk 셰이더 + 숫자 카운터 + 스크롤 애니메이션 기본 골격
- **약점**: 3D 임팩트 부족, 시네마틱 연출 없음, 마이크로인터랙션 미흡
- **초월 전략**: 
  1. Silk → **LiquidChrome** 전환 (AI 유기체 느낌)
  2. 히어로에 **3D holographic logo** 추가 (회전+파티클)
  3. 스크롤 시 **카메라 패닝** (GSAP ScrollTrigger)
  4. 숫자 애니메이션 → **디지털 글리치** 효과
  5. **Magnetic cursor** + grain overlay
