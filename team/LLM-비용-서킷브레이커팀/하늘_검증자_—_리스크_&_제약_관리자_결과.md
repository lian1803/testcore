### Phase 1: 법적 안전장치 (4시간, 준혁+태양 협업)
**목표**: OpenAI/Anthropic API 직접 호출 모델로 전환, 프록시 금지 조항 우회

```typescript
// ❌ 금지: LiteLLM/프록시 중앙화 (ToS 위반)
// ✅ 허용: 클라이언트 직접 API 호출 + 서버 토큰 카운팅만

// 1. 클라이언트에서 개발자 본인 API 키로 직접 호출
const response = await fetch('https://api.openai.com/v1/chat/completions', {
  headers: { 'Authorization': `Bearer ${userOwnApiKey}` } // 사용자 키 저장 안 함
});

// 2. 서버는 토큰 수만 카운팅 (API 키 통과 없음)
const tokenCount = countTokens(response.usage);
await supabase.from('token_usage').insert({ 
  user_id: userId, 
  tokens: tokenCount,
  cost: tokenCount * 0.00002 // gpt-4 가격
});
```
