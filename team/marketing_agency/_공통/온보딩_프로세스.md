# 온보딩 프로세스 — 계약 → 서비스 시작

> 신규 클라이언트 계약 체결 후 실제 서비스 시작까지의 단계별 프로세스

---

## 전체 타임라인

```
Day 1: 계약 체결
  ↓
Day 2~3: 킥오프 미팅 + 정보 수집
  ↓
Day 4~7: 계정 세팅 + 액세스 권한 확보
  ↓
Day 8~14: 데이터 분석 + 전략 수립
  ↓
Day 15: 첫 실행 시작 (광고/콘텐츠 집행)
  ↓
Day 20: 첫 성과 리포트
  ↓
Day 30: 1개월 평가 회의
```

---

## 1단계: 계약 체결 (Day 1)

### 1-1. 계약 최종 확인 및 서명

**담당**: 영업 + 계약팀

체크리스트:
- [ ] 제안서 최종 확인 (가격, 서비스 범위)
- [ ] SOW 검토 및 수정
- [ ] 계약서 인쇄 및 서명 (2부)
- [ ] 세금계산서 발행
- [ ] 계약 기록 보관

**제출물**: 
- 서명된 계약서 2부 (클라이언트 1부, 회사 1부)
- 영수증/세금계산서

**전달자**: 영업팀 담당자

---

### 1-2. 내부 온보딩 시스템 등록

**담당**: 운영팀

체크리스트:
- [ ] CRM에 클라이언트 정보 입력 (회사명, 담당자, 연락처, 예산)
- [ ] 프로젝트 폴더 생성 (구글 드라이브/Notion)
- [ ] PM 배정
- [ ] 팀원 초대 (Slack, 공유 폴더, 프로젝트 관리 도구)
- [ ] 결제 처리 시작 (첫 2개월 선금)
- [ ] 계약 일정 공유 (팀 캘린더)

**제출물**:
- CRM 등록 완료 확인
- 프로젝트 링크 공유

---

## 2단계: 킥오프 미팅 (Day 2~3)

### 2-1. 킥오프 미팅 일정 예약

**담당**: PM

체크리스트:
- [ ] 클라이언트에게 킥오프 일정 제안 (2~3가지 옵션)
- [ ] 미팅 확정 (온라인 Zoom/오프라인)
- [ ] 참석자 확인 (클라이언트, PM, 관련 팀원)
- [ ] 미팅 준비 자료 준비 (프레젠테이션)
- [ ] 회의실 / Zoom 링크 예약

**실무**: 
- 킥오프 미팅 일정: __________ 예정
- 참석자: PM, 분석팀, 크리에이티브팀, 클라이언트

---

### 2-2. 킥오프 미팅 실행 (1.5시간)

**담당**: PM + 팀

**안건** (시간 배분):
1. 인사 및 팀 소개 (10분)
2. 계약 내용 재확인 (10분)
3. 클라이언트 현황 설명 (20분)
   - 제품/서비스 설명
   - 타겟 고객
   - 경쟁사 현황
   - 현재 마케팅 실적
4. 에이전시 전략 제시 (20분)
   - 채널 전략
   - 예상 성과
   - 타임라인
5. 액세스 권한 요청 (15분)
   - 광고 계정 관리자 권한
   - 웹사이트 FTP / 관리자 계정
   - GA4 / 분석 도구 접근
   - SNS 계정 관리자 권한
6. 정기 보고 회의 일정 확정 (10분)
   - 월별 보고 회의 (예: 월 말 목요일 14:00)
   - 긴급 연락 담당자

**미팅 자료 준비**:
- [ ] 팀 소개서 + 담당자 명함
- [ ] 전략 프레젠테이션 (1page = 전략 내용)
- [ ] 액세스 권한 요청 체크리스트 (자동 인쇄)
- [ ] 계약 내용 정리본

**미팅 후 액션**:
- 미팅 기록 (Notion / 구글 독스)
- 액세스 권한 요청 이메일 발송 (템플릿 사용)
- 정기 보고 회의 캘린더 초대

---

## 3단계: 정보 수집 및 계정 세팅 (Day 4~7)

### 3-1. 액세스 권한 확보

**담당**: PM + 기술팀

체크리스트:
- [ ] **광고 계정**
  - [ ] 메타 비즈니스 매니저 (관리자 권한)
  - [ ] 구글 애즈 (관리자 권한)
  - [ ] 네이버 광고 (관리자 권한)
  - [ ] 카카오 광고 (관리자 권한)
  - [ ] 틱톡 광고 (관리자 권한)
  - [ ] 쿠팡 애즈 (관리자 권한)

- [ ] **웹 분석**
  - [ ] 구글 애널리틱스 (GA4, 편집자 권한)
  - [ ] 서치 콘솔 (편집자 권한)
  - [ ] 자체 분석 도구

- [ ] **웹사이트 / 쇼핑몰**
  - [ ] 호스팅 FTP (Cafe24, 아임웹 등)
  - [ ] 워드프레스 관리자 계정
  - [ ] SSL 인증서 (필요 시)

- [ ] **SNS 계정**
  - [ ] 인스타그램 (관리자 권한)
  - [ ] 유튜브 (관리자 권한)
  - [ ] 틱톡 (관리자 권한)
  - [ ] 블로그 (관리자 권한)

- [ ] **기타**
  - [ ] 클라이언트 CRM / 고객 데이터 시스템
  - [ ] 슬랙 또는 협업 채널
  - [ ] 클라우드 스토리지 공유 폴더 (Google Drive / Sharepoint)

**권한 요청 템플릿 이메일**:
```
[제목] OOO 마케팅 서비스 — 액세스 권한 요청

안녕하세요, OOO입니다.

계약해주신 마케팅 서비스를 시작하기 위해 아래 권한들이 필요합니다.
5일 내에 제공해주시면 감사하겠습니다.

[필수 권한 체크리스트]
- 메타 비즈니스 매니저: ___@meta.com으로 관리자 권한 추가
- 구글 애즈: ___@google.com으로 관리자 권한 추가
- GA4: ___@google.com으로 편집자 권한 추가
... (이하 계속)

[첨부]
- 액세스_권한_요청_체크리스트.xlsx

감사합니다.
```

**확인 절차**:
- 클라이언트로부터 권한 확보 메일 받음
- 권한 테스트 (실제 로그인 확인)
- 권한 확보 완료 보고서 작성

---

### 3-2. 클라이언트 정보 수집

**담당**: PM + 분석팀

**수집 정보**:

1. **비즈니스 정보**
   - 회사 소개 자료
   - 제품/서비스 설명 (카탈로그, 스펙 시트)
   - 타겟 고객 정의 (연령, 성별, 관심사, 구매력)
   - 주요 경쟁사 3~5개

2. **마케팅 현황**
   - 지난 6개월 판매액 추이
   - 기존 마케팅 채널 및 예산 (구글/메타/네이버 등)
   - 월별 고객 수 및 구매 건수
   - 주요 제품/서비스 (탑 5)

3. **브랜드 자산**
   - 로고 파일 (AI, PNG 고해상도)
   - 브랜드 가이드라인 (색상, 폰트, 톤앤매너)
   - 기존 마케팅 자료 (배너, 영상, 사진)
   - 상품 이미지 (고해상도)

4. **웹사이트 / 쇼핑몰 정보**
   - 웹사이트 URL
   - 쇼핑몰 URL (쿠팡, 네이버, 자사몰 등)
   - 주요 페이지 설명
   - 월별 방문자 수

5. **기타**
   - 연락처 (담당자, 승인자)
   - 예산 수령 계좌 (정산용)

**수집 방식**:
- [ ] 온라인 양식 (Google Form)
- [ ] 킥오프 미팅 중 직접 수집
- [ ] 이메일로 자료 첨부 요청

**제출 기한**: 킥오프 미팅 후 7일

---

### 3-3. 계정 및 도구 세팅

**담당**: 기술팀 + PM

체크리스트:
- [ ] **분석 도구 연동**
  - [ ] GA4 설치 및 속성 생성
  - [ ] 메타 픽셀 설치
  - [ ] 구글 애널리틱스 클라이언트에 공유 (권한)
  - [ ] 자동 리포팅 설정 (Google Data Studio)

- [ ] **광고 계정 기본 설정**
  - [ ] 메타: 비즈니스 매니저 구조 정리, 광고 계정 생성
  - [ ] 구글: MCC(Multi-Client Center) 설정 (필요 시)
  - [ ] 네이버: 대행사 계정 연동
  - [ ] 각 계정에 클라이언트 이름 + 계약 기간 메모 추가

- [ ] **협업 도구 설정**
  - [ ] Slack 채널 생성 (#clnt-OOO-marketing)
  - [ ] Google Drive 폴더 공유 (구조: 클라이언트명/월별/산출물)
  - [ ] Notion 프로젝트 페이지 생성
  - [ ] 프로젝트 관리 도구 (Asana, Monday 등) 추가

- [ ] **문서 및 템플릿 준비**
  - [ ] 월별 성과 리포트 템플릿 (사전 작성)
  - [ ] 분석 대시보드 (Google Data Studio / Tableau)
  - [ ] 콘텐츠 캘린더 (Google Sheets)
  - [ ] 크리에이티브 체크리스트

**완료 보고**: 모든 세팅 완료 후 PM이 클라이언트에게 확인 이메일 발송

## 4단계: 분석 및 전략 수립 (Day 8~14)

### 4-1. 기존 데이터 분석

**담당**: 분석팀

**분석 대상**:
1. **웹사이트 분석**
   - GA4 기존 트래픽 추이 (최근 3개월)
   - 주요 유입 채널 (직접/검색/SNS/광고)
   - 인기 페이지 / 이탈률 높은 페이지
   - 전환율 및 전환 경로

2. **광고 성과 분석** (기존 광고 운영 중인 경우)
   - 메타 / 구글 / 네이버 등 기존 캠페인 성과
   - CPA, ROAS, CTR, 전환율
   - 주요 오디언스 세그먼트
   - 높은 성과 크리에이티브 분석

3. **경쟁사 분석**
   - 경쟁사 광고 현황 (Semrush, SimilarWeb)
   - 경쟁사 SNS 운영 현황
   - 경쟁사 가격 및 프로모션

4. **시장 현황**
   - 타겟 시장 규모
   - 검색 트렌드 (Google Trends, 네이버 트렌드)
   - 계절성 및 주기성

**산출물**:
- 분석 레포트 (15~20페이지, PPT)
- 핵심 인사이트 3가지
- 개선 기회점 (Opportunity Map)

---

### 4-2. 전략 수립

**담당**: 전략팀 + PM

**수립 내용**:

1. **채널 전략**
   - [ ] 우선 채널 선정 (예: 메타 광고 50%, SNS 30%, SEO 20%)
   - [ ] 각 채널별 목표 KPI
   - [ ] 각 채널별 월 예산 배분

2. **오디언스 전략**
   - [ ] 주 타겟 오디언스 3~5개 정의
   - [ ] 각 오디언스별 메시지
   - [ ] 오디언스별 채널 배분

3. **크리에이티브 전략**
   - [ ] 이미지 vs 영상 비율
   - [ ] 톤앤매너 (진지 vs 재미 vs 교육)
   - [ ] 주요 메시지 (USP: 고유한 판매 제안)

4. **성과 목표**
   - [ ] 3개월 목표 (CPA, ROAS, 매출)
   - [ ] 6개월 목표
   - [ ] 월별 마일스톤

5. **타임라인 및 우선순위**
   - [ ] Week 1-2: 계정 세팅 + 첫 광고 그룹 생성
   - [ ] Week 3-4: 첫 광고 집행 + 성과 모니터링
   - [ ] Month 2: 최적화 + 확장

**산출물**:
- 전략 문서 (5~10페이지)
- 월별 액션 플랜 (로드맵)
- 분기별 목표 수립

---

### 4-3. 전략 리뷰 회의 (Day 12~14)

**담당**: PM + 분석팀

**미팅 내용**:
- 분석 결과 공유 (20분)
- 제안 전략 설명 (30분)
- 클라이언트 피드백 수렴 (15분)
- 다음 단계 확인 (10분)

**사전 준비**:
- 분석 레포트 사전 전달 (미팅 3일 전)
- 프레젠테이션 자료 준비
- 영역별 담당자 준비 (분석팀, 크리에이티브팀)

**피드백 반영**:
- 회의 후 수정사항 정리
- 2~3일 내 최종 전략 문서 제공
- 승인 획득

## 5단계: 첫 실행 시작 (Day 15~20)

### 5-1. 첫 크리에이티브 제작

**담당**: 크리에이티브팀

**제작 범위**:
- [ ] **광고 크리에이티브** (3~5개)
  - 메타 광고: 스태틱 이미지 2개 + 비디오 1개
  - 구글 검색 광고: 텍스트 광고 3~5개
  - 배너 광고: GDN 사이즈 3~5개

- [ ] **SNS 콘텐츠** (5~10개)
  - 인스타 포스트 3개
  - 틱톡 영상 1~2개
  - 유튜브 썸네일 1개 (필요 시)

**제작 프로세스**:
1. 브리프 작성 (클라이언트 승인 필요)
2. 콘셉트 스케치 (3~5안)
3. 클라이언트 선택 및 피드백
4. 최종 제작 + 수정
5. 최적화 (해상도, 크기, 포맷)

**검수 체크리스트**:
- [ ] 브랜드 가이드라인 준수
- [ ] 텍스트 오류 확인
- [ ] 사이즈/해상도 확인
- [ ] 플랫폼별 최적화

---

### 5-2. 광고 캠프 세팅

**담당**: 광고팀

**광고 계정 구조** (예: 퍼포먼스 마케팅):

```
광고 계정
├─ Campaign 1: 신규 고객 확보
│  ├─ Ad Set 1-1: 관심사 타겟 (18~30세)
│  │  └─ Ad 1-1-1, 1-1-2 (크리에이티브 A/B)
│  └─ Ad Set 1-2: 관심사 타겟 (30~45세)
│     └─ Ad 1-2-1, 1-2-2 (크리에이티브 A/B)
├─ Campaign 2: 리타겟팅
│  ├─ Ad Set 2-1: 사이트 방문자 (7일)
│  └─ Ad Set 2-2: 장바구니 포기 고객
└─ Campaign 3: 브랜드 인지
   └─ Ad Set 3-1: 광범위 관심사 타겟
```

**설정 내용**:
- [ ] 캠페인 이름 및 목표 설정
- [ ] 오디언스 정의 (관심사, 행동, 사용자 정보)
- [ ] 예산 배분 (일일 예산, 시간대)
- [ ] 입찰 전략 (ROAS, CPA, 수동 입찰)
- [ ] 전환 추적 설정 (픽셀, 이벤트)

**확인 절차**:
- 캠페인 시뮬레이션 (광고 도달 예상)
- PM 최종 검수
- 클라이언트 승인

---

### 5-3. 첫 광고 집행

**담당**: 광고팀

**집행 절차**:
1. 클라이언트 예산 충전 확인 (매체비 사전 입금)
2. 모든 광고 세트 "활성화"
3. 실시간 모니터링 (첫 2시간)
   - 클릭 수
   - 전환 추적 정상 작동 여부
   - 오류 메시지 확인
4. 첫 성과 데이터 수집 (24시간 대기)

**문제 대응**:
- 픽셀 오류 → 기술팀 확인 + 수정
- 승인 거부 → 콘텐츠 수정 후 재제출
- 이상 CTR → 오디언스 조정

**보고**:
- PM이 클라이언트에게 "광고 집행 완료" 메일 발송
- 실시간 성과 대시보드 링크 제공

## 6단계: 초기 운영 및 모니터링 (Day 20~30)

### 6-1. 성과 모니터링 (주 2~3회)

**담당**: PM + 분석팀

**모니터링 항목**:
- [ ] 일일 클릭 수 / CTR
- [ ] 일일 전환 수 / 전환율
- [ ] CPA (고객 획득 비용)
- [ ] ROAS (광고비 대비 수익)
- [ ] 노출수 vs 클릭 비율 (확인)

**대시보드**:
- Google Data Studio (자동 업데이트)
- 캠프별 성과 실시간 추적
- 클라이언트에게 URL 공유

**이상 감지**:
- [ ] CPA 목표의 20% 초과 → 오디언스 조정
- [ ] CTR 급락 → 크리에이티브 피로도 확인, A/B 테스트 추가
- [ ] 전환 추적 오류 → 픽셀 재확인
- [ ] 예산 소진 속도 이상 → 입찰가 조정

### 6-2. 최적화 작업

**담당**: 광고팀

**가능한 최적화** (Day 20 이후):

1. **오디언스 최적화**
   - 낮은 성과 오디언스 제거
   - 높은 성과 오디언스 예산 증액

2. **크리에이티브 최적화**
   - 성과 낮은 이미지 제거
   - 성과 높은 크리에이티브 복제 + 확대

3. **입찰 전략 최적화**
   - 초기 수동 입찰 → 자동 입찰(ROAS/CPA 타겟)로 전환
   - 입찰가 미세 조정

**최적화 주기**:
- [ ] Week 1~2: 데이터 수집 기간 (변경 최소화)
- [ ] Week 3~4: 첫 최적화 라운드
- [ ] Month 2+: 주 1회 정기 최적화

**담당**: PM + 분석팀

**리포트 구성** (PPT/PDF):

   - 주요 성과 3가지
   - 목표 대비 달성도
   - 다음 달 전망

2. **성과 지표** (1~2page)
   | 지표 | 목표 | 실적 | 달성도 |
   |------|------|------|--------|
   | 지출 | __만원 | __만원 | _% |
   | 클릭 | __ | __ | _% |
   | 전환 | __ | __ | _% |
   | CPA | __원 | __원 | _% |
   | ROAS | _배 | _배 | _% |

3. **채널별 분석** (1page)
   - 메타/구글/네이버/기타별 성과 비교
   - 채널별 성과 미달 이유

4. **크리에이티브 분석** (1page)
   - 최고 성과 크리에이티브 Top 3

   - 일일 성과 차트

   - 경쟁사 벤치마킹 (있을 경우)

- PDF 이메일 발송 + Google Drive 공유
- 보고 회의 (월말 목요일 14:00)에서 30분 프레젠테이션

**보고 회의 안건**:
1. 성과 리포트 공유 (15분)
2. 의문점 및 질의 (10분)
3. 다음 달 액션 플랜 확인 (5분)
