## 5. 실행 타임라인

### 5-1. 킥오프 ~ 온보딩 (1주차)

| 일정 | 활동 | 결과 |
|------|------|------|
| 1일 (금) | 킥오프 미팅 | 전략 확인, 액세스 권한 설정 |
| 2~3일 | 계정 세팅 | 광고 계정, 분석 도구 연동 |
| 4~5일 | 기존 데이터 분석 | 현황 리포트 작성 |
| 7일 (금) | 첫 주 회의 | 1주 성과 검토, 최적화 방향 논의 |

### 5-2. 초기 실행 (2~4주)

| 주차 | 활동 | 산출물 |
|------|------|--------|
| 1주 | 계정 세팅 완료 | 광고 캠페인 구조 설계 |
| 2주 | 첫 광고 집행 | 초기 크리에이티브 (3~5개) |
| 3주 | 성과 모니터링 + A/B 테스트 | 성과 데이터 분석 |
| 4주 | 최적화 + 첫 월별 리포트 | 월간 성과 분석 리포트 |

### 5-3. 진행 중 (5주차 이후)

| 활동 | 주기 | 담당자 |
|------|------|--------|
| 성과 모니터링 | 주 3회 (평일 오전) | PM |
| 최적화 제안 | 주 1회 | PM + 분석팀 |
| 크리에이티브 A/B 테스트 | 월 4~6회 | 크리에이티브팀 |
| 월별 보고 회의 | 월 1회 (월말 목요일 2시) | 전담 PM + 클라이언트 |
| 분기별 전략 회의 | 분기 1회 | 전략팀 + 클라이언트 |
