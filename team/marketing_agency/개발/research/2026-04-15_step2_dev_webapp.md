## 중분류 리스트

---

### 1. 관리자 대시보드 (Admin Dashboard)

- 한 줄 정의: 내부 데이터 조회/관리용 백오피스. CRUD + 테이블 + 차트 + 권한관리 구성. 외부 공개 아님
- 바이브코딩 가능 여부: 가능 (v0.dev로 UI 생성 + Cursor로 백엔드 연동. 기본형 1~3일 가능. 복잡한 권한/다중 테이블 연동 시 1~2주)
- 출처 URL:
  - https://kmong.com/prices/%ED%92%80%EC%8A%A4%ED%83%9D
  - https://soomgo.com/prices/%EC%9B%B9-%EA%B0%9C%EB%B0%9C
  - https://www.wishket.com/project/145236/ (데이터 시각화 대시보드 2,000만원/30일 프로젝트)
  - https://freemoa-blog.com/868 (접근제어 대시보드 500만~1,000만원)
  - https://blog.wishket.com/blog/freelance-developer-rates-2025
- 시장 단가:
  - 최저: 300,000~800,000원 (기본 CRUD 관리자 페이지, 인증 없음)
  - 평균: 1,500,000~5,000,000원 (반응형, JWT 인증, API 연동, 차트 포함)
  - 상한: 5,000,000~15,000,000원 (복잡한 권한 구조, 실시간 데이터, 다중 테이블)
  - 에이전시: 10,000,000원+ (기업 내부 시스템 수준)
- 결제 구조: 위시켓/프리모아 기준 월 단가 계약(풀스택 평균 567만원/월). 크몽 선결제. 납기 7~30일
- 업계 명칭 변형: 백오피스, 어드민 페이지, 관리자 패널, 내부 관리 시스템

---

### 2. 내부 업무툴 (Internal Tool / Ops Tool)

- 한 줄 정의: 특정 업무 프로세스 자동화용 웹 도구. ERP 일부, 재고관리, 예약관리, 직원 스케줄러 등
- 바이브코딩 가능 여부: 가능 (도메인 로직이 복잡하지 않으면 Cursor로 충분. Retool/Appsmith 기반 노코드 대안도 존재)
- 출처 URL:
  - https://freemoa-blog.com/656 (커머스 어드민 3,000만원, 재무제표 시스템 1,000만원)
  - https://freemoa-blog.com/868 (병원 인트라넷 1,200만원)
  - https://www.wishket.com/project/145236/
  - https://hiteksoftware.co.kr/blog/software-development-cost/
- 시장 단가:
  - 최저: 500,000~2,000,000원 (단기능, 소규모팀 사용, 단순 입출력)
  - 평균: 3,000,000~10,000,000원 (복수 모듈, DB 연동, 사용자 인증)
  - 상한: 10,000,000~30,000,000원 (ERP 수준, 복잡한 비즈니스 로직, 유지보수 포함)
  - 에이전시: 30,000,000원+ (SI 프로젝트 수준. 프리모아/위시켓 상위 발주)
- 결제 구조: 마일스톤 기반 (착수금 30% → 중도금 40% → 완료 30%). 납기 30~90일
- 업계 명칭 변형: 사내 툴, 업무 자동화 시스템, 인트라넷, 백오피스 시스템

---

### 3. SaaS MVP (Software as a Service MVP)

- 한 줄 정의: 핵심 기능만 담은 구독형 SaaS 최초 버전. 인증+과금+핵심 1~2기능+관리자 포함
- 바이브코딩 가능 여부: 부분 가능 (프론트엔드 UI, API 엔드포인트, DB 스키마 AI 생성 가능. Stripe/결제 연동, 멀티테넌시 설계는 경험 필요)
- 출처 URL:
  - https://blog.wishket.com/blog/freelance-developer-rates-2025
  - https://yozm.wishket.com/magazine/detail/1731/
  - https://blog.wishket.com/%EC%95%B1-%EA%B0%9C%EB%B0%9C-%EA%B2%AC%EC%A0%81/
  - https://freemoa-blog.com/656 (헬스 포털 플랫폼 3,000만~5,000만원, 개발자 3인 2.5개월)
  - https://velog.io/@koeunyeon/%EC%8A%A4%ED%83%80%ED%8A%B8%EC%97%85-%EC%99%B8%EC%A3%BC-%EB%B9%84%EC%9A%A9-%EC%82%B0%EC%A0%95%ED%95%B4-%EB%B3%B4%EA%B8%B0
- 시장 단가:
  - 최저: 3,000,000~8,000,000원 (개인 프리랜서, 핵심 1기능, 인증 없음)
  - 평균: 10,000,000~30,000,000원 (인증+결제+관리자+핵심 2~3기능)
  - 상한: 30,000,000~80,000,000원 (멀티테넌시, CI/CD, 고도화 UX, 팀 개발)
  - 에이전시: 80,000,000원+ (클라우드 인프라 설계 포함)
- 결제 구조: 마일스톤 3~4단계 분할. 착수금 30% 선결제. 납기 60~180일
- 업계 명칭 변형: 스타트업 MVP, 서비스 프로토타입, 베타 서비스, PoC 개발

### 4. 데이터 시각화 (Data Visualization)

- 한 줄 정의: 수집된 데이터를 차트/그래프/히트맵 등으로 시각화하는 웹 화면. Tableau/Power BI 커스텀 대시보드 포함
- 바이브코딩 가능 여부: 가능 (Chart.js/Recharts/D3.js 기반은 AI로 충분. 복잡한 D3 커스텀 차트나 실시간 스트림 시각화는 전문 경험 필요)
- 출처 URL:
  - https://www.wishket.com/project/145236/ (미디어 데이터 시각화 대시보드 2,000만원/30일)
  - https://kmong.com/gig/662645 (데이터 시각화 대시보드 UI/UX 기획)
  - https://hiteksoftware.co.kr/blog/software-development-cost/
  - http://newsjel.ly/solutions/customizing (데이터 시각화 전문기업 수준)
- 시장 단가:
  - 최저: 500,000~1,500,000원 (Chart.js 기본 차트 3~5개, 정적 데이터)
  - 평균: 2,000,000~8,000,000원 (API 연동, 실시간 업데이트, 필터/드릴다운)
  - 상한: 10,000,000~30,000,000원 (D3.js 커스텀, 대용량 데이터, BI 수준)
  - 에이전시: 30,000,000원+ (뉴스젤리 같은 전문 시각화 기업 수준)
- 결제 구조: 프로젝트 단위 선금 30~50%. 위시켓 기준 30일~60일 납기
- 업계 명칭 변형: BI 대시보드, 리포트 시스템, 인터랙티브 차트, 분석 대시보드

## 바이브코딩/AI코딩 시장 영향

- SaaS MVP 개발 속도: AI 도구 활용 시 프론트엔드 80%, 백엔드 CRUD 60% 자동화 가능
- 원가 혁명: 기존 1,000만원짜리 MVP를 바이브코딩으로 100~200만원 원가에 제작 가능 (마진 80~90%)
- 위시켓 2025 실계약 기준 프리랜서 월 단가:
  - 풀스택 평균: 5,673,000원/월
  - 백엔드 평균: 5,703,000원/월
  - 프론트엔드 평균: 5,721,000원/월
  - 시니어 풀스택: 7,500,000원/월
  - 출처: https://blog.wishket.com/blog/freelance-developer-rates-2025
- 비교 포인트: 바이브코딩 작업자가 1개월에 SaaS MVP 3~5개 납품 가능 (기존 1개)
