# 바이브코딩 개발 서비스 — 1차 서칭: 대분류
조사일: 2026-04-15
조사범위: 2026년 한국 시장, 바이브코딩(AI 코딩) 기반 개발 외주 서비스
조사자: 재원 (리서처)

---

## 대분류 리스트

### 1. 웹사이트 제작
- **한 줄 정의:** 랜딩페이지·기업홈페이지·포트폴리오·이벤트 페이지 등 정보 전달 목적의 정적/반응형 웹사이트 개발
- **왜 바이브코딩에 중요한가:** HTML/CSS/JS 구조가 명확해 AI 코드 생성 적중률이 가장 높음. 시장 수요 폭이 넓고 단가 레인지가 5만~500만원으로 진입 용이
- **업계 실제 명칭 변형:** 홈페이지 제작, 랜딩페이지 제작, 원페이지 사이트, 이벤트 페이지, 반응형 홈피

---

### 2. 웹앱 / SaaS MVP
- **한 줄 정의:** 관리자 대시보드·내부 업무툴·SaaS 초기 버전·데이터 시각화 등 기능이 있는 브라우저 기반 애플리케이션 개발
- **왜 바이브코딩에 중요한가:** CRUD 패턴과 React/Next.js 컴포넌트 구조가 AI 생성에 최적화. 단가 150만~3,000만원으로 마진이 큼
- **업계 실제 명칭 변형:** 웹앱 개발, 대시보드 개발, 관리자 페이지, SaaS 개발, 어드민 패널, 풀스택 개발

---

### 3. 업무자동화
- **한 줄 정의:** 엑셀 자동화·웹 크롤링·API 연동·n8n 워크플로우·이메일 자동화 등 반복 업무를 코드로 자동 처리하는 서비스
- **왜 바이브코딩에 중요한가:** Python 스크립트 패턴이 단순해 AI가 즉시 생성 가능. ROI가 명확해 클라이언트 설득이 쉬움. 납기 1~3일 실현 가능
- **업계 실제 명칭 변형:** 업무자동화, RPA, 엑셀 자동화, API 연동, 크롤링, 스크레이핑, n8n, Zapier 대체

---

### 4. 챗봇 · AI 서비스
- **한 줄 정의:** 카카오톡 챗봇·텔레그램 봇·AI 고객상담·RAG 기반 문서 Q&A·AI 에이전트 등 LLM API를 활용한 지능형 서비스 개발
- **왜 바이브코딩에 중요한가:** API 연동 패턴이 표준화되어 있고 LangChain/LangGraph 보일러플레이트가 존재. 시장 단가 30만~5,000만원으로 스펙트럼 최대
- **업계 실제 명칭 변형:** AI 챗봇, GPT 연동, 카카오 챗봇, RAG 시스템, AI 에이전트, LLM 서비스, ChatGPT API 연동

---

### 5. 쇼핑몰 / 커머스
- **한 줄 정의:** 카페24·아임웹 커스터마이징, 자사몰 MVP, 예약·결제 시스템 연동 등 전자상거래 기능을 포함한 웹사이트 개발
- **왜 바이브코딩에 중요한가:** 토스페이먼츠 MCP 출시(2025)로 결제 연동이 1일 처리 가능해짐. 기존 200~500만원 작업이 바이브코딩으로 극단적으로 단축
- **업계 실제 명칭 변형:** 쇼핑몰 제작, 자사몰 구축, 카페24 커스텀, 아임웹 커스텀, 온라인 쇼핑몰, 결제 연동

---

### 6. 모바일 앱
- **한 줄 정의:** PWA(웹앱 모바일 최적화), React Native / Flutter MVP 등 스마트폰에서 동작하는 앱 개발
- **왜 바이브코딩에 중요한가:** React Native 컴포넌트 구조가 AI 생성에 적합. 단, 네이티브 모듈 연동은 한계 존재. PWA는 완전 바이브코딩 가능
- **업계 실제 명칭 변형:** 앱 개발, 모바일 앱, React Native, Flutter, iOS 앱, 안드로이드 앱, PWA

---

### 7. 크롬 확장 / 데스크톱 툴
- **한 줄 정의:** 크롬 익스텐션·Electron 데스크톱 앱·CLI 자동화 스크립트 등 브라우저 또는 데스크톱 환경에서 동작하는 툴 개발
- **왜 바이브코딩에 중요한가:** 크롬 익스텐션은 manifest.json + content_script 구조가 단순해 AI 생성이 쉬움. 시간당 마진이 가장 높은 카테고리
- **업계 실제 명칭 변형:** 크롬 확장, 크롬 플러그인, 익스텐션 개발, Electron 앱, 데스크톱 툴, CLI 스크립트

## 조사 메모

### 바이브코딩 적용 시 마진 순위 (높은 순)
1. 이벤트 페이지 / 랜딩페이지 (85~95% 마진)
2. 크롬 익스텐션 / CLI 스크립트 (80~90%)
3. 업무자동화 크롤링·API (75~85%)
4. 웹앱 대시보드 (75~85%)
5. 챗봇·AI 서비스 (70~85%)
6. 쇼핑몰 커스텀 (70~80%)
7. 모바일 앱 MVP (60~75%)

### 특이 발견
- **토스페이먼츠 MCP**: 결제 연동 개발이 2025년 공식 MCP 서버 제공으로 1일 처리 가능. 기존 단가 200~500만원짜리를 30~50만원에 납품 가능
- **크롤링 레드라인**: 네이버·카카오·쿠팡 등 크롤링 금지 서비스 존재. 법적 리스크 클라이언트 사전 고지 필수
- **RAG 시장**: 위시켓 기준 평균 단가 2,420만원. 템플릿화하면 반복 수주 가능 — 고부가가치 집중 전략의 핵심

### 한국 주요 플랫폼
- 크몽 (kmong.com): 소규모 단건, 선결제 에스크로
- 숨고 (soomgo.com): 견적 중심, 평균 단가 시장가
- 위시켓 (wishket.com): 중대형 프로젝트, 프리랜서 월 단가 기준
- 프리모아 (freemoa.net): 기업 프로젝트 주류

3. **챗봇·AI 서비스** — 단가 최고, 시장 성장 가파름
