- 바이브코딩 가능 여부: 가능 (Cursor/Claude로 VBA/GAS 코드 생성 → 복붙 납품 구조 성립)

### 4. n8n / Make(Integromat) 워크플로우 자동화

- 한 줄 정의: 노코드/로우코드 자동화 플랫폼(n8n, Make, Zapier)으로 앱 간 연결·트리거·데이터 라우팅 구축
- 바이브코딩 가능 여부: 제한적 (워크플로우는 GUI 기반, 단 커스텀 코드 노드에 AI 활용 가능)
- 출처 URL:
  - https://parseur.com/blog/zapier-n8n-make
  - https://impactflow.kr/product/n8n
  - https://unveilpersonality.com/n8n-가격-월-7천원으로-무제한/
  - https://elancer.co.kr/blog/detail/1018
- 시장 단가:
  - 플랫폼 자체 비용:
    - n8n 클라우드: 월 €24(약 3.5만원)~, 셀프호스팅 월 1~2만원 서버비
    - Make: 무료 플랜 있음, 유료 월 $9(약 1.3만원)~
    - Zapier: 월 $19.99(약 2.9만원)~
  - 외주 구축비 (워크플로우 설계 + 세팅):
    - 최저: 50만~100만원 (단순 3~5단계 자동화, 1~2개 앱 연결)
    - 평균: 100만~300만원 (복합 분기·조건·에러처리 포함)
    - 에이전시: 300만~500만원+ (멀티 시스템 통합, 온보딩 포함)
  - 유지비 절감 효과: 외주 월 200만원 → 자동화 유지 월 30만원 사례 (실사용 기준)
- 결제 구조: 구축비 일시불 + 플랫폼 구독료 월정액 분리
- 업계 명칭 변형: "노코드 자동화", "워크플로우 자동화", "n8n 구축", "Make 자동화", "재피어 연동", "iPaaS 구축"
- 특이사항: 한국 내 n8n 수요 급증 중. 셀프호스팅으로 데이터 보안 이슈 해결 가능. 바이브코딩보다는 "자동화 전문가" 포지션으로 팔리는 영역.

### 6. 데이터 파이프라인 (ETL)

- 한 줄 정의: 여러 소스에서 데이터 추출(Extract) → 정제·변환(Transform) → 저장(Load)하는 자동화 시스템 구축
- 바이브코딩 가능 여부: 부분 가능 (단순 ETL 스크립트는 AI 생성 가능, 대규모 Airflow/Spark 설계는 전문성 필요)
- 출처 URL:
  - https://blog.jetbrains.com/blog/2021/05/20/big-data-world-part-3-building-data-pipelines/
  - https://www.integrate.io/ko/blog/top-7-etl-tools-ko/
  - https://hiteksoftware.co.kr/blog/software-development-cost/
  - https://blog.hyperhire.in/sw-engineer-grade
- 시장 단가:
  - 최저: 100만~300만원 (단순 CSV/DB 연동, Python 스크립트 기반)
  - 평균: 300만~1,000만원 (스케줄링, 에러 알림, 멀티 소스 포함)
  - 에이전시/기업급: 1,000만~5,000만원+ (Airflow/dbt/Spark 기반 대규모 파이프라인)
  - 프리랜서 월 단가: 데이터 엔지니어 경력 3~5년 월 700~900만원 (SI 단가 기준)
  - 특급 데이터 엔지니어: 월 1,000만~1,400만원
- 결제 구조: 프로젝트 기반 또는 인력 파견형 월계약
- 업계 명칭 변형: "ETL 개발", "데이터 파이프라인 구축", "데이터 엔지니어링", "데이터 웨어하우스 구축", "배치 처리 자동화"
- 특이사항: 중소기업 수요는 적고 중견~대기업 또는 스타트업 스케일업 단계에서 수요 발생. 바이브코딩으로 단순 구조는 납품 가능하나 유지보수·장애 대응은 전문인력 필요.

## 승현에게 전달: 전략 힌트

- 단가 스펙트럼이 가장 넓은 건 크롤링(2만~500만)과 API 연동(35만~2,000만). 포지셔닝에 따라 같은 카테고리에서 10~100배 가격 차이 가능.
- n8n/Make 워크플로우 구축은 "바이브코딩"보다 "자동화 전문가" 포지셔닝이 더 유리. 플랫폼 지식 + 구독 세팅 노하우가 진입장벽.
- 엑셀 VBA는 시장 포화 (크몽 2,114개 서비스). 진입 쉬우나 레드오션. AI 연동(GPT+엑셀) 차별화 필요.
- ETL/데이터 파이프라인은 B2B 엔터프라이즈 세그먼트. 건당 단가 높지만 영업 사이클 길고 클라이언트 진입장벽 높음.
- 바이브코딩으로 가장 빠른 수익화: 크롤링 + API 연동 (코드 생성 → 테스트 → 납품 구조 단순)
