## 중분류 리스트

### 1. PWA (웹앱 모바일 최적화)

- 한 줄 정의: 웹 기술(HTML/CSS/JS)로 만들되 모바일 홈 화면 설치, 오프라인 캐싱, 푸시 알림 등 네이티브 앱 경험을 제공하는 방식
- 바이브코딩 가능 여부: 가능 (높음) — 기존 웹 개발의 연장선. manifest.json + Service Worker 추가 수준이라 AI 코딩으로 대부분 처리 가능. Next.js PWA 플러그인 활용 시 더욱 빠름
- 출처 URL:
  - https://velog.io/@dear_sopi9211/%EC%95%B1-%EA%B0%9C%EB%B0%9C-%EB%B0%A9%EB%B2%95%EC%9D%98-%EC%9E%A5%EB%8B%A8%EC%A0%90%EA%B3%BC-PWA
  - https://www.hanl.tech/blog/native-vs-hybrid-vs-pwa/
  - https://brunch.co.kr/@a33f93b357b349e/14
  - https://url.kr/p/guestpost/pages/2025/03/page-1742476350.php
- 시장 단가:
  - 최저: 50만원 (기존 반응형 사이트에 PWA 기능 추가만)
  - 평균: 200~500만원 (신규 PWA 앱 풀개발, 로그인·알림·오프라인 포함)
  - 에이전시: 1,000만원+ (복잡한 비즈니스 로직, 백엔드 포함)
- 결제 구조: 웹 개발과 동일하게 고정가 프로젝트 방식이 일반적. 앱스토어 배포 필요 없어 추가 비용 없음
- 업계 명칭 변형: "모바일 웹앱", "하이브리드 앱", "웹앱 제작", "앱스토어 없이 앱처럼", "모바일 최적화 웹"
- 비고: 네이티브 앱 대비 개발비 1/5 수준. iOS Safari는 PWA 지원이 제한적(푸시 알림 iOS 16.4+부터 가능). 앱스토어 심사 불필요 = 빠른 출시 강점

### 2. React Native / Flutter MVP

- 한 줄 정의: 하나의 코드베이스로 iOS·Android 모두 배포하는 크로스플랫폼 앱. React Native는 JS/TS 기반, Flutter는 Dart 기반
- 바이브코딩 가능 여부: 가능 (중간) — React Native는 기존 웹 개발자 친화적이라 AI 보조 효과 높음. Flutter는 Dart 문법 러닝커브 있으나 AI가 잘 지원. MVP 기능(로그인·목록·상세·결제)은 AI로 빠르게 완성 가능

  - https://blog.wishket.com/blog/41251
  - https://hiteksoftware.co.kr/blog/cost-of-android-app-development/
  - https://blog.gridge.co.kr/app-dev/
  - https://soomgo.com/market/products/6598008af12dec13d51883b5
- 시장 단가:
  - 최저: 120만원 (크몽 프리랜서, 단순 앱 화면 3~5개)
  - 평균: 300~800만원 (크몽 평균 250만원, 숨고 평균 앱 개발 건당 1,510,000원. 기능 포함 MVP는 500~800만원대)
  - 에이전시: 2,000~5,000만원 (디자인+기획+개발+백엔드 풀셋 MVP)
- 결제 구조: 크몽은 에스크로(완료 후 지급). 위시켓은 마일스톤 방식. 직계약 시 선금 30~50% 일반적. 유지보수 포함 계약 시 월 50~200만원 추가
- 업계 명칭 변형: "RN 앱 개발", "플러터 앱", "크로스플랫폼 앱", "iOS Android 동시 개발", "앱 외주"
- 비고: 크로스플랫폼 선택 시 iOS+Android 네이티브 대비 개발비 20~30% 절감. 2026년 기준 Flutter 채택률이 React Native 추월 추세. 바이브코딩 시대에 가장 수요 높은 카테고리

### 3. 앱 유지보수

- 한 줄 정의: 출시된 앱의 버그 수정, OS 업데이트 대응, 기능 소규모 추가, 서버 관리 등을 지속적으로 처리하는 계약
- 바이브코딩 가능 여부: 가능 (높음) — 버그 수정·기능 소규모 추가는 AI가 기존 코드 파악 후 빠르게 처리 가능. 반복 업무 패턴화에 유리

## 승현에게 전달: 전략 힌트
- PWA는 "앱스토어 없이 앱처럼" 포지션으로 소상공인·1인 브랜드 타겟 틈새 포지션 가능. 개발 비용 낮고 마진 높음
- React Native/Flutter MVP는 단가가 높아 1건 수주 시 수익성 최고. 단, 경쟁 치열 — "AI로 3주 납기"로 차별화 필요
- 유지보수는 장기 계약 시 LTV(고객생애가치) 극대화. 바이브코딩 업체가 "개발 + 유지보수 패키지"로 묶어 파는 전략 유효
