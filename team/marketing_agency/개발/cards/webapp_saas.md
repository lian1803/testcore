## 카드 1. 관리자 대시보드 (CRUD)

### A. 돈
- 시장 단가:
  - 최저: 300,000~800,000원 (단순 테이블 조회)
  - 평균: 1,500,000~5,000,000원 (CRUD 완성형)
  - 에이전시: 5,000,000~15,000,000원+
- 바이브코딩 원가: 300,000~800,000원
- 마진율: 75~85%
- 결제 구조: 선금 30~50% + 잔금
- 출처:
  - https://www.wishket.com/project/145236/ (데이터 시각화 대시보드 2,000만원/30일)
  - https://freemoa-blog.com/868 (접근제어 대시보드 500~1,000만원)

### B. 실행
- 납기: 14~30일
- 바이브코딩 가능 여부: 가능 (React + Supabase / Firebase 조합)
- 주요 스택: Next.js + Tailwind + Supabase (DB+Auth 포함)
- 포함 범위: 로그인, CRUD 테이블, 검색/필터, 엑스포트(CSV)
- 제외 범위: 복잡한 권한 관리, 실시간 협업, 멀티테넌시

## 카드 4. SaaS MVP

### A. 돈
- 시장 단가:
  - 최저: 3,000,000~8,000,000원 (기능 3~5개 MVP)
  - 평균: 10,000,000~30,000,000원
  - 에이전시: 30,000,000~80,000,000원+
  - 위시켓 프리랜서 풀스택 월단가: 평균 5,670,000원
- 바이브코딩 원가: 1,000,000~3,000,000원 (MVP 기준)
- 마진율: 70~90%
- 결제 구조: 선금 30%+마일스톤별 납금
- 출처:
  - https://blog.wishket.com/blog/freelance-developer-rates-2025
  - https://hiteksoftware.co.kr/blog/software-development-cost/

### B. 실행
- 납기: 30~90일
- 바이브코딩 가능 여부: 부분 가능 (기능 범위 엄격히 제한 필요)
- 주요 스택: Next.js + Supabase + Stripe + Vercel
- MVP 정의: 로그인·핵심기능 1~2개·결제·대시보드
- 유지보수: 월 500,000~1,500,000원 별도
