## 카드 1. PWA (웹앱 모바일 최적화)

### A. 돈
- 시장 단가:
  - 최저: 500,000원 (기존 웹사이트 PWA 변환)
  - 평균: 2,000,000~5,000,000원 (신규 개발)
  - 에이전시: 10,000,000원+
- 바이브코딩 원가: 300,000~1,000,000원
- 마진율: 75~85%
- 결제 구조: 선금 50%+잔금
- 출처:
  - https://blog.wishket.com/blog/41251
  - https://kmong.com/prices/앱-개발

### B. 실행
- 납기: 7~21일
- 바이브코딩 가능 여부: 완전 가능 (Service Worker + manifest.json 패턴 표준)
- 장점: 앱스토어 심사 없음, 단일 코드베이스, 즉시 업데이트
- 단점: iOS 제한 (push 알림 일부 제한), 카메라·블루투스 등 네이티브 기능 제한

## 카드 2. React Native / Flutter MVP

### A. 돈
- 시장 단가:
  - 최저: 1,200,000~3,000,000원 (기능 1~2개 MVP)
  - 평균: 3,000,000~8,000,000원 (표준 MVP)
  - 에이전시: 20,000,000~50,000,000원+
  - 서울 기준 앱 개발 시간당: $35~75 (Hitek 2026 데이터)
  - 위시켓 프리랜서 모바일 개발자 월단가: 400~700만원
- 바이브코딩 원가: 500,000~2,000,000원
- 마진율: 60~75%
- 결제 구조: 마일스톤별 (설계→개발→배포)
- 출처:
  - https://hiteksoftware.co.kr/blog/cost-of-android-app-development/
  - https://kmong.com/prices/앱-개발

### B. 실행
- 납기: 30~90일
- 바이브코딩 가능 여부: 가능 (컴포넌트 구조는 AI 생성 적합, 네이티브 모듈은 한계)
- 주요 스택: React Native + Expo + Supabase / Firebase
- 포함 범위 (MVP): 로그인, 핵심 기능 1~2개, 푸시 알림, 앱스토어 배포
- 앱스토어 배포: Apple Developer $99/년, Google Play $25 일회성

## 카드 3. 앱 유지보수

### A. 돈
- 시장 단가:
  - 최저: 월 100,000원 (단순 버그 수정)
  - 평균: 월 500,000~1,500,000원 (정기 업데이트 포함)
  - 에이전시: 월 3,000,000원+
- 바이브코딩 원가: 월 100,000~300,000원
- 마진율: 70~80%
- 결제 구조: 월 정기 청구
- 출처: https://blog.wishket.com/blog/40069

### B. 실행
- 납기: 상시 대응 (24~72시간 이내)
- 바이브코딩 가능 여부: 가능 (코드 이해→수정 패턴)
- 포함 범위: OS 업데이트 대응, 버그 수정, 소규모 기능 추가
- 영업 포인트: 개발 완료 후 유지보수 계약 = 재귀 수익 구조
