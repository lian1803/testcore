| 구분 | 플랫폼 중개 (레뷰·디너의여왕·파블로 등) | 에이전시 직접 섭외 |
|------|--------------------------------------|-------------------|
| 블로거 풀 | 플랫폼 내 등록 회원 한정 | 외부 블로거 포함 전체 |
| 선정 방식 | 블로거 자체 신청 → 광고주 선정 | 에이전시가 직접 발굴·컨택 |
| 자영업자 편의 | 셀프 등록 가능, 운영 자동화 | 에이전시 의존, 담당자 필요 |
| 콘텐츠 통제 | 조건(키워드·사진 수) 지정 가능 | 방향 기획까지 에이전시가 설계 |
| 비용 구조 | 플랫폼 수수료 + 체험 제공비 | 에이전시 수수료 + 체험 제공비 (최소 예산 높음) |
| 타겟팅 정밀도 | 낮음 (풀 내 경쟁 신청) | 높음 (지역·업종·등급 지정 가능) |

### 2026년 뒷광고법 현황

- 공정위 「경제적 이해관계 표시 안내서」 개정 적용 중. 핵심 변경: 광고 표시를 게시물 **제목 또는 첫 부분**에 반드시 명시 (이전: 글 하단 허용 → 현재: 불가)
- 해시태그 속 광고 문구 은닉, 배경색과 유사한 색으로 표시하는 행위 모두 위반
- 2024년 네이버 블로그·인스타그램·유튜브 모니터링 결과: 표시광고법 위반 의심 게시물 약 2만6천 건 적발
- 2024년 네오프 제재 사례: SNS 후기 뒷광고 대행 2,300건 → 공정위 제재 확정

### 빠질 뻔했다가 발견

1. **기자단형 체험단** — "체험단"이라는 이름이 붙어 있지만 실제 방문·체험이 없는 원고 대필형 상품. 업계에서는 체험단과 기자단을 혼용하지만 공정위 리스크 구조가 완전히 다름. 별도 소분류로 분리 필수.

2. **숙박 전문 플랫폼(오블리뷰·잠자리체험단)** — 맛집·뷰티 중심 조사에서 놓칠 뻔했으나 오프라인 자영업자(펜션·소형 호텔·게스트하우스) 대상으로는 독립 카테고리로 다룰 필요 있음.

3. **네이버 인플루언서 블로그 상품의 별도 분리 필요성** — 일반 블로그 체험단과 네이버 인플루언서 인증 블로거 체험단은 노출 경로(인플루언서 탭 vs 뷰 탭)와 단가 구조가 완전히 다름. 같은 "방문형 맛집 체험단"이라도 두 상품은 별개로 제안 가능.

4. **모두의체험단(소상공인 특화) 플랫폼** — 무료 등록 가능 구조 → 예산 제약 심한 소규모 자영업자 대상 에이전시 서비스 설계 시 레퍼런스가 되는 플랫폼.

5. **피부과·의료기관 체험단의 이중 규제** — 공정위 광고 표시 의무 + 의료법상 의료광고 사전 심의 대상 중복 적용 가능. 뷰티 체험단 중 의료기관 포함 시 에이전시 리스크 상승.
