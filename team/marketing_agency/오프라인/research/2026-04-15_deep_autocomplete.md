## 서비스 실체

### 정의
네이버 검색창에 특정 키워드를 입력할 때 아래에 뜨는 자동완성 추천어(드롭다운 목록)에 원하는 키워드가 노출되도록 조작하는 마케팅 기법. 예를 들어 "강남 피부과"를 입력하면 자동완성에 "강남 피부과 [업체명]"이 뜨게 만드는 것.

### 작동 방식 (기술적 원리)
네이버 자동완성은 실사용자의 검색 패턴(빈도, 연속 검색 등)을 통계화해서 생성된다. 이를 인위적으로 만드는 방법은 2가지다:

**방식 A — 실사용자 동원 (리워드형)**
- 리워드 앱/금융 플랫폼을 통해 실제 사람들에게 돈을 지급
- "특정 키워드 입력 → 다음 키워드 연속 입력 → 클릭" 행동을 반복시킴
- 네이버 알고리즘에 "이 두 단어를 연속으로 검색하는 사람이 많다"는 신호 주입
- 현재 시장에서 가장 널리 쓰이는 방식. 봇 대비 탐지 어려움

**방식 B — 매크로/프로그램 자동화**
- 타인 명의 다수 계정 + 매크로 프로그램으로 연속 검색 자동 실행
- 아이엠마케터(iammarketer.com) 같은 업체가 90만원짜리 프로그램 판매 (현재 신규 판매 중단 상태)
- 대행사 전용 버전(PC 갯수 무제한): 추가 약 500만원
- 2022~2025년 판결에서 주로 이 방식이 기소됨

### 합법/불법 여부
**불법이다. 명확하게.**

- 법적 근거: 컴퓨터등장애업무방해죄 (형법 제314조 2항)
- 2025년 1월 서울남부지법 판결: 광고대행업체 대표 **징역 1년 + 추징금 23억원** 선고
- 공범 13명 집행유예/추징금/보호관찰
- 재판부 판시: "검색 알고리즘 및 통계 시스템 교란 → 인터넷 정보 생태계 신뢰 훼손"
- 범행 기간 2018~2022년, 총 1만3,845회 조작

네이버 전·현직 연관검색어 조작 관련 2심도 징역형 확정 (법률신문 보도).

### 2026년 현재 상태
**시장은 여전히 작동 중이나 탐지 리스크가 크게 상승했다.**

- 네이버가 2024년 8월 LLM 기반 판별 시스템 업데이트 발표
  - "비정상적 패턴으로 인입된 키워드 자동완성에서 제외"
  - 오탈자·무의미 문자·연관도 낮은 키워드 자동 필터링
- 그럼에도 시장 업체들은 2026년 현재 서비스 판매 지속 중 (GPA KOREA, 업템포AD, 지프라마케팅 등)
- 실행사들 설명: "국내 실사용자 100%로 진행" (리워드형으로 탐지 우회 시도)
- 미노출 시 추가 진행 보장 = 탐지로 날아가는 경우 상시 발생한다는 의미

## 키워드 수 / 기간 스펙

공통적으로 확인된 스펙:
- **키워드 개수**: 메인 키워드 1개 + 자동완성 노출 키워드 3~10개
- **노출 기간**: 25일 보장 (주말 포함) — 업계 표준
- **노출 시작 소요**: 2~7일 (작업 시작 후)
- **미노출 시**: 추가 진행 또는 키워드 변경으로 보장 유지 (탐지 시 재작업)

## 리스크 & 주의사항

### 법적 리스크 (클라이언트 + 에이전시 동시 노출)
1. **형사 처벌 실재**: 2025년 판결로 징역형 + 23억 추징금 실제 집행됨. "몰랐다"는 항변 성립 어려움
2. **의뢰자 공범 가능성**: 법원이 조작 의뢰 광고주도 공범으로 기소한 사례 존재

### 서비스 품질 리스크
3. **네이버 LLM 탐지 강화**: 2024년 8월 이후 비정상 패턴 자동 필터링. 돈 쓰고 노출 안 될 가능성 증가
4. **환불 불가**: 대부분 업체가 작업 시작 후 환불 거부. 탐지로 사라져도 보상 구조 불명확
5. **지속성 없음**: 25일 이후 자연 소멸. 반복 구매 필요 → 월 비용 구조

### 에이전시 운영 리스크
6. **클라이언트 책임 전가 위험**: 클라이언트가 나중에 "에이전시가 불법 서비스 팔았다"고 민사 제기 가능
7. **수익성 의문**: 소매가 25~30만원 - 도매가 10만원 = 마진 15~20만원. 관리 공수 감안 시 저수익
8. **네이버 검색 결과 패널티**: 만약 클라이언트 업체가 조작으로 적발되면 검색 결과 영구 퇴출 가능 (쇼핑 기준 2회 적발 = 영구 퇴출)

### 2026년 현재 시장 실태
- 시장은 살아 있지만 "위험을 알고 하는" 영역으로 전환
- 실행사들이 "실사용자 100%"를 강조하는 것 자체가 탐지 회피 마케팅
- 소상공인 향 판매는 계속되고 있으나 대형 에이전시는 공식적으로 취급하지 않는 추세
