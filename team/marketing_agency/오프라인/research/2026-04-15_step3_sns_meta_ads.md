### 3. 전환 캠페인 광고 대행 (리드·예약 전환)

- 광고 목적: 판매·잠재 고객(전환) — CPA 최소화, 예약/상담 전환 극대화
- 지면: 인스타그램 피드, 릴스, 스토리, 페이스북 피드

### 4. 리드 생성(잠재 고객) 광고 대행
- 한 줄 정의: 인스타그램·페이스북 내 즉시 작성 가능한 리드 폼을 통해 고객 연락처(이름·전화·이메일)를 수집하는 캠페인 서비스
- 광고 목적: 잠재 고객(Leads) — 리드 폼 제출, 고객 정보 DB 확보
- 지면: 인스타그램 피드, 스토리, 페이스북 피드
- 포맷: 단일 이미지 + 즉시 작성 폼(앱 이탈 없음), 동영상 + 폼
- 타겟 업종: 상담 기반 업종(피부관리, 학원, 리모델링, 법무·세무), 고관여 서비스(웨딩, 성형외과 인근)
- 비고: 오프라인 매장에서 DB 마케팅 연계 시 가장 효과적. 메타 내 이탈 없이 정보 수집 가능

### 5. 매장 방문(로컬 도달) 캠페인 광고 대행
- 한 줄 정의: 반경 설정 위치 기반 타겟팅으로 오프라인 매장 근처 잠재 고객에게 노출하고 지도·길찾기 클릭을 유도하는 로컬 캠페인 서비스
- 광고 목적: 매장 트래픽(Store Traffic) / 도달(Reach) — 지역 노출, 방문 의향 유도
- 지면: 인스타그램 피드, 스토리, 페이스북 피드, 메신저
- 포맷: 단일 이미지(지도 CTA 포함), 동영상, 스토리 스와이프업
- 타겟 업종: 동네 카페·음식점, 미용실, 세탁소, 꽃집 등 반경 내 상권 의존 업종 전체
- 비고: 2025년 5월 오프라인 전환 API(Offline Conversions API) 폐기 후 표준 CAPI로 통합. 매장 방문 측정은 CAPI + 오프라인 이벤트 세트 연동 필요

### 9. Advantage+ (AI 자동화) 캠페인 광고 대행
- 한 줄 정의: 메타의 머신러닝 기반 Advantage+ 시스템으로 타겟·지면·소재 배분을 자동 최적화하는 캠페인 운영 서비스
- 광고 목적: 전환·판매 — AI 학습 기반 ROAS 최대화, 수동 조작 최소화
- 지면: 메타 전 지면 자동 배분(피드·릴스·스토리·탐색·페이스북)
- 포맷: 복수 소재 업로드 후 AI 자동 선택 및 최적화
- 타겟 업종: 광고 경험이 부족한 자영업자, 소재 A/B 테스트 여력이 없는 소규모 매장
- 비고: 2026년 메타 권장 방향. ASC(Advantage+ Shopping Campaign)는 쇼핑몰 중심이나, 로컬 서비스업은 Advantage+ Audience(자동 타겟팅) + 전환 목적 조합으로 활용. 수동 세팅 대비 학습 속도 빠름

### 2026년 메타 광고 최신 트렌드 (AI 타겟팅 등)
- Advantage+ 전면 확대: 메타는 2026년까지 광고 제작·타겟팅·성과 분석 전 과정 AI 자동화 완성 목표. 광고주는 URL 또는 소재만 입력하면 AI가 텍스트·이미지·영상 자동 생성
- Advantage+ Audience: 기존 수동 타겟팅(관심사·맞춤 타겟)을 대체. AI가 전환 가능성 높은 사용자를 자동 탐색. 메타 공식 권장 방향
- 캠페인 목표 6개로 단순화 확정: 인지도 / 트래픽 / 참여 / 잠재 고객 / 앱 홍보 / 판매
- 스레드 광고 2026년 공식 시작 (한국 포함) — 기존 캠페인에 지면 추가만으로 노출 가능
- 인스타그램 라이브 광고 2026년 3월 한국 롤아웃 확인
- 릴스 알고리즘 변화: 단순 '좋아요' 기반 → 저장·체류·검색 기반. 릴스가 탐색 검색엔진 역할 강화

### 오프라인 매장 전환 측정 방식
- 2025년 5월: 기존 오프라인 전환 API(Offline Conversions API) 공식 폐기 → 표준 CAPI(Conversions API)로 통합
- 측정 원리: 메타 내 고객 데이터(광고 노출 사용자) + 매장에서 수집한 고객 데이터(POS, CRM) 매칭 → 광고 본 사람이 실제 방문·구매했는지 판단
- 픽셀 + CAPI 병행 설치 메타 공식 권장 (데이터 누락·중복 최소화)
- Signals Gateway: 온·오프라인 시그널 단일 파이프라인 통합 가능
- 자영업자 현실: 소규모 매장은 CAPI 연동 기술 부담 있음 → 대행사가 세팅 대행 시 차별화 포인트

### 빠질 뻔했다가 발견
- 스레드 광고: 2026년 신규 지면으로 대부분 대행사가 아직 서비스 카탈로그에 미반영 상태. 선점 여지 있음
- 라이브 광고: 한국 롤아웃이 2026년 3월이라 데이터 거의 없음. 조사하지 않았으면 놓쳤을 소분류
- 페이스북 피드 단독 캠페인: "인스타만" 필터 강하게 걸었으면 누락될 뻔. 40대 이상 자영업자 고객(예: 동네 한식당) 공략 시 필수
- Click-to-Messenger: 인스타그램 DM 연결 광고와 메신저 연결 광고가 구조적으로 같은데, 업종에 따라 전화보다 메신저 상담이 훨씬 전환율 높음 — 별도 소분류로 분리 가치 있음

### 4차 필요 여부 + 우선순위
- 4차 필요: O
- 우선순위 1위 — 광고 소재 제작 대행 (영상·이미지·카피 포함): 릴스 광고·피드 광고 모두 소재 품질이 성패를 결정. 대행사가 소재까지 묶어서 판매하는지가 핵심 차별화 요소
- 우선순위 2위 — 픽셀·CAPI 세팅 대행: 자영업자 혼자 불가. 오프라인 전환 측정의 기술 진입장벽 → 대행 서비스화 가능
- 우선순위 3위 — 메타 + 네이버플레이스 통합 패키지: 오프라인 매장 특성상 메타 인지 → 네이버플레이스 검색 전환 루프가 실제 고객 여정. 통합 운영 패키지 수요 있음
