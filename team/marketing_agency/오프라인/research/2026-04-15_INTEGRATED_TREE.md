## 🎯 리안 찍은 5계열 (4차 완료, 심화)

### ★1. 네이버 플레이스 마케팅 → 리워드 트래픽 마케팅 (4차 완료)
**⚠️ 전체 회색지대. 합법 0 / 회색 7 / 어뷰징 경계 3. 계약서 책임제한 조항 필수.**

- 3차 SKU 10개:
  - 검색 유입 / 체류 시간 / 저장하기 / 공유하기 / 길찾기 / 전화 연결 / 톡톡 문의 / **퀴즈 미션 ★** / **복합 번들 ★** / Keep하기
- 4차 심화 (19 소소분류, SKU 16):
  - **퀴즈/정답 미션** (6 SKU) — 메뉴/영업시간/가격/위치/특징/메뉴사진 맞추기
  - **복합 번들** (6 SKU) — 검색+저장 / 저장+공유 / 5종콤보 등
  - **검색 유입 키워드별** (4 SKU) — 업종+지역 / 메뉴명 / 브랜드명 / 롱테일
- **주목:** 리워드 매체(앱)가 독립 SKU — 에코/쿠마/원픽/솔리드/비즈/그로우 등 수십 개 매체별 상품 분화 완료
- **업종 분기:** 맛집/카페 = 저장+유입 번들 / 일반업종 = 저장 단독
- **파일:** `step3_naver_place_reward_traffic.md`, `step4_naver_place_reward_traffic.md`

### ★2. 블로그·바이럴 마케팅 → 블로그 체험단 (4차 완료)
**⚠️ 의료법 사전심의 필수 업종 분리. 뒷광고 고위험 3개.**

- 3차 SKU 8개:
  - 방문형 맛집·카페 / 방문형 뷰티·시술 / 방문형 숙박·여가 / **인플루언서급 프리미엄** / **기자단형 (뒷광고 고위험)** / **플랫폼 중개형** / **에이전시 직접 섭외형** / 배송형(오프라인 해당없음)
- 4차 심화 (14 SKU):
  - **A. 맛집·카페 심화 (5 SKU)**: 단발 블로그 단독 / 블로그+인스타 / 블로그+숏폼 / 정기 월 N회 / 프리미엄 매크로
  - **B. 플랫폼 vs 직접 섭외 (5 SKU)**: 레뷰 / 디너의여왕 / 리뷰플레이스·리뷰노트 / 에이전시 DB보유 / 에이전시 DM네고
  - **C. 뷰티·시술 (4 SKU)**: 네일·마사지·피부관리실 / **피부과·성형외과 (법무검토 전 판매 보류)** / **치과 (심의 대행 필수)** / 헤어샵·미용실
- **주목:** 블로그 1건도 의료광고 사전심의 대상. 마사지/네일/피부관리실은 표시광고법만 → **차별화 포인트**
- **파일:** `step3_blog_viral_experience.md`, `step4_blog_viral_experience.md`

### ★3. SNS 마케팅 (인스타·메타) → 메타 광고 대행 (4차 완료)
**주목: 픽셀+CAPI 세팅이 독립 원타임 과금 상품. 스레드 광고 = 선점 기회.**

- 3차 SKU 14개:

- 4차 심화 (12 소소분류, SKU 10 확정 + 2 조건부):
  - **A. 매장 방문 (4)**: 반경 기반 / 복수 매장 위치 자산 / 위치 관심(유동인구) / **CAPI+Signals Gateway 세팅**
  - **B. 전환 캠페인 (4)**: 네이버 예약·톡톡 연동 / 자사 예약·아임웹 결제 (픽셀+CAPI 포함) / 인스타 DM 리드 / 전화 전환(Click-to-Call)
  - **C. 릴스 광고 (4)**: 실사 촬영 + 집행 / 템플릿 기반 + 집행 / AI 생성 + 집행(조건부) / 6초 숏 집행(소재 미포함, 조건부)
- **주목:** Offline Conversions API 2025-05 폐기 → 픽셀+CAPI 세팅 필수 선행 상품 / 인스타 라이브 광고 2026-03 한국 롤아웃
- **파일:** `step3_sns_meta_ads.md`, `step4_sns_meta_ads.md`

### ★4. 당근 마케팅 → 피드광고(동네광고) (4차 완료)
**시장 미성숙: 전문 대행사 거의 없음 → 진입 기회.**

- 4차 심화 (신규 5 SKU, 누적 10):
  - **A. CPC 최적화 관리형 (3 SKU)**: 월 관리 / 주 관리 / 일일 관리

- **주목:** "전문가모드 권한 부여"가 독립 상품 (기술장벽 아닌 **인식장벽**) / 크몽 "1회 결제 평생 관리" 프라이싱 / 치과 당근 불법광고 적발 → 심의 서류 처리 대행 수요
