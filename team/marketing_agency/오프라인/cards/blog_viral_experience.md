## 카드 1. 방문형 맛집·카페 — 단발 블로그 단독 (4차 A-1)

### A. 돈
- 시장 판매가 (에이전시 대행 기준, 체험 제공비 별도):
  - 최저: 10,000원/건 (플랫폼 등록 수수료만, 체험 제공비 광고주 직부담)
  - 평균: 15,000~25,000원/건 (플랫폼 수수료 포함, 체험 제공비 별도)
  - 에이전시 대행 시: 20,000~50,000원/건 (섭외·관리비 포함)
  - 출처: https://blog.assaview.co.kr/블로그-체험단-업체-비용-한방에-알아보기 / https://soomgo.com/prices/체험단-마케팅
- 체험 제공비 (광고주 실비): 맛집 기준 1팀(2인) 2만~7만원 상당 식사·음료
  - 예시: 풀코스 메뉴 2인 47,800원 / 2.5만원 식사권 / 6만원 식사권 등 사례 확인
  - 출처: biz.revu.net 캠페인 실사례 (레뷰 총정리 brunch.co.kr/@@dskC/73)
- 플랫폼 수수료 구조: 레뷰 기준 캠페인 등록 단위 과금. 선금 결제. 부가세 별도
- 에이전시 마진율: 체험 제공비 제외 수수료 기준 약 30~50% 추정 (미확정 — 에이전시별 상이)
- 결제 구조: 선금 100% (플랫폼), 에이전시 선금 50~70% + 납품 후 잔금 일반적

### B. 실행
- 실행 주체: 플랫폼 중개 또는 에이전시 대행
- 실행사 후보:
  - 레뷰(REVU): https://biz.revu.net — 국내 최대, 122만 인플루언서, 방문·배송·기자단 전부
  - 디너의여왕: https://dinnerqueen.net — 맛집·푸드 특화, 블로그+인스타+클립 통합
  - 파블로: https://pavlovu.com — 24시간 빠른 선정, 전 업종 범용
  - 모두의체험단: https://www.modan.kr — 소상공인·자영업자 특화
  - 아싸뷰: https://assaview.co.kr
- 블로거 DB: 플랫폼 내부 회원 풀 (레뷰 122만, 파블로 규모 미확정)
- 리드타임: 캠페인 등록 → 포스팅 완료 약 3~4주 (모집 7~10일 + 방문 7~10일 + 작성 10일)
- 납품 결과물: 포스팅 URL + 6개월 모니터링 리포트 (레뷰 기준)
- 고객 자원: 매장명·주소·메뉴·제공 내역·예약 가능 시간대·키워드 요청사항
- 적합 업종: 한식·일식·중식·양식·카페·디저트·분식 등 오프라인 음식점
- 부적합 업종: 의료기관, 배달 전용 매장(방문 불가)

### C. 계약
- 최소 계약: 캠페인 1건 단위 가능 (플랫폼 기준). 에이전시는 최소 5건~10건 패키지 일반
- 해지·위약금: 캠페인 시작 전 취소 가능 (플랫폼별 상이). 시작 후 환불 어려움
- KPI 정의: 포스팅 게재 완료 (URL 납품) 기준. 조회수·상위노출 보장은 별도 추가 계약 필요. 표준 KPI는 "포스팅 게재 건수"만
- 재작업 한도: 미확정 — 플랫폼별·에이전시별 상이. 계약서 명시 필수
- 지적재산권: 포스팅 소유권은 블로거 개인 귀속이 일반적. 광고주 무단 수정·재게시 불가
- 분쟁 예방 조항:
  - 포스팅 삭제 요구 시 처리 방법 명시 (블로거 자체 삭제 시 재게시 불가 원칙)
  - 리뷰 톤·부정적 내용 포함 시 처리 기준 사전 명시
  - 6개월 게재 유지 의무 조항 (레뷰 표준)

### G. 리스크
- 뒷광고 공정위 리스크: 중~고. 표시 위치 미준수(본문 맨 앞 아닌 경우) 적발 빈번. 과징금 대상
- 의료광고 심의: 해당없음 (음식점)
- 네이버 저품질 제재: 중. 과도한 상업성·협찬 미표기 포스팅 검색누락 처리. 블로거 블로그 전체 저품질 판정 시 포스팅 노출 소멸
- 고객 매장 평판 리스크: 블로거가 부정적 체험 시 부정 리뷰 게재 가능. 삭제 요청 법적 강제 불가
- 환불·클레임: 포스팅 미납품 시 플랫폼 재진행 또는 환불. 품질 불만 시 환불 어려움

- 나머지: 카드 1과 동일

### G. 리스크
- 뒷광고: 중. 블로그+인스타 둘 다 표시 의무. 위반 채널 수 2배 → 제재 노출 2배

- 나머지: 카드 1과 동일

### G. 리스크
- 뒷광고: 중~고. 반복 포스팅으로 표시 누락 누적 위험 상승

- 계약 장기화 리스크: 효과 없는 경우에도 위약금 때문에 유지 강제 상황 발생 가능

### 리안에게: 피부과·성형외과 판매 가능성 판정

**현재 상태: 판매 보류**

이유 3가지:
1. 의료법 제57조 사전심의 필수 — 포스팅 1건도 심의 없이 집행 시 형사처벌(3년 이하 징역) 가능
2. 의료법 제27조 환자 유인 금지 — 무료 시술 제공으로 후기 작성 유도 = 환자 유인 해당 해석 가능. 체험단 형태 자체가 위법 소지
3. 에이전시 공동 책임 — 대행사가 광고를 설계하고 진행한 경우 에이전시도 공모·방조 해석 가능

판매 가능하려면:
- 의료광고 전문 법무 검토 완료
- 심의 대행 서비스 패키지 구성 (에이전시가 직접 심의 대행 → 통과 후 집행)
- 계약서에 "의료기관 법적 책임 귀속, 에이전시 면책" 조항 삽입
- 현재 에이전시 역량으로 가능한지 별도 검토 필요

**치과**: 동일 구조. 심의 대행 포함 패키지가 갖춰지면 판매 가능 가능성 있음. 법무 검토 병행 필수.

**네일·마사지·피부관리실·헤어샵**: 즉시 판매 가능. 의료광고법 적용 없음. 표시광고법만 준수하면 됨. 이게 경쟁 에이전시 대비 차별화 포인트 — "의료법 해당 없는 뷰티 체험단은 빠르게 시작 가능"을 세일즈 포인트로 쓸 수 있음.
