## 카드 1. 인지도 캠페인 광고 대행

### A. 돈
- 시장 판매가 (운영 대행료): 월 광고비의 10~20% 마크업. 고정 월 35만~100만 원 (소규모)
- 운영 대행료 범위: 월 35만~300만 원 (광고비 규모 비례)
- 최소 집행 광고비: 월 100만~200만 원 (인지도 목적, 도달 축적 필요)
- 픽셀 CAPI 세팅: 해당 없음 (인지도 캠페인은 전환 추적 불필요)
- 결제 구조: 광고비 선결제(메타 직접) + 대행료 월 청구. 부가세: 대행료 10% 별도. 메타 광고비는 사업자번호 등록 시 부가세 면제
- 출처: [아이보스 수수료 현황](https://www.i-boss.co.kr/ab-2110-6771), [메타 광고비 부가세](https://onlyfashionmkt.blog/entry/%ED%8E%98%EC%9D%B4%EC%8A%A4%EB%B6%81-%EC%9D%B8%EC%8A%A4%ED%83%80-%EA%B4%91%EA%B3%A0%EB%B9%84-%EB%B6%80%EA%B0%80%EC%84%B8-%EB%82%B4%EC%A7%80-%EB%A7%88%EC%84%B8%EC%9A%94-feat-%EB%B9%84%EC%9A%A9-%EA%B2%BD%EB%B9%84-%EC%B2%98%EB%A6%AC)

### B. 실행
- 실행 주체: 에이전시 직접 운영 (메타 비즈니스 매니저 내 캠페인 세팅)
- 필요 역량: 메타 비즈니스 매니저, 광고소재 제작 또는 소재 수령
- 광고 소재 제작: 미포함이 기본. 별도 과금 (이미지 소재 10만~30만 원/건, 영상 50만~150만 원/건)
- 실행사 후보: AMPM Global(ampm.co.kr), 메타애드(meta-ad.com), 심팩트(simpact.co.kr), 애즈그램(azgram.co.kr), 원플랜마케팅(oneplan.co.kr)
- 리드타임: 계약 후 5~10일 (소재 수령 완료 기준)
- 납품 결과물: 광고 매니저 접근 권한 공유, 월 성과 리포트 (도달 수, 노출 수, CPM, 빈도)
- 고객 자원: 메타 비즈니스 매니저 접근 권한, 광고비 결제 카드, 브랜드 소재(로고, 이미지)
- 적합 업종: 신규 오픈 맛집·카페, 지역 헬스장, 신설 학원 — 인지 확보가 선행 필요한 업종
- 부적합 업종: 즉시 전환이 목표인 업종 (인지도 단독으로 ROAS 측정 불가)

### C. 계약
- 최소 계약 기간: 3개월 (데이터 학습 주기 기준. 업계 표준)
- 해지 조건: 월 단위 해지 가능한 곳 vs 3개월 락인 혼재. 계약서 명시 필수
- KPI 정의: 노출 수(Impressions), 도달 수(Reach), CPM, 빈도(Frequency). 전환 추적 없음
- 재작업 한도: 소재 수정 월 1~2회 (대행사별 상이)
- 광고비: 별도 (메타 직접 결제 또는 대행사 경유 후 정산)
- 분쟁 예방: 광고 매니저 접근 권한을 광고주 계정에 귀속. 대행사가 계정 소유하는 구조는 클레임 다발 — 계약서에 "광고주 계정 소유" 명시 필수

### G. 리스크
- 광고 심사 반려: 과장 표현, 개인 신체 부정 표현 포함 시 반려. 의료·성형·다이어트 소재 고위험
- 계정 정지: 반려 누적 시 광고 계정 제한 또는 정지. 복구에 1~4주 소요
- KPI 모호성: 인지도 캠페인은 매출 기여 직접 측정 불가 — 성과 분쟁 발생 가능
- 업종 심의: 의료·금융·부동산은 별도 메타 정책 + 한국 의료광고 사전심의 대상 (2024 서울행정법원 판결: 플랫폼 기준 10만 명 이상 → 인스타그램 포함)
- 출처: [메타 광고 반려 사례](https://inside.ampm.co.kr/insight/10696), [의료광고 심의](https://myoungjae.co.kr/lawguidedetail/KtAPBaYgOUibreOUkglr)

## 카드 2. 트래픽 캠페인 광고 대행

### A. 돈
- 시장 판매가: 월 대행료 광고비의 10~20%. 고정 35만~100만 원 (소규모)
- 운영 대행료 범위: 월 35만~300만 원
- 최소 집행 광고비: 월 100만~200만 원 (CPC 최적화 학습 필요)
- 픽셀 CAPI: 해당 없음 (클릭 기반 측정, 전환 최적화 없음)
- 결제 구조: 동일 (광고비 별도, 대행료 부가세 10% 별도)

### B. 실행
- 실행 주체: 에이전시 직접 운영
- 광고 소재 제작: 미포함 기본. 이미지 별도 10만~30만 원/건
- 리드타임: 5~10일
- 납품 결과물: 월 성과 리포트 (클릭 수, CPC, CTR, 랜딩 방문 수)
- 고객 자원: 메타 비즈니스 매니저, 광고비 카드, 랜딩페이지(또는 네이버플레이스·카카오채널 URL)
- 적합 업종: 예약제 음식점, 뷰티샵, 학원 입학 안내, 인테리어 상담 업종

### C. 계약
- 최소 계약 기간: 3개월
- KPI 정의: 링크 클릭 수, CPC (클릭당 비용), CTR (클릭률), 랜딩 방문 수
- 광고비: 별도

### G. 리스크
- 랜딩페이지 품질 문제: 클릭은 나와도 랜딩에서 이탈 시 대행사 책임 회피 가능. 랜딩 품질은 계약 범위 외로 처리되는 경우 많음
- 픽셀 미설치: 전환 추적 불가로 최적화 한계 — 트래픽 캠페인 단독으로는 ROAS 측정 불가
- 광고 심사 반려: 클릭 유도 문구 과잉 시 반려

## 카드 3. 전환 캠페인 광고 대행

### A. 돈
- 시장 판매가: 월 대행료 광고비의 15~20% (전환 캠페인은 기술 난도 높아 상단 적용). 고정 50만~200만 원
- 운영 대행료 범위: 월 50만~400만 원
- 최소 집행 광고비: 월 200만~300만 원 (메타 학습 완료 기준 — 세트당 주 50건 전환 달성에 필요한 예산)
- 픽셀 CAPI 세팅: 원타임 30만~100만 원 별도 (자사 사이트 직설치 기준). 아임웹·카페24 공식 CAPI 파트너로 설정 간소화 가능
- 결제 구조: 광고비 별도. 초기 세팅비(픽셀+CAPI) 원타임. 월 대행료 부가세 10% 별도
- 출처: [CAPI 가이드](https://blog.ab180.co/posts/facebook-conversion-api-1), [아임웹 CAPI 설정](https://imweb.me/faq?mode=view&category=29&category2=47&idx=71239)

### B. 실행
- 실행 주체: 에이전시 직접 운영
- 필요 역량: 픽셀 설치, CAPI 연동, EMQ(이벤트 매칭 품질) 점수 8.0 이상 유지
- 광고 소재 제작: 미포함 기본. 별도 과금
- 리드타임: 7~14일 (픽셀+CAPI 세팅 포함)
- 납품 결과물: 월 성과 리포트 (전환 수, CPA, ROAS, EMQ 점수)
- 고객 자원: 메타 비즈니스 매니저, 결제 카드, 자사 사이트 관리자 접근(픽셀 설치용)
- 적합 업종: 학원(체험 수업 신청), 피부·헤어샵(첫 방문 예약), 헬스장(무료 체험 등록), PT센터
- 부적합 업종: 픽셀 설치 불가 사이트(네이버 예약 단독) — B-1 상품으로 분기

### C. 계약
- 최소 계약 기간: 3개월 (학습 완료 최소 기간)
- KPI 정의: 전환 수, CPA (전환당 비용), ROAS
- 분쟁 예방: 전환 이벤트 정의를 계약서에 명시 ("예약 완료", "폼 제출" 등 구체적 이벤트명 기재)

### G. 리스크
- 픽셀 CAPI 설치 실패: 전환 데이터 40% 유실 가능 (픽셀 단독 기준). CAPI 병행 필수
- 학습 미완료: 초기 2~4주 데이터 불안정 — 이 기간 성과로 계약 해지 요구 분쟁 가능
- 의료 업종 심의: 성형·피부과·한의원 전환 광고는 메타 정책 + 한국 의료광고법 이중 규제

## 카드 4. 리드 생성(잠재 고객) 광고 대행

### A. 돈
- 시장 판매가: 월 광고비의 10~20%. 고정 35만~150만 원
- 최소 집행 광고비: 월 100만~200만 원
- 픽셀 CAPI: 해당 없음 (메타 내부 이벤트로 리드 폼 제출 자동 집계)
- 결제 구조: 광고비 별도. 대행료 부가세 10% 별도

### B. 실행
- 실행 주체: 에이전시 직접 운영
- 필요 역량: 메타 즉시 작성 폼 설계, 질문 항목 기획
- 광고 소재 제작: 미포함 기본
- 리드타임: 5~10일 (폼 설계 포함)
- 납품 결과물: 월 성과 리포트 (리드 수, CPL — 리드당 비용), 수집된 리드 DB 엑셀
- 고객 자원: 메타 비즈니스 매니저, 결제 카드, 수집 개인정보 활용 동의 문구(법적 필수)
- 적합 업종: 피부관리·학원·리모델링·법무·세무·웨딩

### C. 계약
- 최소 계약 기간: 3개월
- KPI 정의: 리드 수, CPL (리드당 비용)
- 분쟁 예방: "리드 품질"은 계약 범위 외임을 계약서 명시 (리드 수집 ≠ 계약 성사 보장)

### G. 리스크
- 리드 품질 저하: 무관심 리드 다수 포함 가능. 상담 연결률 30~50%가 현실적 수준
- 개인정보보호법: 수집된 개인정보 활용 목적 명시·동의 받지 않으면 법적 리스크
- 폼 완료율 저하: 질문 항목 과다 시 리드 이탈

## 카드 5. 매장 방문(로컬 도달) 캠페인 광고 대행

### A. 돈
- 시장 판매가: 월 광고비의 10~20%. 고정 35만~150만 원
- 최소 집행 광고비: 월 100만~200만 원 (로컬 반경 도달 최적화 필요)
- CAPI+Signals Gateway 세팅: 원타임 50만~150만 원 별도 (실방문 측정 연동 포함 시). 단, 소규모 매장은 클릭 기반만으로 운영하는 경우 많음
- 결제 구조: 광고비 별도. 세팅비 원타임. 월 대행료 부가세 10% 별도

### B. 실행
- 실행 주체: 에이전시 직접 운영
- 필요 역량: 메타 위치 자산 등록, 반경 타겟 설정, CAPI 연동 (선택)
- 광고 소재 제작: 미포함 기본. 지도 CTA 포함 이미지 별도
- 리드타임: 7~14일 (위치 자산 등록 포함)
- 납품 결과물: 월 성과 리포트 (길찾기 클릭, 전화 클릭, 추정 방문 수)
- 고객 자원: 매장 주소, 메타 비즈니스 매니저, 결제 카드
- 적합 업종: 카페·음식점, 미용실, 헬스장·필라테스, 학원, 네일샵
- 부적합 업종: 온라인 전용 사업, 방문 없는 배달 전용 업종

### C. 계약
- 최소 계약 기간: 3개월
- KPI 정의: 길찾기 클릭 수, 전화 버튼 클릭 수. (주의: 메타 "추정 매장 방문 수"는 과대계상 논란 있음 — 클릭 기준으로 계약서 KPI 설정 권장)
- 분쟁 예방: KPI를 "추정 방문 수"로 설정 시 메타 데이터 신뢰도 논란으로 분쟁 가능. "클릭 수" 기준으로 명시

### G. 리스크
- CAPI 기술 장벽: 소규모 자영업자 CAPI 직설치 어려움. 세팅 비용 부담으로 실방문 측정 포기 사례 많음
- Store Visit Lift 신뢰도: 국내 소규모 자영업자 사이 과대계상 논란 다수 확인
- 2025-05 구API 폐기: 기존 Offline Conversions API 세팅 무효화 — 기존 계약 재협상 필요
- 출처: [오프라인 이벤트 측정](https://marketinkerbell.com/entry/메타-광고-오프라인-이벤트-란-오프라인-구매-측정-방법), [CAPI 1편](https://blog.ab180.co/posts/facebook-conversion-api-1)

## 카드 6. 릴스 광고 대행 (세로형 숏폼 광고)

### A. 돈
- 시장 판매가: 월 광고비의 10~20% + 소재 제작비 별도. 또는 통합 패키지 (소재+집행) 월 100만~400만 원
- 운영 대행료: 월 35만~200만 원 (소재 미포함 기준)
- 최소 집행 광고비: 월 100만~200만 원

### 대행료 % 구조 (업계 관행)
한국 메타 광고 대행 시장은 "광고비 + 대행료" 분리 구조가 표준이다. 대행료는 두 가지 방식이 혼재한다.

1. 마크업(%) 방식: 광고비의 10~20%. 페이스북·구글 등 해외 매체는 국내 매체처럼 대행사에게 수수료를 지급하지 않으므로, 대행사가 광고주에게 직접 "마크업"으로 청구한다. 15%가 업계 시작점. 능력과 협상에 따라 조정된다.
2. 고정 월 관리비 방식: 광고비 규모에 무관하게 월 35만~300만 원 고정. 소규모 자영업 대상 프리랜서·소형 대행사에서 많이 사용.
