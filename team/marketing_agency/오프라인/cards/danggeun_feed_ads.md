## 시장 특성 요약

- 크몽 프리랜서 주류. 당근 전문 에이전시 독립 운영 사례 거의 없음 (네이버·메타 전문 에이전시의 부가 채널로 편입)
- 1회 결제 + 광고비 별도 충전 구조 vs 월 고정 대행료 구조 공존. 크몽 TOP 셀러(아이로즈 4,410건)는 1회 결제형 주력
- 2026.01.01 당근 광고 심사 정책 변경 공지 확인. 의료 업종 SNS 광고 사전심의 의무 강화 (일평균 이용자 10만 이상 플랫폼 전체 적용 → 당근 해당)
- 대행료 투명성 이슈: 다수 셀러가 "수수료 0원, 광고비 직접 충전" 구조를 전면 내세움 (소상공인의 광고비 불투명 불신 반영)
- CPC 평균: 서울·경기 기준 300~600원 (간편모드 기준), 전문가모드 최소 200원, 강남권 500원+ (출처: 커넥트리, 제이영컨설팅)
- 전문가모드 PC 전용 운영 + 사업자 등록번호 필수 (대행사가 계정 대리 운영 시 주의)

## 카드 1. 간편모드 피드광고 (3차 SKU)

### A. 돈
- 시장 판매가 (크몽 기준):
  - 초기 세팅 단발형: 80,000~110,000원 (아이로즈 STANDARD/DELUXE)
  - 단발 진단 + 피드백: 10,000원 (우강마케팅 STANDARD — 진단만)
  - 소재 포함 종합형: 200,000~300,000원
- 결제 구조: 1회 결제 (세팅비) + 광고비 별도 직접 충전
- 광고비 직접 충전: 당근 캐시로 광고주가 충전. 대행료에 포함 안 됨
- 월 광고비 권장: 최소 30만 원 (크몽 공식 안내)
- CPC 단가: 300~600원 (서울·경기 기준)
- 최소 일예산: 1~3만 원 (일반 지역 자영업 기준)

### B. 실행
- 실행 주체: 크몽 프리랜서 개인 셀러가 주류. 에이전시 직접 운영 드뭄
- 실행사 후보:
  1. 아이로즈 (kmong.com/gig/566030) — 판매 4,410건, 리뷰 122건. 크몽 최다 판매. 전문가모드 특화. STANDARD 80,000원
  2. 우강마케팅 (kmong.com/gig/634423) — 판매 557건, 리뷰 71건. 병원·법무·학원 다업종. 200,000원
  3. 루카쿠 (kmong.com/gig/486969) — 판매 563건, 리뷰 1건. 소재 제작 포함, 전국 지역광고
  4. 너드마케팅 (kmong.com/gig/268197) — 판매 2,827건, 리뷰 1건. 299,000~499,000원
  5. 매출공장 (kmong.com/gig/459578) — 판매 31건, 9년차 마케터, STANDARD 99,000원
- 전문가모드 권한 보유 여부: 간편모드는 별도 권한 불필요. 사업자+비즈프로필로 즉시 운영 가능
- 리드타임: 세팅 후 당일~2일 내 라이브 (심사 통과 기준)
- 납품 결과물: 광고 매니저 세팅 완료, 월별 성과 보고(셀러에 따라 다름)
- 고객 자원: 당근 비즈프로필 계정, 당근 캐시 충전 수단, 소재(이미지/소식 게시물)
- 적합 업종: 단일 매장 음식점·카페·미용실·생활 서비스 소상공인
- 부적합 업종: 외부 랜딩 연결 필요 업종 (간편모드는 외부 URL 연결 불가)

### C. 계약
- 최소 계약 기간: 단발(1회) 가능. 평생 관리 구조 시 별도 계약 없음
- 해지 조건: 1회 결제형 — 해지 개념 없음. 계정 관리 중단 시 광고주가 직접 중지
- KPI 정의: 노출수, 클릭수, CPC (전환 추적은 전문가모드에서만 가능 — 간편모드 미지원)
- 재작업 한도: 셀러별 상이. 아이로즈 DELUXE — 소식글 1건 포함, A/B 테스트 포함
- 광고비 투명성: "수수료 0원, 광고비 직접 충전" 구조가 시장 표준으로 자리잡는 중. 월% 수수료 구조는 에이전시 중형 이상에서만 존재

### G. 리스크
- 간편모드 타겟팅 제한: 광고주 위치 동네 기반만 노출. 지역 확장 불가 → 기대 이하 노출 발생 가능
- 성과 데이터 제한: 노출·클릭만 제공. 전환(예약·구매) 추적 불가 → ROI 증명 어려움
- 계정 전환 리스크: 3개월 이상 당근 캐시 미충전 시 대행 권한 박탈 가능 (커넥트리 확인)
- 소재 의존: 소상공인이 소재 없을 경우 기존 소식 게시물로 대체 — 품질 저하 가능

- 실행 주체: 크몽 프리랜서 개인 셀러. 에이전시는 메타·네이버 병행 채널로만 운영

  3. 루카쿠 (kmong.com/gig/486969) — 기업 전담팀 운영 명시. 소재 제작 포함
- 전문가모드 권한 보유: 사업자 등록번호 입력 후 당근 공식 ads-pro.business.daangn.com에서 직접 생성 가능. PC 전용

- 납품 결과물: 광고 매니저 접근권 부여 또는 월 리포트 (엑셀 다운로드 가능), 전환 추적 픽셀 설치 가능

- 부적합 업종: 의료 업종 (2026.01 심사 정책 강화로 사전심의 필수)

### C. 계약
- 최소 계약 기간: 단발 가능 (1회 세팅), 월 관리형은 1개월 테스트 가능 (매출공장)

- KPI 정의: 노출, CTR, CPC, 방문(픽셀 설치 시 전환 추적 가능)
- 재작업 한도: 미확정 (셀러별 상이, 명시 없음)
- 광고비 투명성: 전문가모드 광고주 직접 충전 구조 동일. 대행료 별도 청구

### G. 리스크

- 외부 랜딩 URL 심사: 당근 광고 소재 심사 시 랜딩 URL도 함께 심사. 심사 반려 위험
- 전문가모드 전환 후 캠페인 구조 재설계 필요 — 간편모드 기존 데이터와 연계 불가

- 실행사 후보:
  1. 아이로즈 — 일 2회 계정 모니터링 명시. 월 1회 전략 리뷰 포함 (DELUXE 기준)

  3. 매출공장 — 30일 풀서비스 (컨설팅+세팅+운영)
- 관리 주기: 월 1회 입찰가 조정 + 월말 성과 보고
- 최적화 항목: CPC 입찰가 월 1회, 소재 교체 판단, 타겟 세분화 조정
- 납품 결과물: 월 성과 리포트, 광고 매니저 접근 (셀러별 상이)

- 해지 조건: 월 단위 해지 가능 (월 관리형). 1회 결제형은 해지 없음
- KPI 정의: CPC 절감율, 월간 CTR, 월간 클릭수, 노출수. 전환은 픽셀 설치 시 추가
- 재작업 한도: 미확정
- 광고비 투명성: 광고비 직접 충전 구조. 수수료율 공개 셀러와 비공개 셀러 혼재

### G. 리스크
- "평생 관리" 실제 이행 불확실성: 셀러 크몽 탈퇴 또는 계정 삭제 시 관리 중단. 크몽 플랫폼 의존 → 분쟁 시 환불 불가 가능성
- 월 관리 성과 미보장: 크몽 셀러 중 KPI 미달 시 환불/재작업 정책 미명시 사례 존재
- 시장 미성숙: "당근 CPC 월 관리" 독립 상품 에이전시급 5곳 이상 미확인 → 수준 편차 큼

## 카드 5. 전문가모드 CPC 최적화 관리형 — 일일 관리 (4차 A-3)

  - 고강도 일일 운영 월 대행료 추정: 300,000~500,000원/월 (시장 미성숙으로 추측치)
- 결제 구조: 월 고정 대행료가 현실적. 1회 결제 후 일일 관리는 구조적으로 불안정

- 실행 주체: 크몽 프리랜서 (아이로즈가 사실상 유일한 "일 2회 모니터링" 명시 셀러)
- 실행사 후보:
  1. 아이로즈 (kmong.com/gig/566030) — "매일 1일 2회 광고 계정 접속" 명시
- 관리 주기: 일 1~2회 계정 확인, 이상 시 즉시 입찰가·예산 조정
- 최적화 항목: 입찰가 일 단위 조정, 시간대별 예산 배분, 소재 긴급 교체
- 적합 업종: 병원(예약 마감 관리), 음식점(런치/디너 시간대), 부동산(매물 즉시성)
- 실제 CPC 절감 성과: 타이밍 최적화 CPC 18% 감소 + 지역 세분화 15% 추가 감소 = CPC 약 30% 절감, ROAS 2배 이상 개선 (마케팅 인사이드 케이스 스터디, inside.ampm.co.kr/insight/12119)
