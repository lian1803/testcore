# CPO 분석 -- 소상공인 비대면 영업 퍼널 전략 재설계

> 작성일: 2026-03-27
> 목적: 방문 미팅 중심 -> 비대면 텍스트 클로징 중심으로의 전환 전략
> 대상: 리안 (1인 영업자, 경기 북부, 네이버 플레이스 마케팅 대행)

---

## 제품 전략

대량 DB가 확보된 지금, 영업의 병목은 "연락처 확보"가 아니라 "텍스트만으로 계약을 성사시키는 구조"다.
방문 미팅은 시간당 1건이지만, 텍스트 클로징은 시간당 10건 이상 동시 진행이 가능하다.
핵심 포지셔닝: **"만나지 않고도 사장님이 돈 내게 만드는 퍼널"을 설계하고 자동화한다.**

---

## 타겟 + Pain

- **타겟**: 경기 북부 소상공인 (미용실, 네일, 식당, 카페, 피부관리, 학원)
- **Pain**: "네이버에서 왜 내 가게가 안 나오지?" -- 검색 순위가 낮아 손님을 놓치고 있지만, 뭐가 문제인지 모르고, 어디서부터 손대야 할지 모른다
- **아하 모먼트**: 진단 결과를 받고 "아, 내가 이런 상태였어?"라고 느끼는 순간. 이때 문제를 인식하면 해결책(= 우리 서비스)에 대한 수요가 자동으로 생긴다

## 1. 비대면 영업 퍼널 전체 재설계

### 전체 구조 (6단계)

```
[0단계] 타겟 선별 (자동)
    DB에서 영업 우선순위 자동 분류 -> 1순위만 추출
    ↓
[1단계] 1차 접촉 — 충격 1개 던지기
    "보내드려도 될까요?" -> 답장 유도
    ↓ 답장 오면 (5~10%)
[2단계] 관심 유발 — 진단 결과 텍스트 발송
    문제를 시각적으로 보여준다. 해결법은 안 준다.
    ↓ 반응 오면 ("어떻게 해요?")
[3단계] 설득 — 패키지 + 손익분기 제시
    "9명만 더 오면 본전" -> 결정 요청
    ↓ YES / 보류 / 거절
[4단계] 클로징 — 계약 처리
    글로싸인 + 입금 안내 -> 60초 안에 완료
    ↓ 보류/무응답이면
[4-B] 재촉진 — 마감 압박 1회
    독점/마감/손실 강조 -> 마지막 기회
    ↓
[5단계] 포기 or 방문 전환
    텍스트로 안 되면 -> 방문 미팅 제안 (선별적)
```

### 각 단계별 소요 시간 (리안 기준)

| 단계 | 소요 시간 | 동시 진행 가능? |
|------|-----------|----------------|
| 0단계 타겟 선별 | 0분 (자동) | - |
| 1단계 1차 접촉 | 업체당 30초 (복붙) | 30건/시간 |
| 2단계 진단 발송 | 업체당 1분 (복붙) | 답장 온 것만 |
| 3단계 패키지 제시 | 업체당 2분 | 반응 온 것만 |
| 4단계 클로징 | 업체당 3분 | 계약 의향 있는 것만 |
| 4-B 재촉진 | 업체당 30초 (복붙) | 일괄 |
| 5단계 방문 | 1건당 2~3시간 | 1건씩 |

**일일 최대 처리량**: 1~2단계 50건 + 3단계 5~7건 + 4단계 1~2건 = 약 3시간

## 2. 결과물 공개 범위 원칙 (핵심)

### 대원칙: "문제는 확실히, 해결법은 절대 안 준다"

사장님이 스스로 해결할 수 있다고 느끼면 계약 안 한다.
"문제가 심각하다"는 느끼되 "어떻게 해야 하는지는 모르겠다"는 상태를 만들어야 한다.

### 단계별 공개/비공개 매트릭스

```
                    1차        2차        3차        4차       계약 후
종합 점수/등급       X          O          O          O         O
항목별 점수          X          O          O          O         O
경쟁사 리뷰 격차     O(숫자만)   O          O          O         O
손실 고객 추정       O(1개만)   O          O          O         O
취약 항목 목록       X          O(뭐가비어) O          O         O
취약 항목 해결법     X          X          X          X         O <<<
경쟁사 구체 전략     X          X          X          X         O <<<
순위 올리는 방법     X          X          X          X         O <<<
키워드 전략 상세     X          X          X          X         O <<<
개선 전후 예상 점수  X          X          O(대략)    O         O
패키지 가격          X          X          O          O         O
손익분기 계산        X          X          O          O         O
PPT 제안서           X          X          필요시     O         O
독점/마감 압박       X          X          X          O         -
```

### 왜 이 순서인가

**1차에서 던지는 것: 충격 숫자 딱 1개**
- 이유: 첫 메시지가 길면 안 읽는다. 숫자 하나만 던져서 "어?" 하게 만든다.
- 공개: 경쟁사 리뷰 격차 OR 손실 고객 수 OR 방치 일수 중 1개만
- 비공개: 나머지 전부. 궁금하게 만들어야 답장이 온다.

**2차에서 보여주는 것: 문제의 전체 그림**
- 이유: 1차에 답장 온 사람은 이미 관심이 있다. 이제 문제의 심각성을 시각적으로 보여준다.
- 공개: 종합 점수, 항목별 현재 상태, 경쟁사 격차 숫자, 취약항목 목록
- 비공개: 해결 방법, 패키지, 가격. "어떻게 해요?"를 유도해야 한다.
- 마무리 멘트: "어떻게 생각하세요?" (열린 질문으로 끝낸다)

**3차에서 제시하는 것: 솔루션 + 가격 + 결정 근거**
- 이유: 상대가 "어떻게 해요?"를 물었다 = 구매 의향 신호. 이제 팔아야 한다.
- 공개: 패키지 3가지 + 추천 패키지 + 손익분기 계산
- 비공개: 구체적 실행 방법 (그건 계약 후에)
- 마무리 멘트: "한번 해보실 의향 있으세요?" (Yes/No 질문으로 끝낸다)

## 3. 비대면 텍스트 클로징 구조 — "만나자 없이 계약 성사"

### 3-1. 텍스트만으로 신뢰를 형성하는 5가지 장치

#### (1) 데이터가 곧 신뢰다
- 사장님 업체의 실제 숫자를 보여준다 (리뷰 116개, 경쟁사 2,636개)
- 이건 내가 조사했다는 증거이자, 전문성의 증거다
- "아무한테나 보내는 영업 아니에요"를 숫자가 증명한다

#### (2) 먼저 무료로 준다 (호혜성)
- 진단 결과를 무료로 보내준다
- 받기만 한 사람은 심리적 빚을 진다
- "무료로 이만큼 해줬는데, 유료면 뭘 더 해주려나" 기대를 만든다

#### (3) 전문가 포지셔닝
- 메시지 톤: 조언자, 컨설턴트 톤 (판매자 톤 절대 금지)
- "저희가 분석해봤는데요" / "이 부분이 좀 아쉽더라고요"
- 절대 "저희 서비스 좋아요" 같은 말 안 한다

#### (4) 사회적 증거 (Social Proof)
- "같은 지역 미용실 중 문의 주신 곳이 있어서요"
- "이번 달 신규 N곳만 더 받아요"
- 내가 유일한 고객이 아니라는 안심감

#### (5) 손실 회피 (Loss Aversion)
- "매달 N명이 경쟁사로 가고 있어요" > "N명을 더 모을 수 있어요"
- 인간은 얻는 것보다 잃는 것에 2배 민감하다
- 모든 메시지에서 "지금 잃고 있는 것"을 먼저 말한다

### 3-2. 가격 저항 극복 4가지 전략

#### 전략 1: 앵커링 — 시선 플랜(89만원)을 먼저 보여준다
```
패키지 나열 순서:
  주목 29만원 (싸게 느낌)
  집중 49만원 (적당해 보임) << 추천
  시선 89만원 (비싸 보임 = 앵커)
```
이 순서가 아니라 역순으로 보여주는 것도 방법이지만, 소상공인 특성상 아래에서 위로 읽으므로 오름차순이 맞다. 대신 시선 플랜을 별 표시로 눈에 띄게 하면, 먼저 89만원을 인지하고 49만원을 "저렴"하게 느낀다.

#### 전략 2: 손익분기로 프레이밍 전환
- "49만원 비싸요" -> "고객 9명만 더 오면 본전이에요"
- 가격을 "비용"이 아닌 "투자 대비 회수"로 바꾼다
- 업종별 객단가 기반으로 자동 계산해서 보여준다

#### 전략 3: 기회비용 강조
- "안 하는 것도 비용이에요. 매달 N명이 경쟁사로 가고 있는 거니까요"
- 서비스 비용 vs 매달 놓치는 매출을 비교한다
- 놓치는 매출이 서비스 비용보다 크면 결정은 쉽다

#### 전략 4: 시간 프레이밍
- "3개월만 해보세요. 효과 없으면 그때 말씀해주세요"
- 무기한 계약이 아닌 단기 시도 느낌을 준다
- 실제로 3개월 해보면 효과가 보이기 시작하고 해지율이 낮다

### 3-3. "생각해볼게요" 대응 매뉴얼

"생각해볼게요"는 거절이 아니다. 결정을 미루는 것이다.
이유별로 대응이 다르다.

| 숨은 이유 | 판별 신호 | 대응법 |
|-----------|-----------|--------|
| 비싸서 | "가격이..." / 망설임 | 손익분기 재계산 + "안 하는 게 더 비싸요" |
| 효과 의심 | "진짜 되나요?" | "3개월만, 안 되면 말씀해주세요" + 수치 근거 |
| 직접 하고 싶어서 | "혼자 해볼게요" | "X항목 7개, 매주 N시간 드는데 그 시간에 손님 보세요" |
| 지금 안 급해서 | "좀 있다가..." | 2~3일 후 마감/독점 메시지 1회 발송 |
| 진짜 관심 없어서 | 1~2자 답장 / 무시 | 4차 1회 후 포기. 3개월 뒤 재진단 후 재접촉 |

**핵심 원칙: "생각해볼게요" 후 3일 이내에 딱 1번만 추가 접촉. 그 이후는 포기.**
끈질기게 하면 스팸이 되고, 추후 재접촉 기회까지 날린다.

### 3-4. 계약 처리 프로세스

비대면 계약은 "결정의 순간"에서 "돈 내는 순간"까지의 거리가 짧아야 한다.
방문 미팅은 현장에서 서명하지만, 텍스트는 링크만 보내면 끝이어야 한다.

```
사장님 "할게요" 답장
    ↓ (즉시, 5분 이내)
글로싸인 링크 + 입금 안내 발송
    ↓
사장님이 핸드폰으로 서명 (3분)
    ↓
입금 확인 (카카오페이 or 계좌이체)
    ↓
"서명 + 입금 확인됐습니다. 세팅 시작합니다" 확인 메시지
    ↓
온보딩 시작
```

**결제 수단 우선순위:**
1. 카카오페이 송금 링크 (가장 빠름, 10초면 결제)
2. 계좌이체 (전통적, 소상공인이 익숙함)
3. 토스페이/네이버페이 (선택지로 제공)

**입금 미확인 대응:**
- 서명 완료 + 입금 미완 -> 다음날 오전 "입금 확인이 안 됐는데 혹시 불편하신 게 있으셨나요?" 1회
- 그래도 미입금 -> 2일 후 "이번 달 세팅 마감이 N일인데, 혹시 일정 맞추실 수 있으세요?" 1회
- 이후 미입금 -> 포기

## 4. 방문 미팅 트리거 기준

### 방문을 제안하는 조건 (아래 중 하나라도 해당하면)

| 신호 | 해석 | 방문 이유 |
|------|------|-----------|
| "직접 봐야 믿겠다" | 의심이 높음. 텍스트론 안 됨 | PPT 보여주면서 전문성 증명 |
| "시선 플랜 관심있는데..." | 89만원짜리 관심 = 고가 계약 가능 | 89만원은 대면이 계약률 높음 |
| 3차까지 긍정 반응인데 결제 안 함 | 마지막 한 방이 필요 | 현장 서명 유도 |
| "우리 가게 한번 와보세요" | 본인이 초대 | 거절할 이유 없음, 확정에 가까움 |
| 객단가 높은 업종 (학원/피부관리) | LTV가 높아서 투자 가치 있음 | 방문 1건 = 월 49~89만원 확보 |

### 방문 시 추가로 보여줄 것 (텍스트에서 안 보여준 것)

| 자료 | 왜 방문 때만? |
|------|-------------|
| PPT 전체 9슬라이드 | 텍스트로는 PPT 못 보여줌. 대면에서 한 장씩 넘기며 설명하는 게 임팩트 |
| 경쟁사 플레이스 실제 화면 | 노트북으로 실시간 비교. "여기가 1위인데 이렇게 하고 있어요" |
| 개선 전/후 시뮬레이션 | "사진 50장 올리면 이렇게 바뀝니다" -- 기대감 극대화 |
| 기존 고객 사례 (있다면) | "이 미용실이 3개월 전에 시작해서 지금 이렇게 됐어요" |
| 리안 본인의 전문성 | 대면에서 사람 자체를 신뢰하게 만든다. 텍스트론 한계가 있다 |

### 방문을 제안하지 않는 조건

- 주목 플랜(29만원) 관심 -> 방문 왕복 시간 대비 ROI 낮음. 텍스트로 끝내라
- 1~2차에서 반응 미지근 -> 방문 가봐야 안 된다. 텍스트 관심도 없는데 시간 쓰지 마라
- 프랜차이즈 -> 결정권자가 아닐 확률 높음. 본사 확인해야 한다고 하면 끝
- 거리 편도 40분 이상 -> 시간 대비 효율 없음. 비대면으로 클로징하거나 포기

## 5. 볼륨 영업 최적화

### 5-1. DB 우선순위 분류 기준 (자동화 대상)

#### S급 (즉시 연락 -- 오늘 중)

아래 조건 3개 이상 해당:
- 등급 D (39점 이하)
- 사진 10장 미만
- 리뷰 10개 미만
- 새소식 90일+ 없음
- 키워드 순위 11위 이하 또는 없음
- 블로그 리뷰 5개 미만
- 사장님 답글 0%

**이유**: 문제가 명확하고 진단 결과가 충격적. 1차 메시지 임팩트가 가장 크다.

#### A급 (이번 주 연락)

아래 조건 2개 이상 해당:
- 등급 C (40~59점)
- 경쟁사 대비 리뷰 1/3 이하
- 사진 20장 미만
- 키워드 4~10위 (아슬아슬 -- "조금만 올리면" 설득 가능)

**이유**: 약점이 있지만 S급만큼 심각하진 않다. 설득에 시간이 좀 더 걸린다.

#### B급 (여유 있을 때)

- 등급 B (60~79점)
- 특정 항목 1~2개만 약한 경우
- 개선하면 A로 갈 수 있는 업체

**이유**: 니즈를 직접 못 느끼고 있을 확률 높음. 응답률 낮지만 계약 시 유지율 높다.

#### 패스 (연락 안 함)

- 등급 A (80점 이상) -> 이미 잘하고 있어서 필요성 못 느낌
- 최신 리뷰가 1년 이상 전 -> 폐업 가능성
- 프랜차이즈 본사 직영 -> 본사에서 마케팅 담당
- 리뷰에 "폐업", "이사" 등 키워드 -> 영업 무의미

### 5-2. 하루 발송량 최적화

| 지표 | 수치 | 근거 |
|------|------|------|
| 하루 1차 발송 | 30건 | 25건 미만 = 파이프라인 마른다. 50건 이상 = 응답 관리 불가 |
| 발송 시간대 | 오전 10~11시 | 영업 준비 시간, 폰 확인 빈도 높음 |
| 답장 대응 시간 | 1시간 이내 | 답장 후 2시간 지나면 관심 50% 감소 |
| 팔로업 (무응답) | 3일 후 1회 | 2회 이상은 스팸 인식 |
| 동시 진행 파이프라인 | 최대 10건 | 3~4단계에 있는 건이 10건 넘으면 관리 불가 |

**일일 루틴 (3시간 기준):**
```
09:30~10:00  오늘 발송할 30건 리스트 확인 + 1차 메시지 복붙 준비
10:00~11:00  1차 메시지 30건 발송
11:00~12:00  어제/그저께 답장 확인 -> 2차/3차 발송 + 클로징 진행
14:00~15:00  3일 전 무응답 건 팔로업 + 새 진단 돌리기
```

### 5-3. 메시지 타입 자동 매칭 우선순위

1차 메시지 A/B/C 타입 자동 선택 우선순위:

```
[판정 순서]
1. 경쟁사 리뷰 >= 내 리뷰 x 5  ->  A타입 (리뷰 격차)
   * 가장 임팩트 큰 숫자. "2,600개 vs 116개"는 답장 안 할 수가 없다

2. 순위 > 10 AND 월검색 >= 500  ->  B타입 (손실 고객)
   * "매달 N명이 경쟁사로" -- 손실 회피 자극

3. 새소식 >= 90일 OR 저장수 = 0  ->  C타입 (방치)
   * "6개월째 새소식 없어요" -- 관리 안 하고 있다는 자각

4. 위 3개 모두 해당 안 됨  ->  A타입 기본 (리뷰 격차 각도)
```

**A/B 테스트 권고:**
- 첫 100건은 A타입 위주로 보내고 응답률 측정
- 다음 100건은 B타입 위주로 보내고 비교
- 응답률 높은 타입을 기본값으로 세팅
- 업종별로 반응 다를 수 있음 (미용실은 리뷰, 학원은 순위에 민감)

## 6. 패키지/가격 전략 검토

### 6-1. 현재 3패키지 구조 -- 비대면에서 유효한가?

**결론: 3패키지 유지하되, 비대면용으로 미세 조정 필요.**

| 문제 | 비대면 특성 | 해결 |
|------|------------|------|
| 3개 다 설명하면 메시지가 길다 | 텍스트에서 긴 메시지 = 안 읽음 | 3차 메시지에서 3줄 요약 + 추천 1개만 강조 |
| 시선(89만원)은 텍스트론 비싸 보인다 | 대면이면 열정/전문성으로 보완, 텍스트는 숫자만 보임 | 시선은 방문 미팅 전용으로 전환 검토 |
| 주목(29만원)은 마진이 너무 낮다 | 29만원 계약에 4차까지 메시지 보내는 건 비효율 | 주목을 미끼로 쓰고, 첫 달 이후 집중으로 업그레이드 유도 |

### 6-2. 비대면 전용 퍼널 패키지 전략

```
텍스트 퍼널 기본 흐름:
  1차: 무료 진단 (미끼 -- 비용 0원)
  2차: 진단 결과 제공 (가치 증명)
  3차: 집중 플랜 추천 (49만원 -- 핵심 수익)

방문 미팅 흐름 (선별적):
  시선 플랜 제안 (89만원 -- 고가)
```

**핵심 포인트:** 비대면에서는 **집중 플랜(49만원)**을 메인으로 판다.
- 29만원은 "써봤는데 안 되더라"가 나올 수 있다 (내용이 빈약)
- 89만원은 텍스트만으로 결정하기 무겁다
- 49만원이 "해볼 만한 금액"과 "의미 있는 서비스"의 교차점

### 6-3. 첫 달 할인/무료 체험 -- 넣을 것인가?

**결론: 넣지 마라. 대신 "3개월 커밋 시 1개월 할인"으로 가라.**

| 전략 | 장점 | 단점 | 판단 |
|------|------|------|------|
| 첫 달 무료 | 진입 장벽 제거 | 무료 쓰고 해지, 매출 0원, 서비스 가치 하락 | X |
| 첫 달 50% 할인 | 시도 장벽 낮춤 | "원래 할인해주는 거 아니야?" 인식, 2달째 탈퇴 | X |
| 3개월 약정 시 1개월 할인 | 최소 3개월 확보, 효과 볼 시간 벌기 | 약정 거부감 | O (권장) |
| 효과 보장제 (무효과시 환불) | 신뢰 극대화 | 리스크 큼, 기준 애매 | X |

**추천 구조:**
```
일반: 집중 플랜 490,000원/월
3개월 약정: 집중 플랜 390,000원/월 (첫 3개월)
           -> 4개월부터 490,000원/월
```

이렇게 하면:
- "3개월만 해보세요"가 자연스러워진다
- 첫 3개월 = 117만원 (vs 147만원). 30만원 절약 어필
- 3개월 안에 효과 보면 해지 안 한다
- 손익분기 계산도 더 쉬워진다: "월 39만원이면 고객 7명만 더 오면 본전"

### 6-4. 앵커링 효과 극대화

**3차 메시지에서 앵커링 순서:**

```
1단계: 손실 금액 먼저 제시 (앵커)
   "지금 매달 약 35명이 경쟁사로 가고 있어요.
    미용실 평균 객단가 6만원이면 매달 210만원 놓치는 거예요."

2단계: 패키지 가격 제시 (앵커 대비 저렴하게 느낌)
   "집중 플랜이 월 49만원이에요."
   (210만원 놓치는 거 vs 49만원 투자)

3단계: 손익분기로 마무리
   "고객 9명만 더 오면 본전이에요.
    35명 중 9명이면 26%만 잡으면 되는 거고요."
```

이 순서로 하면 49만원이 "싸다"고 느껴진다.
절대 패키지 가격을 먼저 말하지 마라. 손실 금액을 앵커로 먼저 깔아야 한다.

## 7. 주간/월간 성과 관리

### 핵심 퍼널 지표 (매주 금요일 체크)

| 단계 | 지표 | 목표치 | 측정법 |
|------|------|--------|--------|
| 1차 발송 | 주간 발송 건수 | 150건 (30x5일) | 발송 기록 |
| 1차->2차 | 응답률 | 7~10% | 답장 수 / 발송 수 |
| 2차->3차 | 관심 전환율 | 40~50% | "어떻게 해요" 비율 |
| 3차->계약 | 계약 전환율 | 30~40% | 계약 수 / 3차 발송 수 |
| 전체 | 월간 계약 | 5건 | 계약서 기준 |
| 전체 | 월 매출 | 245만원+ | 입금 기준 |

### 역산

```
월 5건 계약을 위해:
  3차 제안 필요: 5 / 0.35 = 약 14건
  2차 발송 필요: 14 / 0.45 = 약 31건
  1차 발송 필요: 31 / 0.08 = 약 390건

월 영업일 22일 기준: 390 / 22 = 하루 18건 (최소)
안전 마진 포함: 하루 30건 발송 -> 월 660건 -> 월 5~7건 계약 기대
```

## 최종 판단

### 이번 전환의 핵심은 "볼륨"이다.

방문 미팅은 하루 2~3건이 한계다. 텍스트는 30건 이상 동시 진행이 가능하다.
같은 계약률이라도 모수가 10배면 결과도 10배다.

하지만 볼륨을 높이려면 **각 단계가 자동화**되어야 한다:
- 타겟 선별: 자동 (이미 있음)
- 진단: 자동 (이미 있음)
- 메시지 생성: 자동화 필요 (가장 급한 것)
- 발송: 수동 (리안이 직접 -- 자동 발송은 스팸법 위반)
- 응답 관리: 반자동 (트래킹 필요)
- 계약 처리: 글로싸인 + 카카오페이 (셋업 필요)

**지금 당장 리안이 할 수 있는 것:**
1. 기존 메시지 템플릿으로 내일부터 30건씩 발송 시작
2. 구글시트로 영업 트래킹 시작 (업체명/단계/날짜/메모)
3. 글로싸인 계약서 1개 만들기
4. 카카오페이 송금 링크 세팅

**시스템이 도와줘야 하는 것:**
1. 메시지 자동 생성 (복붙만 하면 되게)
2. DB 우선순위 분류 (어디에 연락할지 자동 추천)
3. 발송 이력 관리 (누구한테 몇 차까지 보냈는지)

시스템 없이도 영업은 시작할 수 있다. 하지만 시스템 없이 하루 30건을 유지하는 건 2주 안에 지친다.
**영업 시작과 시스템 개발을 병렬로 돌려라.**
