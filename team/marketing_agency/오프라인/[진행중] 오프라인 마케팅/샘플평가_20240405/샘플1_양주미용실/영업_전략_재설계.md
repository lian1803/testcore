# 리안 컴퍼니 — 양주 미용실 네이버 플레이스 영업 전략

---

## 핵심 인사이트 (자료 + 리서치 분석)

**인사이트 1 — 손실 프레이밍이 기능 설명보다 2배 강하다** (출처: 손실회피 편향 자료)
"상위노출 해드릴게요"는 팔리지 않는다. "지금 매달 557명이 경쟁사로 넘어가고 있어요"가 팔린다. 손실회피 연구에서 동일 금액이라도 손실 표현이 이익 표현보다 반응률 2배. 557명 × 평균객단가(미용실 3~5만원) = 월 1,671만~2,785만원 손실이라는 수치가 핵심 무기다.

**인사이트 2 — 신뢰 붕괴가 거절의 본질이다** (출처: 거절 이유 Top5 자료)
"비싸다"는 표면 이유다. 진짜는 "어차피 또 사기당하는 거 아니야?" 이 의심을 깨려면 업체명, 경쟁사 실명, 현재 순위 수치를 첫 메시지에서 꺼내야 한다. 모르는 척 접근하면 스팸 처리된다.

**인사이트 3 — 디지털 마케팅 활용률 15.4%는 기회이자 교육 부담이다** (출처: 경기도 소상공인 디지털 마케팅 인식 조사)
양주 미용실 사장님은 90% 확률로 네이버 플레이스 로직을 모른다. 복잡한 설명은 독이다. "13위 → 3위 → 월 557명 회수"로 단순화해서 전달해야 한다. 교육이 아니라 번역이 임무다.

**인사이트 4 — 첫 접촉에서 업체 특정 데이터를 꺼내면 무장해제된다** (출처: 사회적 증거 자료 + SPIN Selling 적용)
"안녕하세요 마케팅 대행사입니다"는 초당 삭제된다. "양주 미용실 네이버 검색하면 ○○○헤어가 1위, 사장님 매장은 현재 13위입니다"로 시작하면 사장님이 먼저 "어떻게 아세요?"를 묻는다. 이 질문이 나오는 순간 영업 주도권이 넘어온다.

**인사이트 5 — 앵커링은 89만 → 49만 → 29만 순서가 맞다** (출처: 앵커링 효과 가격 제시 전략 자료)
SaaS 기업 전환율 데이터상 고가 먼저 노출 시 중간 티어 선택률 40% 증가. 미용실 사장님에게도 동일 구조 적용. 단, 첫 제시에서 가격을 꺼내는 게 아니라 손실 수치 확인 후 "해결책 비용"으로 포지셔닝해야 앵커가 작동한다.

## 영업 전략 설계

### 타겟 선별 기준

**1순위 타겟 — 오늘 당장 DM 넣어야 할 조건 (전부 해당해야 함)**

```
✅ 네이버 플레이스 등록 완료 (미등록은 별도 프로세스)
✅ 양주시 내 미용실 — 네이버 검색 "양주 미용실" 기준 순위 8위~20위
✅ 리뷰 수 30개 미만 또는 최근 3개월 리뷰 업데이트 없음
✅ 플레이스 사진 5장 이하 또는 2년 이상 미갱신
✅ 영업시간/가격 정보 미기재 또는 부정확
✅ 인스타/블로그 운영 흔적 없음 (= 마케팅 여력 없음 = 우리 필요성 극대화)
```

**2순위 타겟 — 접근하되 설득 공수 더 필요**
```
△ 순위 5~7위 (이미 어느 정도 되는데 더 올리고 싶은 욕구)
△ 인스타 운영 중이나 플레이스 관리는 미흡
△ 개업 6개월 이내 (신규 우대 알고리즘 타이밍)
```

**제외 타겟 — 시간 낭비**
```
❌ 순위 1~3위 (설득 명분 없음)
❌ 폐업 예정 or 영업 불규칙 (리뷰 최근 1개월 내 "문 닫혀있었어요" 언급)
❌ 이미 타 대행사 계약 중 표시 (블로그/플레이스에 "마케팅 문의" 외부 링크)
```

### 접촉 전략

#### STEP 0 — 발송 전 준비 (10분, 한 업체당)

아래 4가지를 미리 수집해서 메시지에 실명 박아 넣어야 한다.

```
1. 타겟 업체명 + 현재 순위 (네이버 검색 직접 확인)
2. 1위 경쟁사 업체명 + 리뷰 수
3. 타겟 업체 리뷰 수 + 최근 리뷰 날짜
4. 타겟 업체 플레이스 사진 장수 + 마지막 업데이트 날짜
```

이 4가지 없으면 발송하지 마. 없으면 찌라시가 된다.

---

#### STEP 1 — 첫 접촉 메시지 (카카오톡 or 문자)

**발송 타이밍:** 오전 10시~11시 (영업 시작 직후, 아직 바빠지기 전) or 오후 2시~3시 (점심 후 잠깐 여유)
**발송 채널 우선순위:** 네이버 플레이스 문의 → 카카오톡 채널 → 문자

```
[업체명] 사장님, 안녕하세요.

네이버에서 "양주 미용실" 검색하면
지금 사장님 매장이 13위에 있어요.

1위는 [경쟁사명]인데요 —
거기 리뷰가 [N]개, 사장님 매장은 [N]개입니다.

이 순위 차이로 매달 약 557명이
사장님 매장 대신 상위 매장으로 가고 있어요.

미용실 평균 객단가 기준으로 보면
한 달에 약 1,600만~2,700만 원이
그냥 빠져나가는 구조예요.

이게 고쳐질 수 있는 건지
3분짜리 진단 결과 보내드려도 될까요?

— 리안 마케팅 [담당자 이름]
```

**이 메시지가 작동하는 이유:**
- 업체명 실명 → "아 나한테 직접 한 말이구나" (스팸 필터 통과)
- 557명, 1,600만원 → 손실 수치 즉시 각인
- 경쟁사 실명 → "어떻게 알았지?" 반응 유도
- 마지막 질문은 YES/NO가 아니라 "보내드려도 될까요?" → 거절 부담 최소화

---
