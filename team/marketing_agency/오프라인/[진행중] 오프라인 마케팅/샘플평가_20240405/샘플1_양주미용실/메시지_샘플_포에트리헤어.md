## 3차 메시지

> 조건: 2차 읽고 반응 온 경우 ("오", "어떻게 해요", "심각하네요" 등 어떤 반응이든)
> 목표: 계약.

```
사장님, 정리해드리면요.

지금 제일 급한 게 두 가지예요.
리뷰 수 격차랑, 메뉴/가격표 + 새소식 같은
기본 항목들이 비어있는 거거든요.

이걸 채워드리는 게 저희 서비스예요.

─────────────────────
🔹 주목 플랜   290,000원/월
  기본정보 세팅 + 키워드 최적화 + 월 리포트

🔸 집중 플랜   490,000원/월
  주목 포함 + 사진 촬영 + 블로그 리뷰 관리
  + 사장님 답글 대행

⭐ 시선 플랜   890,000원/월
  집중 포함 + 인스타 관리 + 새소식 월 2회
  + 전담 매니저
─────────────────────

사장님한테는 집중 플랜이 맞을 것 같아요.

미용실 손님 한 분 평균 6만원이라 치면,
한 달에 9명만 더 오시면 본전이에요.
지금 리뷰가 2,600개인 업체들이랑 같은 화면에
노출되는 거 생각하면 충분히 가능한 숫자고요.

한번 해보실 의향 있으세요?
```

## 4차 메시지 — 보류/무응답 대응

> 조건: 3차 발송 2~3일 후 무응답 or "생각해볼게요"

### [무응답/보류용]
```
사장님, 혹시 확인하셨나요?

저희가 이번 달 양주/옥정 지역
신규 업체를 2곳만 더 받거든요.
같은 지역 미용실이 문의 주신 게 있어서요.

같은 지역 두 군데를 같이 진행하진 않아서,
이번 달 안에 결정해주시면
옥정 독점으로 진행해드릴 수 있어요.

어떻게 하시겠어요?
```

### [비싸다고 할 때]
```
그 마음 충분히 이해해요.

계산해보면요,
집중 플랜이 한 달 49만원인데
지금 이 지역 상위 미용실들이 리뷰 2,600개인 거
그냥 두면 격차가 계속 벌어지거든요.

손님 9명만 더 오면 본전이에요.
요즘 네이버 플레이스 보고 오는 손님이
그 정도면 충분히 나오고요.

일단 3개월만 해보시고,
효과 없으면 그때 말씀해주세요.
```

### [직접 해보겠다고 할 때]
```
당연히 하실 수 있어요.

근데 아까 X 뜬 항목이 7개였는데,
영수증 리뷰 관리, 블로그 리뷰, 새소식 작성,
사장님 답글 일일이 다 시간이 걸리는 거라서요.

미용실 하시면서 그거까지 챙기기가 쉽지 않으시잖아요.
저희가 대신 할게요.
```

## 발송 체크리스트

```
□ 1차 발송 전: 업체명 맞는지 확인
□ 2차 발송 전: 답장 내용 확인 (긍정 반응인지)
□ 3차 발송 전: 추천 플랜 + 손익분기 숫자 맞는지 확인
□ 4차 발송 전: 3차 발송 후 2~3일 지났는지 확인
□ 클로징 전: 글로싸인 링크 + 계좌번호 채워넣기
```
