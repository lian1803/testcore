# 리안 플레이북 — 네이버 플레이스 마케팅 영업

> 이 문서는 리안이 직접 참고하는 영업 운영 매뉴얼입니다.
> 시스템이 자동으로 해주는 것과 리안이 직접 해야 할 것을 구분해서 정리했습니다.

---

## 1. 전체 영업 플로우

```
[1단계] 번호 수집        → 시스템 자동
[2단계] 진단 생성        → 시스템 자동
[3단계] PPT 생성         → 시스템 자동
[4단계] 콜드DM 발송      → 리안 직접
[5단계] 반응 확인/연락   → 리안 직접
[6단계] 방문 미팅        → 리안 직접
[7단계] 계약 + 온보딩    → 리안 직접
```

---

## 2. 단계별 역할 분리

### [1단계] 번호 수집

| 구분 | 내용 |
|------|------|
| 시스템이 함 | 지역 + 업종 조합으로 네이버 플레이스 업체 목록 자동 수집, 010번호 추출, CSV 저장 |
| 리안이 함 | 수집할 지역과 업종만 입력 (예: "포천시 미용실") |

### [2단계] 진단 + PPT 생성

| 구분 | 내용 |
|------|------|
| 시스템이 함 | 업체 플레이스 URL 자동 크롤링 → 사진 수/리뷰 수/블로그 수/순위 수집 → 점수 산출 → PPT 자동 생성 |
| 리안이 함 | 업체명 또는 URL 입력 → 다운로드 버튼 클릭 |

### [3단계] 콜드DM 발송

| 구분 | 내용 |
|------|------|
| 시스템이 함 | 없음 (현재는 수동) |
| 리안이 함 | 아래 스크립트 문서에서 템플릿 복붙 → 업체명만 바꿔서 카카오/문자 발송 |

**하루 발송 목표: 30~50건. 오전 10시~오후 1시 사이가 응답률 가장 높음.**

### [4단계] 반응 확인 + 약속 잡기

| 구분 | 내용 |
|------|------|
| 시스템이 함 | 없음 |
| 리안이 함 | 답장 온 업체에 전화 → 방문 일정 잡기 |

**답장 온 업체는 24시간 이내 연락. 늦으면 관심 식음.**

### [5단계] 방문 미팅

| 구분 | 내용 |
|------|------|
| 시스템이 함 | 없음 |
| 리안이 함 | 노트북 또는 출력 PPT 들고 방문 → 슬라이드 설명 → 패키지 제안 |

**방문 시간: 오전 10~12시 또는 오후 2~4시 (점심/저녁 피하기)**

### [6단계] 계약

| 구분 | 내용 |
|------|------|
| 시스템이 함 | 없음 |
| 리안이 함 | 계약서 서명, 첫 달 선결제 확인 |

## 3. 진단 결과 해석법

진단 결과에 나오는 숫자들이 실제로 뭘 의미하는지 정리.

### 사진 수

| 사진 수 | 의미 | 영업 포인트 |
|---------|------|-------------|
| 0~10장 | 심각 (네이버가 노출 안 줌) | "사진이 거의 없어서 검색에서 뒤로 밀려요" |
| 11~30장 | 부족 | "경쟁 업체 평균이 50장 이상이에요" |
| 31~50장 | 보통 | "조금만 더 채우면 확 올라갈 수 있어요" |
| 50장 이상 | 양호 | 다른 약점 포인트로 설명 전환 |

### 리뷰 수 (영수증 + 방문자 리뷰)

| 리뷰 수 | 의미 | 영업 포인트 |
|---------|------|-------------|
| 0~10개 | 신뢰도 없음 | "손님들이 리뷰 보고 선택하는데, 지금은 거의 없어요" |
| 11~50개 | 부족 | "이 지역 상위 업체들은 100개 넘어요" |
| 51~200개 | 보통 | "수는 있는데 최신 리뷰가 중요해요" |
| 200개 이상 | 양호 | 블로그 리뷰 또는 키워드 순위로 전환 |

### 블로그 리뷰 수

| 블로그 수 | 의미 |
|-----------|------|
| 0~5개 | 검색 노출 거의 없음. 네이버는 블로그 많은 업체 우선 노출 |
| 6~20개 | 보통. 상위권 진입 가능성 있음 |
| 20개 이상 | 상위 노출 유리 |

### 키워드 순위

| 순위 | 의미 | 영업 포인트 |
|------|------|-------------|
| 1~3위 | 상위 노출 (사람들이 봄) | 이미 잘 됨 → 다른 약점으로 |
| 4~10위 | 아슬아슬 (스크롤 내려야 보임) | "지금 4위인데, 1위로 올리면 클릭 3배 차이나요" |
| 11위 이하 | 사실상 안 보임 | "이 키워드로는 지금 검색해도 안 나와요" |
| 순위 없음 | 검색 자체가 안 됨 | 가장 강력한 영업 포인트 |

### 종합 점수 / 등급

| 등급 | 점수 | 현실 의미 |
|------|------|-----------|
| A | 80점 이상 | 이미 잘 관리 중. 영업 어려움 |
| B | 60~79점 | 약간 부족. 개선 여지 있음 |
| C | 40~59점 | 많이 부족. 영업하기 좋은 타겟 |
| D | 39점 이하 | 방치 상태. 가장 좋은 타겟 |

**C등급, D등급 업체를 우선 공략한다.**

## 4. 어떤 업체를 먼저 공략해야 하는가

### 우선순위 1순위 (바로 연락)

아래 조건 2개 이상 해당하면 최우선 타겟:
- 종합 등급 D (39점 이하)
- 사진 10장 미만
- 리뷰 10개 미만
- 키워드 순위 11위 이하 또는 없음
- 블로그 리뷰 5개 미만

이런 업체는 지금 네이버에서 거의 노출 안 되고 있음. 개선 효과가 눈에 보임.

### 우선순위 2순위

- 종합 등급 C (40~59점)
- 경쟁 업체 대비 사진/리뷰 수 절반 미만
- 키워드 4~10위

조금만 건드려도 순위 올라가는 업체들. 성과 내기 쉬움.

### 피해야 할 타겟

- 종합 등급 A (이미 잘 관리 중 → 필요성 못 느낌)
- 폐업 징후 있는 업체 (리뷰 최신이 1년 이상 전)
- 프랜차이즈 본사 직영점 (본사에서 마케팅 담당)

### 업종별 추천 우선순위

1위: 미용실, 네일샵 (경쟁 심하고 플레이스 순위 민감)
2위: 식당, 카페 (리뷰 수가 직접 매출로 연결)
3위: 학원, 피부관리 (블로그 리뷰 효과 크게 느낌)
4위: 인테리어, 세탁소 (경쟁 덜 해서 순위 올리기 쉬움)
