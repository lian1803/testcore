# 소상공인 네이버 플레이스 영업툴 — 업그레이드 지시서

## 핵심 목표

리안이 소상공인에게 네이버 플레이스 마케팅 대행 서비스를 영업하는 툴.
업체 진단 → PPT 제안서 → 영업 메시지 자동 생성까지 한 흐름으로 완성.

**지금 있는 코드를 버리지 말고, 더 정교하게 업그레이드해라.**

## ⛔ 영업 설득 핵심 원칙 — 모든 아웃풋에 적용

**"문제는 확실히 보여주되, 해결법은 절대 안 알려준다"**

이 툴의 목적은 사장님을 설득해서 계약시키는 것이다.
너무 많이 알려주면 "혼자 해볼게요"로 끝나고, 너무 적으면 설득이 안 된다.

### 보여줘야 할 것 (공포/충격 유발)
- 종합 점수 + 등급 (D등급이면 시각적으로 충격)
- 경쟁사와의 격차 숫자 ("경쟁사 리뷰 2,600개 vs 사장님 116개")
- 매달 손실 추정 고객 수 ("매달 약 N명이 경쟁사로")
- 항목별 X 표시 (뭐가 비어있는지 목록)
- 현재 순위의 의미 ("11위면 사실상 안 보여요")

### 절대 안 보여줄 것 (계약 후에만 제공)
- 구체적인 개선 방법 ("사진을 이렇게 찍어라" ❌)
- 경쟁사가 구체적으로 뭘 잘했는지 ❌
- 순위 올리는 구체적 노하우 ❌
- 각 항목을 채우는 실행 방법 ❌

### PPT/메시지 톤
- "사장님 지금 이런 상태예요" ← 여기까지만
- "이렇게 하면 돼요" ← 이건 계약 후
- 개선 슬라이드에는 "이 부분이 비어있어요" 수준만. "이렇게 하세요"는 삭제
- 모든 메시지에서 해결책은 "저희가 해드릴 수 있어요" 한 줄로만

## 현재 상태 (기존 코드 요약)

### naver-diagnosis/ — 메인 코드 (✅ 실행 가능)

**스택**: FastAPI + Playwright + python-pptx + SQLite

**구현된 기능:**
- 업체명 검색 → 네이버 플레이스 크롤링
- 7개 항목 점수 산출 (사진/리뷰/블로그/정보완성도/키워드/편의기능/참여도)
- 가중치 기반 종합 점수 + A~D 등급
- 7슬라이드 PPT 생성 및 다운로드
- SQLite DB 저장 (이력 관리)
- 네이버 검색광고 API 연동 (키워드 검색량)

**현재 점수 가중치:**
- 사진 15% / 리뷰 20% / 블로그 10% / 정보완성도 15% / 키워드 10% / 편의기능 15% / 참여도 15%

**현재 PPT 구성 (7슬라이드):**
1. 표지 (업체명, 종합점수, 등급)
2. 분석 결과 (항목별 점수)
3. 세부 체크리스트
4. 개선사항
5. 키워드 분석
6. 연관 키워드
7. 마케팅 제안

**없는 기능:**
- 일괄 처리 (xlsx에서 업체 목록 불러와서 배치 진단)
- 영업 메시지 자동 생성 (1차~4차 메시지 템플릿에 데이터 자동 주입)
- 경쟁사 비교 데이터
- 업체 영업 우선순위 자동 분류

## 영업 문서 현황 (업그레이드 시 반드시 반영)

아래 문서들이 이미 존재한다. 이걸 코드에 녹여야 한다.

### 영업 퍼널 (영업_퍼널_설계.md)
4단계 메시지 클로징 퍼널:
- **1차**: 충격 숫자 1개 + "보내드려도 될까요?" (3가지 버전 자동 선택)
  - [A] 리뷰 격차형: 경쟁사 리뷰 >= 내 리뷰 × 5
  - [B] 손실 고객형: 순위 > 10 AND 월검색 >= 500
  - [C] 방치형: 새소식 >= 90일 or 저장수 = 0
- **2차**: 진단 결과 전체 텍스트 카드 (카톡 형식)
- **3차**: 솔루션 + 패키지 + 기회비용 계산 + 계약 요청
- **4차**: 보류/무응답 대응 (마감 압박)

### 패키지 (영업_스크립트.md)
| 패키지 | 금액 | 내용 |
|--------|------|------|
| 주목 | 290,000원/월 | 기본정보 최적화 + 키워드 + 월 리포트 |
| 집중 | 490,000원/월 | 주목 + 사진 촬영 + 블로그 리뷰 + 답글 대행 |
| 시선 | 890,000원/월 | 집중 + 인스타 + 새소식 + 전담 매니저 |

### 업종별 객단가 (손익분기 계산용)
| 업종 | 평균 객단가 |
|------|------------|
| 미용실/네일 | 5~8만원 |
| 식당/카페 | 1.5~3만원 |
| 피부관리 | 8~15만원 |
| 학원 | 20~40만원 |

### 영업 우선순위 타겟 기준 (영업_플레이북.md)
- **1순위**: D등급 + (사진<10 OR 리뷰<10 OR 순위>10 OR 블로그<5) 중 2개 이상
- **2순위**: C등급 + 경쟁사 대비 절반 미만
- **피할 것**: A등급, 프랜차이즈 본사 직영, 최신 리뷰 1년+ 된 업체

## 업그레이드 방향 (AI가 판단하고 개선해야 할 것들)

### 1. 점수 로직 정교화
현재 점수 기준이 단순 수치 비교 수준. 아래를 반영해서 더 정교하게:
- 경쟁사 대비 상대 점수 개념 추가 (이 지역 상위 10개 업체 평균 대비)
- 새소식 업데이트 빈도 (90일+ = 방치 패널티)
- 사장님 답글률 (최근 10개 리뷰 기준)
- 저장 수 (북마크) 반영
- 업종별 가중치 차등 (미용실은 사진 가중치 높이고, 식당은 리뷰 가중치 높이는 식)

### 2. PPT 전환 최적화
현재 PPT는 정보 전달 수준. 영업용 제안서로 리디자인:
- 표지: "사장님, 지금 매달 N명이 경쟁사로 가고 있어요" 메시지 강화
- 경쟁사 비교 슬라이드 추가 (내 업체 vs 이 지역 1위/평균)
- 손익분기 계산 슬라이드 추가 (패키지 선택 시 몇 명만 더 오면 본전인지)
- 개선 전/후 예상 시각화
- 각 슬라이드에 "어?" 유발 포인트 1개씩 (영업_스크립트.md 참고)

### 3. 영업 메시지 자동 생성
진단 완료 후, 진단 데이터를 기반으로 1차~4차 메시지 자동 완성:
- 업체 상황에 따라 1차 메시지 A/B/C 버전 자동 선택
- 2차 카톡 카드 형식 텍스트 자동 생성
- 3차 패키지 추천 + 손익분기 숫자 자동 계산 주입
- 4차 보류 대응 메시지 자동 생성
- "복붙하면 바로 쓸 수 있는" 형태로 화면에 표시

### 4. xlsx 연동 (배치 처리)
소상공인_010수집 최종본/db/ 에 xlsx 파일들이 있음.
- xlsx 업로드 또는 파일 경로 지정 → 업체 목록 파싱
- 업체명 컬럼 자동 감지 → 일괄 진단 큐 등록
- 진단 완료된 업체에 영업 우선순위 태그 자동 부여 (1순위/2순위/패스)
- 결과를 업체별로 관리할 수 있는 목록 UI

## 시장 컨텍스트

- **타겟**: 지역 소상공인 (미용실, 식당, 학원, 피부관리 등)
- **구매자**: 리안 (1인 영업자, 직접 DM 발송 + 방문 영업)
- **경쟁 상황**: 네이버 플레이스 마케팅 대행사 다수 존재, 차별점은 진단 리포트 선 제공
- **영업 지역**: 경기 북부 (양주, 포천, 의정부 등)
- **수익 모델**: 월 구독 (29만~89만원)
- **월 계약 목표**: 5건 → 월 280~635만원
