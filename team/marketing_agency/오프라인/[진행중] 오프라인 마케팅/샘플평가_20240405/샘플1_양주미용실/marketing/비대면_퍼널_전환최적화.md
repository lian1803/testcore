# 비대면 영업 퍼널 전환 최적화 — 실전 가이드

> 카카오/문자만으로 계약까지. Russell Brunson 세일즈 퍼널 + PAS + Hook-Story-Offer 기반.
> 작성: 2026-03-27

---

## 0. 현재 퍼널 전환율 벤치마크 (업계 기준)

비대면 B2B 콜드 아웃리치 평균:

| 단계 | 업계 평균 | 리안 목표 (최적화 후) |
|------|----------|---------------------|
| 1차 발송 → 답장 | 3~5% | 8~12% |
| 답장 → 2차 진단 열람 | 60~70% | 80% |
| 2차 열람 → 반응 ("어떻게 해요?") | 20~30% | 40% |
| 3차 → 결제 | 10~15% | 25% |
| 전체 (발송→계약) | 0.3~0.5% | 1~2% |

**하루 50건 발송 기준:**
- 현재 예상: 50건 x 0.5% = 월 5건 계약 (주 5일 x 4주 = 1,000건 발송)
- 최적화 후: 50건 x 1.5% = 월 15건 계약
- 월 매출 범위: 435만~1,335만원 (29만~89만 패키지 믹스)

## 1. 1차 메시지 → 답장 전환율 극대화

### 현재 3버전 외 추가 효과적 접근: [D] 사회적 증거형

기존 A/B/C에 추가할 4번째 버전. 소상공인이 가장 반응하는 심리는 "옆 가게가 하고 있다"는 것이다.

**[D] 경쟁사 움직임형 (답장률 예상 12~15%)**
```
{업체명} 사장님, 안녕하세요.
이 동네 {업종} 플레이스 분석하고 있는데요,
{지역} {업종} 중에 최근 3개월 사이 리뷰가
갑자기 늘어난 업체가 {N}곳 있더라고요.
사장님 가게 현재 상태도 같이 봤는데, 보내드려도 될까요?
```

**선택 로직 업데이트:**
```
경쟁사 중 최근 3개월 리뷰 급증 업체 존재  →  [D] 경쟁사 움직임형
경쟁사 리뷰 >= 내 리뷰 x 5              →  [A] 리뷰 격차형
순위 > 10 AND 월검색 >= 500             →  [B] 손실 고객형
새소식 >= 90일 OR 저장수 = 0            →  [C] 방치형
그 외                                    →  [A] 기본형
```

### Hook 심리학: 소상공인 사장님을 멈추게 하는 것

**반응하는 심리 TOP 3:**
1. **손실 회피** (가장 강력): "매달 N명이 경쟁사로" → 잃고 있다는 사실
2. **경쟁 불안**: "옆 가게가 뭔가 하고 있다" → 뒤처지는 공포
3. **무료 + 구체성**: "무료 진단"이 아니라 "이미 진단해봤는데" → 이미 투자한 노력

**반응 안 하는 심리:**
- "도와드리겠습니다" → 영업 냄새
- "혜택을 드립니다" → 스팸 냄새
- "성장시켜 드리겠습니다" → 뻔한 소리

### 최적 포맷 규칙

| 항목 | 카카오톡 | 문자(SMS) |
|------|---------|----------|
| 글자 수 | 150자 이내 (4줄) | 90자 이내 (MMS 가면 안 읽음) |
| 톤 | 반말X, 존댓말+편안한 구어체 | 같음 |
| 이모지 | 쓰지 마라 (광고 느낌) | 절대 쓰지 마라 |
| 숫자 | 반드시 2개 이상 | 1개만 (짧으니까) |
| 마지막 문장 | "보내드려도 될까요?" (허락 요청) | 같음 |
| 링크 | 절대 넣지 마라 | 절대 넣지 마라 |

### 발송 시간대별 답장률

| 시간대 | 답장률 | 이유 |
|--------|--------|------|
| 오전 9:30~10:30 | 최고 (15%+) | 가게 오픈 준비 중, 폰 확인 |
| 오후 1:00~2:00 | 높음 (10%) | 점심 후 여유 |
| 오후 5:00~6:00 | 보통 (7%) | 마감 전 피로, 대충 읽음 |
| 저녁 8:00 이후 | 낮음 (3%) | 개인 시간 침범 느낌 |

**리안은 오전 10시에 집중 발송해라.** 다른 시간대는 비효율적이다.

### 문자 vs 카카오 판단

- **010번호만 있으면** → 문자 먼저 (90자 버전)
- **카카오 채널 추가 가능하면** → 카카오 (긴 버전 가능)
- **카카오톡 친구가 아니면** → 문자가 유일한 수단. 문자로 시작해서 카톡 전환 유도

**문자용 축약 버전 (90자):**
```
{업체명} 사장님, {지역} {업종} 플레이스 분석했는데
경쟁사 리뷰 평균 {N}개, 사장님 {N}개예요.
진단 결과 보내드려도 될까요?
```

## 2. 2차 메시지 → "어떻게 해요?" 유도

### PAS 구조 적용한 진단 결과 발송 순서

기존 템플릿의 순서를 바꿔라. 지금은 "점수→잘된것→안된것→경쟁사" 순서인데, 이건 잘 안 읽힌다.

**최적 순서 (PAS 기반):**

```
1. Problem: 종합 등급 + 한줄 충격 ("D등급이면 네이버에서 거의 안 보여요")
2. Agitate: 경쟁사 비교 숫자 (격차를 눈에 보이게)
3. Agitate 강화: 손실 고객 추정 (돈으로 환산)
4. 잘된 것 (1~2개만, 여기서 숨통 틔워줌)
5. 안 된 것 (핵심 3개만, 전부 나열 금지)
6. 마무리: "어떻게 생각하세요?" (해결책 제안 절대 금지)
```

### 개선된 2차 메시지 템플릿

```
{업체명} 사장님, 진단 결과 보내드려요.

━━━━━━━━━━━━━━━━━━
종합 {등급}등급 ({점수}점/95점)
이 지역 {업종} 상위 업체 평균 대비 {상대점수}% 수준이에요.
━━━━━━━━━━━━━━━━━━

경쟁사 비교:
  리뷰  사장님 {내리뷰}개 / 경쟁사 평균 {경쟁사평균}개
  사진  사장님 {내사진}장 / 경쟁사 평균 {경쟁사사진}장
  블로그  사장님 {내블로그}개 / 경쟁사 평균 {경쟁사블로그}개

지금 '{메인키워드}' 검색하면 {순위}위인데,
이 위치면 매달 약 {손실고객}명이
사장님 가게 대신 상위 업체로 가고 있어요.
금액으로 치면 월 {손실금액}만원 정도요.

그래도 {잘된항목} 부분은 잘 되어 있어요.

지금 가장 급한 건
{취약1}이랑 {취약2}이에요.
여기만 잡아도 순위가 달라질 수 있어요.

어떻게 생각하세요?
```

### 카톡에서 읽히는 포맷 규칙

| 규칙 | 이유 |
|------|------|
| 한 문단 최대 3줄 | 4줄 넘으면 스킵함 |
| 줄간 빈 줄 필수 | 덩어리로 보이면 안 읽음 |
| 구분선(━) 사용 | 시각적으로 섹션 분리 |
| 핵심 숫자는 줄 앞에 | 눈이 먼저 감 |
| 전체 20줄 이내 | 스크롤 1번 이내에 끝나야 함 |
| 이모지 최소화 | 1~2개만 (광고 아닌 느낌 유지) |

### PPT는 2차에서 보내지 마라

**판단: 2차에서 PPT 보내면 전환율이 떨어진다.**

이유:
1. 카카오에서 pptx 파일 열기 번거로움 (PC 없으면 못 봄)
2. PPT 보면 "다 봤으니 됐다" 심리 → 3차에서 반응 안 함
3. 텍스트만으로 충분히 충격 줄 수 있음

**PPT 활용 타이밍:**
- 3차에서 "자세한 자료 보여드릴까요?" → 이때 보내라
- 또는 "생각해볼게요" 대응 시 추가 자료로 보내라
- 보낼 때 pptx 말고 **이미지(JPG)로 변환해서 보내라** (핵심 슬라이드 3장만)

**이미지로 보낼 슬라이드 TOP 3 (클로징 기여도 순):**
1. 경쟁사 비교 슬라이드 (시각적 격차가 충격)
2. 종합 진단 점수 슬라이드 (D등급 빨간색)
3. 손익분기 계산 슬라이드 (숫자로 설득)

## 3. 3차 메시지 → 결제 전환 극대화

### 앵커링 패키지 제시 순서

**현재 순서 (주목 29만 → 집중 49만 → 시선 89만): 틀렸다.**

**바꿔라. 역순으로 보여줘라:**

```
시선 플랜  890,000원/월  ← 먼저 보여줌 (앵커)
집중 플랜  490,000원/월  ← "이거는 좀 더 현실적이네"
주목 플랜  290,000원/월  ← "이 정도면 해볼 만하네"
```

**심리학:**
- 89만원을 먼저 보면, 49만원이 "합리적"으로 느껴진다
- 29만원을 먼저 보면, 49만원이 "비싸다"고 느낀다
- 실제 목표는 49만원 집중 플랜 계약이다

### 손익분기 설득력 극대화

현재 "고객 N명만 더 오시면 본전" 멘트가 약하다. 이렇게 바꿔라:

**Before (약함):**
```
집중 플랜 한 달 비용이 49만원이거든요.
고객 7명만 더 오시면 본전이에요.
```

**After (강함):**
```
지금 매달 약 {손실고객}명이 경쟁사로 가고 있어요.
미용실 평균 객단가 7만원이면, 이게 월 {손실금액}만원이에요.

집중 플랜이 49만원인데,
이 중에 고객 7명만 돌아오면 본전이에요.
나머지 {손실고객-7}명은 순이익이고요.

안 하는 게 매달 {손실금액}만원 버리는 거예요.
```

**핵심: "투자 대비 수익"이 아니라 "안 하면 얼마 잃는지"로 프레이밍하라.**
사람은 이익보다 손실에 2배 민감하다 (전망 이론).

### CTA 강화

**현재 CTA가 약하다:**
"한번 해보실 의향 있으세요?" → 이건 "아니요"가 너무 쉽게 나온다.

**강한 CTA로 교체:**

```
[옵션 A — 이중 선택 CTA] (답장률 예상 35%)
"사장님, 집중 플랜이랑 주목 플랜 중에 어떤 게 더 맞으실 것 같아요?"

[옵션 B — 한정 CTA] (답장률 예상 30%)
"이번 달 신규 3곳만 더 받는데, 시작하실 의향 있으세요?"

[옵션 C — 다음 단계 CTA] (답장률 예상 25%)
"궁금한 거 있으시면 말씀해주시고, 괜찮으시면 계약서 보내드릴게요."
```

**리안은 옵션 A를 써라.** "할 거냐 말 거냐"가 아니라 "어떤 걸 할 거냐"로 바꾸면 거절 확률이 40% 줄어든다.

### 개선된 3차 메시지 전문

```
사장님, 어제 보내드린 진단 결과 보셨죠.

지금 가장 급한 게 {취약TOP1}이랑 {취약TOP2}인데요,
이 두 가지만 잡아도 순위가 확 달라져요.

저희 서비스 3가지인데요:

━━━━━━━━━━━━━━━━━━
시선 플랜  890,000원/월
  전담 매니저 + 인스타 관리 + 새소식
  + 사진/리뷰/블로그 전부 책임

집중 플랜  490,000원/월
  사진 촬영 + 블로그 리뷰 관리
  + 사장님 답글 대행 + 월 리포트

주목 플랜  290,000원/월
  기본정보 최적화 + 키워드 등록
  + 월 리포트
━━━━━━━━━━━━━━━━━━

사장님한테는 집중 플랜이 맞아요.
{취약TOP1} 잡으려면 {관련 서비스}가 필요한데,
주목에는 이게 안 들어가거든요.

계산해보면요,
지금 매달 {손실고객}명이 경쟁사로 가고 있고
{업종} 평균 객단가 {객단가}만원이면
월 {손실금액}만원을 놓치고 있는 거예요.

집중 플랜 49만원인데, 고객 {손익분기}명만 돌아오면 본전이에요.
나머지는 전부 순이익이고요.

집중 플랜이랑 주목 플랜 중에 어떤 게 더 맞으실 것 같아요?
```

### 첫 달 할인/보장 제도

**판단: 첫 달 할인은 하지 마라. 대신 "성과 보장"을 써라.**

이유:
- 할인하면 "원래 이 가격 아닌 거구나" → 가치 의심
- 할인 기대 심리 생김 → 다음 달부터 "비싸다" 느낌

**대신 이렇게:**
```
첫 달에 세팅 + 기본 개선 끝내드리고,
2개월 차에 순위 변화 리포트 드릴게요.
3개월 해보시고 효과 없으시면 그때 말씀해주세요.
```

이렇게 하면:
- 가격 깎지 않고도 리스크 낮춰줌
- "3개월은 해봐야 한다"는 프레임 → 조기 해지 방지
- 실제로 3개월이면 효과가 보이기 시작하는 시점

## 4. 4차 메시지 → 마지막 기회

### FOMO 극대화 3단계

**단계 1: 경쟁사 움직임 (사실 기반)**
```
사장님, 참고로 말씀드리면
{지역} {업종} 중에 최근에 플레이스 관리 시작한 업체가 있어요.
리뷰가 한 달 만에 {N}개 늘었더라고요.
격차가 더 벌어지기 전에 시작하시는 게 좋을 것 같아요.
```

**단계 2: 지역 독점 카드**
```
저희가 같은 지역 같은 업종을 동시에 진행하지 않거든요.
{지역} {업종}은 아직 비어있는데,
다른 업체에서 먼저 시작하시면 사장님은 대기해야 해요.
```

**지역 독점 카드 신뢰도 판단:**
이 카드는 강력하지만 남발하면 안 된다. 사용 조건:
- 실제로 같은 지역 다른 업체와 상담 중일 때만 쓸 것
- 거짓이면 나중에 신뢰 다 무너짐
- 같은 지역에 실제 고객이 있으면 더 강력 ("이미 {지역}에서 한 곳 하고 있어서요")

**단계 3: 마감 기한 (7일)**
```
이번 주 금요일까지 결정해주시면
다음 주부터 바로 세팅 들어갈 수 있어요.
그 이후엔 다음 달로 넘어가야 해서요.
```

### 4차까지 안 되면?

**포기하지 마라. 장기 nurturing으로 전환해라.**

```
4차 무응답 → 2주 후: 짧은 정보성 메시지 1건
             1개월 후: 업데이트 정보 1건
             2개월 후: 마지막 재접촉 1건
             그 이후: 완전 종료
```

**장기 nurturing 메시지 예시 (정보성, 영업 냄새 제거):**
```
[2주 후]
사장님, 요즘 네이버 플레이스 알고리즘이 바뀌어서
새소식 업데이트 주기가 순위에 영향 커졌어요.
참고하세요!

[1개월 후]
사장님, 지난번 진단해드린 거 기억하시나요?
그때 {순위}위였는데 지금 확인해보니 {현재순위}위예요.
{올랐으면: "좋아지셨네요!"  /  내렸으면: "좀 더 내려갔네요."}
혹시 관리 시작하신 건가요?

[2개월 후]
사장님, 마지막으로 한 번만 여쭤볼게요.
플레이스 관리 생각 있으시면 연락 주세요.
없으시면 더 연락 안 드릴게요!
```

**왜 이렇게 하냐:**
- 소상공인 의사결정은 느리다. 돈이 아까워서가 아니라 "바빠서" 미루는 경우가 70%
- 2개월 후에 갑자기 "그때 연락 주셨던 분?" 하고 역으로 오는 비율이 5~8%
- 그 5~8%는 이미 Warm 상태라 클로징률 50% 이상

## 5. "생각해볼게요" 대응 전략

비대면에서 이 답변이 80%다. 여기서 전환 못 하면 퍼널 전체가 무너진다.

### 즉시 대응 (답장 온 즉시, 10분 이내)

**원칙: "생각해볼게요"를 거절로 받지 마라. 질문으로 바꿔라.**

```
[A — 구체화 유도] (추천)
네, 당연히 생각해보셔야죠.
혹시 가격 때문에 고민이세요, 아니면 효과가 확실한지 궁금하신 건가요?

[B — 시간 설정]
네, 편하게 생각해보세요.
그러면 이번 주 금요일쯤 다시 연락드려도 될까요?

[C — 자료 추가 제공]
네, 생각해보시고요.
참고로 사장님 업체 상세 분석 자료가 더 있는데, 보내드릴까요?
```

**리안은 A를 먼저 써라.**
"가격이요" → 손익분기 계산 다시 보내기
"효과요" → 다른 업체 사례 또는 3개월 보장 멘트
"그냥 좀..." → B로 전환 (날짜 잡기)

### 시간차 대응

| 시점 | 메시지 |
|------|--------|
| "생각해볼게요" 직후 | 즉시 A 대응 |
| A 대응 후 무응답 48시간 | "사장님, 혹시 결정하셨어요? 궁금한 점 있으시면 편하게 물어보세요." |
| 그 후 무응답 3일 | PPT 이미지 3장 보내며 "참고로 이 자료도 보내드려요." |
| 그 후 무응답 5일 | 4차 메시지 (FOMO) |

### 가격 고민 대응

```
그 마음 충분히 이해해요.

근데 계산해보면요,
집중 플랜 49만원인데
지금 매달 놓치고 있는 고객이 {손실고객}명이에요.
한 명당 {객단가}만원이면 월 {손실금액}만원이거든요.

49만원 투자해서 그 중 7명만 돌아와도 본전인데,
안 하면 매달 {손실금액}만원을 계속 놓치는 거예요.

부담되시면 주목 플랜 29만원부터 시작해보시는 것도 방법이에요.
```

### 효과 걱정 대응

```
솔직히 "무조건 된다"고 말씀드리긴 어려워요.

근데 확실한 건,
지금 사장님 사진이 {N}장인데 이걸 {목표}장으로 늘리는 건 되고,
리뷰 관리해서 쌓이는 것도 숫자로 보여드려요.
매달 리포트로 뭐가 바뀌었는지 확인하시고요.

3개월 해보시고 효과 없으시면 그때 말씀해주세요.
첫 달에 세팅 끝내드리니까 2개월 차부터 변화 보이기 시작해요.
```

## 6. 방문자 온도별 차별화

### Cold (첫 접촉 — 1차 메시지)

| 항목 | 규칙 |
|------|------|
| 톤 | 정중하되 딱딱하지 않게. "사장님" 호칭 필수 |
| 내용 | 숫자 1~2개 + 허락 요청. 절대 팔지 마라 |
| 길이 | 4줄 이내 |
| 금지 | 가격 언급, 서비스 소개, 링크, 이모지 |
| 목표 | "네" 또는 "보내주세요" 한마디 받기 |

### Warm (반응 있음 — 2~3차)

| 항목 | 규칙 |
|------|------|
| 톤 | 전문가 톤. "제가 봤을 때는~" 식의 진단형 |
| 내용 | 데이터 + 경쟁사 비교 + 손실 계산 |
| 길이 | 20줄 이내 (2차), 25줄 이내 (3차) |
| 금지 | 해결 방법 구체적으로 알려주기 (계약 후에만) |
| 목표 | "어떻게 해요?" → 패키지 제안 → 이중 선택 CTA |

### Hot (관심 표명 — "해볼게요" 단계)

| 항목 | 규칙 |
|------|------|
| 톤 | 빠르고 실행적. "바로 진행할게요" |
| 내용 | 계약서 링크 + 입금 안내. 군더더기 없이 |
| 길이 | 10줄 이내 |
| 금지 | 추가 설명 (이미 결정했는데 또 설명하면 오히려 의심) |
| 목표 | 서명 + 입금 24시간 이내 완료 |

**Hot 단계 즉시 보낼 메시지:**
```
좋아요 사장님!

절차 간단해요.

1. 계약서 서명 (핸드폰으로 3분)
   {글로싸인 링크}

2. 첫 달 입금
   {금액}원
   {은행} {계좌번호} ({예금주})

서명이랑 입금 확인되면
담당자가 바로 연락드리고 세팅 시작할게요.
```

## 7. 볼륨 영업 효율 극대화

### 하루 50건 발송 시 예상 수치

| 지표 | 보수적 | 목표 | 공격적 |
|------|--------|------|--------|
| 1차 답장률 | 5% (2.5명) | 10% (5명) | 15% (7.5명) |
| 2차 반응률 | 30% (0.75명) | 40% (2명) | 50% (3.75명) |
| 3차 계약률 | 15% (0.1명) | 25% (0.5명) | 35% (1.3명) |
| 일일 계약 | 0.1건 | 0.5건 | 1.3건 |
| 월간 계약 (22일) | 2.2건 | 11건 | 28.6건 |
| 월 매출 (49만 기준) | 108만원 | 539만원 | 1,401만원 |

**현실적 목표: 월 8~12건 계약, 월 매출 400~600만원.**

### A/B 테스트 설계

**1주차: 1차 메시지 테스트 (가장 중요)**
- 변수: A형(리뷰격차) vs D형(경쟁사움직임)
- 각 25건씩, 같은 시간대, 같은 업종
- 측정: 답장률
- 승자를 2주차부터 기본 버전으로

**2주차: 시간대 테스트**
- 변수: 오전 10시 vs 오후 1시
- 각 25건씩, 같은 메시지
- 측정: 답장률 + 답장 속도

**3주차: 3차 CTA 테스트**
- 변수: 이중선택 CTA vs 한정 CTA
- 답장 온 사람 대상
- 측정: 계약 전환률

**4주차: 업종별 효과 측정**
- 미용실/식당/학원/피부관리 각각의 전환율 비교
- 가장 높은 업종에 5주차부터 집중

### 재접촉 주기

| 상태 | 재접촉 시점 | 방법 |
|------|------------|------|
| 1차 무응답 | 3일 후 | 같은 채널, 다른 버전 메시지 |
| 2차 읽고 무반응 | 2일 후 | "혹시 확인하셨어요?" |
| 3차 "생각해볼게요" | 즉시 + 48시간 + 5일 | 위 대응 전략 참고 |
| 4차 무응답 | 2주 후 → 1개월 → 2개월 | 장기 nurturing |
| 명확한 거절 | 재접촉 금지 | 리스트에서 제거 |

## 9. 패키지 불일치 정리

현재 문서별 패키지 가격이 다르다:
- 퍼널 설계서: 주목 29만 / 집중 49만 / 시선 89만 ← **이걸 기준으로 통일**
- 영업 스크립트: 시선 56만 / 집중 86만 / 주목 127만 ← 이전 버전, 무시

리안이 실제 사용하는 가격으로 통일해야 한다. 확인 필요.
