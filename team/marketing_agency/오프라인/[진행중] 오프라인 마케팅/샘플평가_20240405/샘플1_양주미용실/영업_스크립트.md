# 영업 스크립트 — 네이버 플레이스 마케팅 대행

> 복붙해서 바로 쓸 수 있는 실전 스크립트입니다.
> [업체명], [지역], [순위], [숫자] 부분만 바꿔서 사용하세요.

---

## A. 콜드DM 문자 템플릿

### 버전 1 — 진단 결과 공유형 (가장 반응률 높음)

```
안녕하세요, [업체명] 사장님.
네이버 플레이스 무료 진단해봤는데,
현재 '[키워드]' 검색 시 [순위]위에 계세요.
사진이 [N]장으로 경쟁 업체 평균 절반 수준이라
노출이 많이 줄어든 상태예요.
간단한 자료 드려도 될까요?
```

> 글자 수: 약 110자. 숫자 2개 이상 넣으면 신뢰도 올라감.

---

### 버전 2 — 궁금증 유발형

```
사장님, 네이버에서 '[지역] [업종]' 검색하면
지금 몇 위에 나오시는지 확인해보셨어요?
생각보다 훨씬 뒤에 계실 수 있어요.
무료로 확인해드릴게요, 부담 없이 연락 주세요.
```

> 글자 수: 약 95자. 직접적인 질문 형식이라 호기심 자극.

---

### 버전 3 — 무료 제공형

```
안녕하세요 사장님!
[업체명] 네이버 플레이스 진단 리포트
무료로 만들어드렸어요.
리뷰/사진/키워드 순위 한 번에 확인 가능하고
개선하면 어떻게 되는지도 나와 있어요.
받아보실래요? 부담 없이 보내드릴게요.
```

> 글자 수: 약 115자. "무료"라는 단어가 들어가면 거절 장벽 낮아짐.

---

### 발송 팁

- 오전 10시~오후 1시 사이 발송 (식사 전후 확인율 높음)
- 하루 25~30건 이상 보내면 피로감 생기고 실수 남. 25건 고수.
- 답장 없으면 3일 후 1회만 재발송 (이후 미련 버리기)
- 재발송 멘트: "사장님, 지난번 진단 자료 아직 안 보셨을 것 같아서 다시 연락드렸어요. 부담 없이 받아보세요!"

## B. 전화/방문 오프닝 스크립트

### 처음 30초 — 전화 오프닝

```
"사장님, 안녕하세요. 저 며칠 전에 문자 드린 리안이라고 해요.
네이버 플레이스 마케팅 관련해서요.
지금 잠깐 30초만 시간 괜찮으세요?

[OK 하면]

네, 감사합니다. 사장님 업체 네이버에서 분석해봤는데요,
지금 '○○ [업종]' 키워드에서 ○위에 계시거든요.
같은 지역에 비슷한 업체들이 1~3위 차지하고 있고,
거기서 클릭이 다 일어나는 구조예요.
제가 자료로 정리해뒀는데, 잠깐 뵙고 보여드려도 될까요?
딱 20분이에요."
```

> 핵심: "30초만요"로 시작, 숫자로 구체적으로, "20분"으로 부담 낮추기.

---

### "저 관심 없어요" 반론 대응

```
"아, 네. 그렇군요. 혹시 지금 네이버 쪽은 별도로 관리 중이신 건가요?

[아니오] → "그렇구나요. 사실 안 하셔도 괜찮아요. 그냥 지금 상태가 어느 정돈지
          한 번만 보여드리고 싶었어요. 자료 자체가 손해될 건 없거든요.
          딱 한 번만 보시고 필요 없으면 그때 거절하셔도 돼요."

[네, 하고 있어요] → "오, 그렇군요. 어떤 식으로 관리하고 계세요?
                  (들은 후) 저희가 하는 방식이랑 다를 수 있어서,
                  비교해서 보여드리면 도움이 될 것 같아요.
                  잠깐만 보시겠어요?"
```

---

### 방문 현장 오프닝 (앉자마자 첫 마디)

```
"사장님, 바쁘신데 시간 내주셔서 감사해요.
오래 걸리지 않을게요. 딱 20분만요.

저 사장님 업체 미리 분석해서 자료로 만들어 왔거든요.
지금 네이버에서 어떤 상태인지, 어디서 손해 보고 있는지,
개선하면 어떻게 되는지 순서대로 보여드릴게요.

보시면서 궁금한 거 바로 여쭤봐도 되고요."
```

## C. PPT 설명 스크립트 (슬라이드별)

---

### 슬라이드 2 — 업체 분석 (키워드 조회수, 현재 순위, 업체정보)

**설명 스크립트:**

```
"이게 사장님 업체 분석 결과예요.

여기 보시면, '[메인키워드]'라는 키워드가
한 달에 [N]번 검색돼요. [지역]에서요.

근데 지금 사장님 업체는 이 키워드에서 [N]위예요.
사람들이 보통 1~3위까지만 클릭하거든요.
[N]위면 사실상 안 보이는 거예요."

[여기서 멈추고 사장님 표정 확인]

"놀라셨죠? 저도 처음 봤을 때 이 부분이 제일 아쉬웠어요."
```

> "어?" 유발 포인트: 검색량이 생각보다 많다는 것 + 내 순위가 생각보다 뒤라는 것. 이 두 개를 같이 보여줄 때 임팩트 가장 큼.

---

### 슬라이드 3 — 확장 키워드 전략

**설명 스크립트:**

```
"이건 메인 키워드 외에 사람들이 같이 검색하는 키워드들이에요.

예를 들어 '강남 카페' 말고도
'강남 조용한 카페', '강남 혼자 가기 좋은 카페' 이런 것들이요.

이런 키워드들은 경쟁이 덜해서
잘만 하면 빨리 상위에 올릴 수 있어요.

저희가 이 키워드들도 같이 공략하는 전략이에요."
```

> "어?" 유발 포인트: 내가 몰랐던 키워드들이 존재한다는 것. 거기서도 기회가 있다는 것.

---

### 슬라이드 4 — 네이버 검색 결과 현황

**설명 스크립트:**

```
"이게 실제로 [키워드] 검색했을 때 나오는 화면이에요.

보시면 상위 3개 업체가 화면에 딱 나오죠.
여기 들어가는 업체들이 클릭을 다 가져가요.

지금 사장님 업체는 스크롤 한참 내려야 나와요.
핸드폰으로 검색하면 거의 안 보인다고 보시면 돼요.

이게 지금 [월 N명]이 이 키워드 검색할 때
사장님한테는 연결이 안 되고 있는 거예요."
```

> "어?" 유발 포인트: 시각적으로 바로 보임. "월 N명"이라는 숫자가 손해를 구체화시킴.

---

### 슬라이드 5 — 네이버 플레이스 종합 진단

**설명 스크립트:**

```
"이게 사장님 플레이스를 항목별로 점수 낸 거예요.

사진은 지금 [N]장인데, 이 지역 상위권이 보통 [N]장이에요.
리뷰는 [N]개. 경쟁 업체들 평균이 [N]개고요.
블로그 리뷰도 [N]개. 여기서 많이 차이 나요.

종합 점수가 [N]점, [등급]등급이에요.
A~D 중에서 [등급]이면 개선할 여지가 많다는 뜻이에요.

반대로 말하면, 지금 이 부분들 채워주면
순위가 올라갈 확률이 높다는 거예요."
```

> "어?" 유발 포인트: 점수라는 구체적인 수치. 등급이 D면 시각적으로 충격적. "개선 여지 많다 = 올릴 수 있다"로 긍정 전환.

---

### 슬라이드 6/7 — 패키지 소개

**설명 스크립트:**

```
"저희 서비스가 크게 3가지 구성이에요.

시선 패키지 월 56만원은
기본 사진 관리, 리뷰 관리, 월 리포트 포함이에요.
지금 가장 많이 선택하시는 옵션이에요.

집중 패키지 월 86만원은
여기에 블로그 리뷰 작성까지 포함해서
검색 노출 효과가 더 빠르게 나와요.

주목 패키지 월 127만원은
키워드 전략 + 콘텐츠 전체를 저희가 다 책임지는 거예요.
결과를 빠르게 보고 싶으신 분들이 선택하세요.

[잠깐 멈추고]

사장님 상황 봤을 때 저는 솔직히
[추천 패키지]가 맞을 것 같아요.
[이유 한 문장].
같이 보실까요?"
```

> 핵심: 3개 다 설명하되, 반드시 "저는 이거 추천해요"로 끝낼 것. 결정 못 하면 그냥 나옴.

## D. 자주 나오는 반론 대응 (5개)

---

### "혼자 해도 되는 거 아닌가요?"

```
"맞아요, 직접 하셔도 돼요. 사진 올리고, 리뷰 관리하고, 키워드 조사하고...
근데 이게 매주 꾸준히 해야 해서 시간이 생각보다 많이 들거든요.
사장님이 직접 하시는 거 vs 저희한테 맡기시는 거,
솔직히 사장님 시간이 더 비쌀 것 같아요.
한 달에 한 번 리포트 보시고 결과만 확인하는 게
더 효율적이지 않을까요?"
```

---

### "효과가 있어요?"

```
"당연히 보장은 못 드려요, 솔직하게 말씀드리면.
근데 이건 말씀드릴 수 있어요.
지금 사진이 [N]장이면 [N]장으로 늘리는 건 확실히 되고,
리뷰가 쌓이는 것도 숫자로 확인이 돼요.
그게 순위에 반영되는 데 보통 [4~8주] 정도 걸려요.
그래서 저희는 최소 3개월은 해보시길 권해요.
1개월 해보고 안 되면 바로 말씀해 주세요."
```

---

### "비싸네요"

```
"맞아요, 적은 돈은 아니에요.
근데 이렇게 한번 생각해보시면요.
'[메인키워드]'로 한 달에 [N]명이 검색해요.
지금 사장님한테 연결이 안 되고 있는 거잖아요.
그 사람들 중에 10명만 와도 얼마나 되실 것 같아요?
[업종별로] 한 명당 [객단가] 정도면 [N*객단가]잖아요.
광고비 관점에서 보시면 나쁘지 않을 거예요."
```

---

### "다른 업체랑 뭐가 달라요?"

```
"좋은 질문이에요. 차이가 2가지 있어요.
하나는, 저희는 시작할 때 이렇게 진단 리포트부터 드려요.
지금 어떤 상태인지 숫자로 보여드리고 시작하는 업체는 많지 않아요.
또 하나는, 매달 어떻게 변했는지 리포트로 드려요.
돈 쓰는데 뭐가 달라졌는지 모르면 불안하잖아요.
그 불안을 없애드리는 게 저희 방식이에요."
```

---

### "지금 바빠서요"

```
"네, 충분히 이해해요. 저도 사장님 바쁘신 거 알아서
자료 다 준비해서 온 거예요.
오늘 결정 안 하셔도 돼요.
자료만 두고 갈게요. 편하실 때 한번 보시고,
궁금하신 거 있으면 문자 주세요.
자료 자체가 도움이 되실 거예요."
```

> 핵심: 바쁘다는 건 거절이 아님. 결정 압박 없애고 자료 남기고 오는 것으로 마무리.

## E. 미팅 마무리 스크립트

### 계약 의향 있을 때

```
"좋아요. 그럼 이렇게 하죠.
오늘 결정하시면 이번 달부터 바로 시작하고,
첫 달 리포트를 [4주 후]에 보내드릴게요.
계약서는 간단해요. 제가 가져왔어요."
```

### 계약 의향 불분명할 때

```
"결정 급하게 안 하셔도 돼요.
오늘 자료 다시 한번 보시고, [3일 후]에 연락드릴게요.
그때 결정하셨으면 좋겠고, 아니어도 괜찮아요.
오늘 시간 내주셔서 감사했어요."
```

> 반드시 날짜 픽스하고 나올 것. "연락드릴게요"로 끝나면 안 됨.

## F. 빠른 참고 — 패키지 요약표

| 패키지 | 월 금액 | 핵심 포인트 |
|--------|---------|-------------|
| 시선 | 56만원 | 기본 관리 (사진+리뷰+월 리포트) |
| 집중 | 86만원 | 블로그 리뷰 포함 → 검색 노출 빠름 |
| 주목 | 127만원 | 키워드 전략 + 전체 콘텐츠 책임 |

**추천 공식:**
- 사진/리뷰 둘 다 부족 + 예산 타이트 → 시선
- 블로그 리뷰 0~5개 + 빠른 결과 원함 → 집중
- 경쟁 심한 업종 + 결과 확실히 원함 → 주목
