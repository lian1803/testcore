# 영업 퍼널 설계 — 비대면 텍스트 클로징 전용
> 만나지 않고 DM/문자/카톡만으로 계약까지

---

## 원칙

- **만남 없음.** 전화도 최소화. 텍스트로 끝낸다.
- **메시지 3번 안에 결정 유도.** 그 이상 가면 가망 없는 거다.
- **각 메시지는 딱 하나의 행동**만 요청한다.
- **숫자가 설득한다.** 말이 아니라 데이터가 계약을 만든다.

---

## 전체 흐름

```
1차 메시지 ─── 충격 숫자 1개 + "보내드려도 될까요?"
      ↓ 답장 오면
2차 메시지 ─── 진단 결과 전체 텍스트 발송 (카톡 카드 형식)
      ↓ 읽었으면
3차 메시지 ─── 기회비용 계산 + 패키지 + 결정 요청
      ↓ "생각해볼게요" or 무응답
4차 메시지 ─── 손실 강조 + 마감 기한 설정
      ↓ YES
계약 ─────── 구글폼 or 카카오페이 링크 발송
```

---

## 1차 메시지 — 충격 1개

**목표:** 답장 받기. 딱 하나.

**길이:** 3~4줄. 150자 이내.

**선택 로직 (업체 상황에 따라 자동 선택):**

```
경쟁사 리뷰 >= 내 리뷰 × 5  →  [A] 리뷰 격차형
순위 > 10 AND 월검색 >= 500  →  [B] 손실 고객형
새소식 >= 90일 OR 저장수 = 0  →  [C] 방치형
그 외                          →  [A] 기본형
```

### [A] 리뷰 격차형
```
{업체명} 사장님, 안녕하세요.
이 지역 미용실들 플레이스 분석하다가 연락드렸는데요.
이 동네 상위 업체 리뷰 평균이 {경쟁사평균}개인데
사장님 가게는 {내리뷰}개더라고요.
진단 결과 보내드려도 될까요?
```

### [B] 손실 고객형
```
{업체명} 사장님, 안녕하세요.
'{메인키워드}' 지금 {순위}위이신데, 이 위치면
매달 약 {손실고객}명 정도가 상위 업체로 가고 있어요.
무료 진단 결과 보내드려도 될까요?
```

### [C] 방치형
```
{업체명} 사장님, 안녕하세요.
플레이스 새소식이 {N}일째 없으신 걸 봤는데,
네이버가 요즘 이걸 되게 중요하게 보거든요.
진단 결과 보내드려도 될까요?
```

**응답 없으면:** 3일 후 1회만 재발송. 그 이후엔 다음 업체로.

## 2차 메시지 — 진단 결과 전체 발송

**목표:** "문제가 있다" 인식시키기. 솔루션은 아직 꺼내지 마라.

**조건:** 1차에 "네", "보내주세요", "어떤 거요" 등 긍정 답장 온 경우에만.

**형식:** 카톡에서 잘 읽히는 텍스트 카드 형태. 이모지로 시각화.

### 템플릿
```
📊 [{업체명}] 네이버 플레이스 진단 결과입니다.

━━━━━━━━━━━━━━━━━━
⭐ 종합 점수: {점수}점 / 95점 ({등급}등급)
   이 지역 상위 업체 대비 {상대점수}% 수준
━━━━━━━━━━━━━━━━━━

✅ 잘 되어 있는 것
{잘된항목들 나열 — 최대 3개만}

❌ 지금 비어있는 것 ({X}개)
{취약항목1}: {현재값} → 기준 {기준값}
{취약항목2}: {현재값} → 기준 {기준값}
{취약항목3}: {현재값} → 기준 {기준값}
{나머지 취약항목들 간단히}

📊 경쟁사 비교
  리뷰: 사장님 {내리뷰}개  |  경쟁사 평균 {경쟁사평균}개
  사진: 사장님 {내사진}장  |  경쟁사 평균 {경쟁사사진}장
━━━━━━━━━━━━━━━━━━
{손실고객 > 0이면:}
⚠️ 현재 순위 기준, 매달 약 {손실고객}명이
   경쟁사로 가고 있는 걸로 추정돼요.
━━━━━━━━━━━━━━━━━━

어떻게 생각하세요?
```

**이 메시지에서 가격/패키지 절대 언급 금지.**
상대가 "어떻게 해요?" 물어봐야 3차로 넘어간다.

## 3차 메시지 — 솔루션 + 가격 + 결정 요청

**목표:** 계약. 이 메시지에서 끝낸다.

**조건:** 2차 읽고 반응 온 경우 (질문, "오" 등 어떤 반응이든)

**구성:**
1. 핵심 문제 1줄 요약 (2차 내용 상기)
2. 솔루션 설명 (1~2줄)
3. 패키지 3가지 (가격 포함)
4. 추천 패키지 명시
5. 기회비용 계산
6. 행동 요청 (YES/NO 질문)

### 템플릿
```
사장님, 진단 결과 정리해드리면

지금 가장 급한 게 {취약TOP1}이랑 {취약TOP2}인데요,
이 두 가지만 잡아도 순위가 달라져요.

저희 서비스 3가지 플랜이에요:

─────────────────────
🔹 주목 플랜  290,000원/월
  기본정보 최적화 + 키워드 등록 + 월 리포트

🔸 집중 플랜  490,000원/월
  주목 + 사진 촬영 + 블로그 리뷰 관리
  + 사장님 답글 대행

⭐ 시선 플랜  890,000원/월
  집중 + 인스타 관리 + 새소식 2회/월
  + 전담 매니저
─────────────────────

사장님한테는 {추천플랜}이 맞을 것 같아요.

{손실고객 > 0이면:}
지금 매달 {손실고객}명이 경쟁사로 가고 있는데,
{추천플랜} 한 달 비용이 {플랜금액}원이거든요.
고객 {손익분기고객수}명만 더 오시면 본전이에요.

한번 해보실 의향 있으세요?
```

**손익분기 계산:** `ceil(플랜금액 / 업종평균객단가)`
- 미용실 기준 평균 객단가 5~8만원 → 집중 플랜이면 7~10명
- 이 숫자가 작을수록 계약률 올라간다

## 4차 메시지 — 클로징 (무응답/보류 대응)

**조건:** 3차 발송 후 2~3일 무응답 or "생각해볼게요"

**목표:** 결정 촉구. 마지막 1번.

### [무응답/보류용]
```
사장님, 혹시 확인하셨나요?

저희가 이번 달 신규 업체 {N}곳 더 받으면 마감인데,
사장님 지역 경쟁사 중 {1~2곳}도 문의 주셨거든요.

같은 지역 업체를 동시에 진행하진 않아서요,
이번 달 안에 결정해주시면 {지역} 독점으로 진행해드릴 수 있어요.

어떻게 하시겠어요?
```

### [비싸다고 할 때]
```
그 마음 충분히 이해해요.

근데 계산해보면요,
{추천플랜} 한 달 {플랜금액}원인데
지금 매달 {손실고객}명이 경쟁사로 가고 있고
한 명당 {객단가}원이면 {손실금액}원이거든요.

안 하는 게 더 비싼 거예요.

일단 3개월만 해보시고,
효과 없으면 그때 말씀해주세요.
```

### [직접 해보겠다고 할 때]
```
당연히 하실 수 있어요.

근데 아까 진단 결과에서 X 뜬 항목이 {X}개였는데,
하나하나 다 시간이 걸리는 거라서요.
사장님이 직접 하시는 데 매주 {N}시간은 써야 해요.

그 시간에 손님 보시는 게 낫지 않으세요?
저희가 대신 할게요.
```

## 계약 클로징 메시지

**조건:** YES 답장 왔을 때

```
감사합니다 사장님!

절차 간단해요, 두 가지만 하시면 돼요.

① 계약서 서명 (3분)
   아래 링크 클릭하시면 핸드폰으로 바로 서명 가능해요.
   📋 {글로싸인 링크}

② 첫 달 비용 입금
   {플랜금액}원
   계좌: {은행} {계좌번호} ({예금주})

서명 + 입금 확인되면 담당자가 연락드리고
바로 세팅 들어갈게요.

궁금한 거 있으시면 편하게 물어보세요!
```

**운영 포인트:**
- 글로싸인 링크는 업체별로 미리 생성해두거나, 공용 템플릿 링크 1개 고정 사용
- 입금 확인 전까지는 착수하지 않음
- 서명만 하고 입금 안 한 경우 → 다음날 "입금 확인이 안 됐는데 혹시 불편하신 게 있으셨나요?" 1회 추가 발송

## 메시지 발송 타이밍

| 메시지 | 발송 시간 | 이유 |
|--------|-----------|------|
| 1차 DM | 오전 10~11시 | 오전 영업 준비 시간, 폰 많이 봄 |
| 2차 (답장 후) | 즉시 or 1시간 이내 | 관심 있을 때 바로 |
| 3차 | 2차 발송 다음날 오전 | 하루 생각할 시간 줌 |
| 4차 | 3차 후 2~3일 | 너무 빠르면 부담, 너무 늦으면 식음 |

---

## 이 퍼널에서 버려야 할 것들

| 하면 안 되는 것 | 이유 |
|----------------|------|
| 첫 메시지에 가격 언급 | 바로 차단 |
| 미팅 먼저 제안 | "바쁜데요"로 끝남 |
| 3차 이후 계속 팔로업 | 스팸 이미지, 관계 망가짐 |
| "혜택" 남발 | 신뢰 감소 |
| 긴 메시지 | 모바일에서 안 읽힘 |
| 링크만 달랑 보내기 | 클릭 안 함 |

## 업종별 객단가 기준 (손익분기 계산용)

| 업종 | 평균 객단가 | 집중 플랜 손익분기 |
|------|------------|-------------------|
| 미용실 | 5~8만원 | 7~10명 |
| 네일샵 | 5~7만원 | 7~10명 |
| 식당/카페 | 1.5~3만원 | 17~33명 |
| 피부관리 | 8~15만원 | 4~7명 |
| 학원 | 20~40만원 | 2~3명 |
| 세탁소 | 1~3만원 | 17~50명 |
