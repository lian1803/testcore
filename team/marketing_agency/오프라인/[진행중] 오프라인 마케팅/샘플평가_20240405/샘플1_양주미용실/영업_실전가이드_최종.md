# 소상공인 비대면 영업 실전 가이드

> 폰에 저장해놓고 복붙해서 바로 써라.
> 하루 50건 발송 / 오전 10시 / 3시간이면 끝.
> 패키지: 주목 29만 / 집중 49만 / 시선 89만 (월, 3개월 이상 계약)

---

## 0. 누구한테 먼저 연락할까

### S급 — 오늘 당장 연락

아래 중 3개 이상 해당하는 업체:
- 사진 10장 미만
- 리뷰 10개 미만
- 새소식 90일 넘게 없음
- 검색 순위 11위 이하
- 블로그 리뷰 5개 미만
- 사장님 답글 0개

이런 업체가 S급이다. 진단 결과가 충격적이라 1차 메시지 임팩트가 가장 크다.

### A급 — 이번 주 연락

아래 중 2개 이상:
- 등급 C (40~59점)
- 경쟁사 대비 리뷰 1/3 이하
- 사진 20장 미만
- 순위 4~10위 (아슬아슬해서 "조금만 올리면" 설득 가능)

### B급 — 여유 있을 때

- 등급 B (60~79점)
- 특정 항목 1~2개만 약한 경우

### 패스 — 연락 안 함

- 등급 A (80점 이상) -- 이미 잘하고 있어서 필요성 못 느낌
- 최신 리뷰 1년 이상 전 -- 폐업 가능성
- 프랜차이즈 본사 직영 -- 결정권자가 아님
- 리뷰에 "폐업", "이사" 키워드

## 1. 1차 메시지 -- 첫 접촉 (답장 받기)

**목표: "네" 또는 "보내주세요" 한마디 받기. 그것만.**

### 어떤 버전 쓸지 판단 기준

```
경쟁사 중 최근 3개월 리뷰 급증 업체 있음  →  D형
경쟁사 리뷰 >= 내 리뷰 x 5              →  A형
순위 > 10 AND 월검색 >= 500             →  B형
새소식 >= 90일 OR 저장수 = 0            →  C형
위에 다 해당 안 되면                      →  A형
```

### [A] 리뷰 격차형 -- 카카오톡용

```
{업체명} 사장님, 안녕하세요.
{지역} {업종} 플레이스 분석하다가 연락드려요.
경쟁사 리뷰 평균 {경쟁사리뷰}개인데
사장님은 {내리뷰}개더라고요.
분석 결과 보내드려도 될까요?
```

### [A] 리뷰 격차형 -- 문자용 (90자 이내)

```
{업체명} 사장님, {지역} {업종} 플레이스 분석했는데
경쟁사 리뷰 평균 {경쟁사리뷰}개, 사장님 {내리뷰}개예요.
진단 결과 보내드려도 될까요?
```

### [B] 손실 고객형 -- 카카오톡용

```
{업체명} 사장님, 안녕하세요.
{메인키워드} 검색했을 때 사장님 가게가
지금 {순위}위인데요,
이 위치면 매달 약 {손실고객}명이
사장님 가게 대신 상위 업체로 가고 있어요.
상세 분석 보내드려도 될까요?
```

### [B] 손실 고객형 -- 문자용

```
{업체명} 사장님, {메인키워드} 검색 {순위}위인데
매달 약 {손실고객}명이 상위 업체로 가고 있어요.
분석 결과 보내드릴까요?
```

### [C] 방치형 -- 카카오톡용

```
{업체명} 사장님, 안녕하세요.
{지역} {업종} 플레이스 분석하다 봤는데요,
새소식이 {방치일수}일 동안 업데이트가 없어서
순위에 영향 가고 있더라고요.
현재 상태 진단 보내드려도 될까요?
```

### [C] 방치형 -- 문자용

```
{업체명} 사장님, 플레이스 새소식이 {방치일수}일째 없어서
순위에 영향 가고 있어요.
진단 결과 보내드릴까요?
```

### [D] 경쟁사 움직임형 -- 카카오톡용 (답장률 가장 높음)

```
{업체명} 사장님, 안녕하세요.
이 동네 {업종} 플레이스 분석하고 있는데요,
{지역} {업종} 중에 최근 3개월 사이 리뷰가
갑자기 늘어난 업체가 {N}곳 있더라고요.
사장님 가게 현재 상태도 같이 봤는데, 보내드려도 될까요?
```

### [D] 경쟁사 움직임형 -- 문자용

```
{업체명} 사장님, {지역} {업종} 중에 최근 3개월
리뷰 급증 업체가 {N}곳 있어요.
사장님 가게 진단도 봤는데 보내드릴까요?
```

### 1차 규칙

| 항목 | 규칙 |
|------|------|
| 보내는 시간 | 오전 10시 (답장률 15%로 가장 높음) |
| 카카오톡 | 150자 이내, 4줄 이내 |
| 문자 | 90자 이내 (MMS 되면 안 읽음) |
| 이모지 | 쓰지 마라 (광고 느낌) |
| 링크 | 절대 넣지 마라 |
| 마지막 문장 | "보내드려도 될까요?" (허락 요청) |
| 가격/서비스 | 절대 언급 금지 |
| 010만 있으면 | 문자 먼저, 답장 오면 카톡 전환 유도 |

## 2. 2차 메시지 -- 진단 결과 발송 (충격 주기)

**목표: "어떻게 해요?" 를 끌어내기. 해결법은 절대 안 준다.**

답장이 왔으면 바로 보내라. 순서가 중요하다 (PAS 구조):

```
{업체명} 사장님, 진단 결과 보내드려요.

━━━━━━━━━━━━━━━━━━
종합 {등급}등급 ({점수}점/95점)
이 지역 {업종} 상위 업체 평균 대비 {상대점수}% 수준이에요.
━━━━━━━━━━━━━━━━━━

경쟁사 비교:
  리뷰  사장님 {내리뷰}개 / 경쟁사 평균 {경쟁사평균}개
  사진  사장님 {내사진}장 / 경쟁사 평균 {경쟁사사진}장
  블로그  사장님 {내블로그}개 / 경쟁사 평균 {경쟁사블로그}개

지금 '{메인키워드}' 검색하면 {순위}위인데,
이 위치면 매달 약 {손실고객}명이
사장님 가게 대신 상위 업체로 가고 있어요.
금액으로 치면 월 {손실금액}만원 정도요.

그래도 {잘된항목} 부분은 잘 되어 있어요.

지금 가장 급한 건
{취약1}이랑 {취약2}이에요.
여기만 잡아도 순위가 달라질 수 있어요.

어떻게 생각하세요?
```

### 2차 규칙

| 항목 | 규칙 |
|------|------|
| 한 문단 | 최대 3줄 |
| 빈 줄 | 문단 사이 필수 |
| 구분선(━) | 핵심 숫자 섹션에 사용 |
| 전체 길이 | 20줄 이내 (스크롤 1번에 끝나야 함) |
| PPT | 2차에서 안 보냄 |
| 잘된 것 | 1~2개만 (숨통 틔워주기) |
| 안 된 것 | 핵심 3개만 (전부 나열 금지) |
| 해결법 | 절대 안 알려줌 ("여기만 잡아도" 수준까지만) |
| 마지막 문장 | "어떻게 생각하세요?" (열린 질문) |

## 3. 3차 메시지 -- 패키지 제안 (돈 얘기)

**목표: 집중 플랜 49만원 계약. 이중선택 CTA로 끝내기.**

상대가 "어떻게 해요?" 또는 관심 표현하면 바로 보내라:

```
사장님, 어제 보내드린 진단 결과 보셨죠.

지금 가장 급한 게 {취약TOP1}이랑 {취약TOP2}인데요,
이 두 가지만 잡아도 순위가 확 달라져요.

저희 서비스 3가지인데요:

━━━━━━━━━━━━━━━━━━
시선 플랜  890,000원/월
  전담 매니저 + 인스타 관리 + 새소식
  + 사진/리뷰/블로그 전부 책임

집중 플랜  490,000원/월
  사진 촬영 + 블로그 리뷰 관리
  + 사장님 답글 대행 + 월 리포트

주목 플랜  290,000원/월
  기본정보 최적화 + 키워드 등록
  + 월 리포트
━━━━━━━━━━━━━━━━━━

사장님한테는 집중 플랜이 맞아요.
{취약TOP1} 잡으려면 {관련서비스}가 필요한데,
주목에는 이게 안 들어가거든요.

계산해보면요,
지금 매달 {손실고객}명이 경쟁사로 가고 있고
{업종} 평균 객단가 {객단가}만원이면
월 {손실금액}만원을 놓치고 있는 거예요.

집중 플랜 49만원인데, 고객 {손익분기}명만 돌아오면 본전이에요.
나머지는 전부 순이익이고요.
안 하는 게 매달 {손실금액}만원 버리는 거예요.

기본 3개월 계약이고요,
첫 달에 세팅 + 기본 개선 끝내드리고
2개월 차에 순위 변화 리포트 드릴게요.

집중 플랜이랑 주목 플랜 중에 어떤 게 더 맞으실 것 같아요?
```

### 손익분기 계산 참고

| 업종 | 평균 객단가 | 집중 49만원 손익분기 | 주목 29만원 손익분기 |
|------|------------|---------------------|---------------------|
| 미용실/네일 | 6만원 | 고객 9명 | 고객 5명 |
| 식당/카페 | 2만원 | 고객 25명 | 고객 15명 |
| 피부관리 | 10만원 | 고객 5명 | 고객 3명 |
| 학원 | 30만원 | 고객 2명 | 고객 1명 |

### 3차 규칙

| 항목 | 규칙 |
|------|------|
| 패키지 순서 | 89만 → 49만 → 29만 (앵커링, 비싼 것 먼저) |
| 추천 패키지 | 무조건 집중 플랜 49만원 (주목은 서비스 빈약, 시선은 텍스트론 무거움) |
| 손실 금액 먼저 | 손실 금액 말한 뒤에 패키지 가격 (49만원이 싸게 느껴짐) |
| CTA | "집중이랑 주목 중 어떤 게 더 맞으세요?" (이중선택) |
| 계약 기간 | "기본 3개월" (월 단위 없음. 29/49/89가 이미 할인가) |
| 해결 방법 | 여전히 구체적으로 안 알려줌 ("잡아드릴게요" 수준까지만) |

## 4. "생각해볼게요" 대응

비대면에서 이 답변이 80%다. 여기서 못 잡으면 끝이다.

### 즉시 대응 (답장 온 10분 이내)

```
네, 당연히 생각해보셔야죠.
혹시 가격 때문에 고민이세요, 아니면 효과가 확실한지 궁금하신 건가요?
```

이 질문으로 진짜 이유를 파악하고 아래처럼 대응:

### "가격이요" → 손익분기 다시

```
그 마음 충분히 이해해요.

근데 계산해보면요,
집중 플랜 49만원인데
지금 매달 놓치고 있는 고객이 {손실고객}명이에요.
한 명당 {객단가}만원이면 월 {손실금액}만원이거든요.

49만원 투자해서 그 중 {손익분기}명만 돌아와도 본전인데,
안 하면 매달 {손실금액}만원을 계속 놓치는 거예요.

부담되시면 주목 플랜 29만원부터 시작해보시는 것도 방법이에요.
```

### "효과요" → 3개월 보장

```
솔직히 "무조건 된다"고 말씀드리긴 어려워요.

근데 확실한 건,
지금 사장님 사진이 {N}장인데 이걸 {목표}장으로 늘리는 건 되고,
리뷰 관리해서 쌓이는 것도 숫자로 보여드려요.
매달 리포트로 뭐가 바뀌었는지 확인하시고요.

3개월 해보시고 효과 없으시면 그때 말씀해주세요.
첫 달에 세팅 끝내드리니까 2개월 차부터 변화 보이기 시작해요.
```

### "그냥 좀..." → 날짜 잡기

```
네, 편하게 생각해보세요.
그러면 이번 주 금요일쯤 다시 연락드려도 될까요?
```

### 시간차 대응 순서

| 시점 | 뭘 보내냐 |
|------|-----------|
| "생각해볼게요" 직후 | 위 즉시 대응 멘트 |
| 즉시 대응 후 무응답 48시간 | "사장님, 혹시 결정하셨어요? 궁금한 점 있으시면 편하게 물어보세요." |
| 그 후 무응답 3일 | PPT 이미지 3장 보내며 "참고로 이 자료도 보내드려요." |
| 그 후 무응답 5일 | 5번(4차 FOMO)으로 이동 |

### PPT 이미지 3장 (보낼 때 JPG로 변환해서)

1. 경쟁사 비교 슬라이드 (시각적 격차가 충격)
2. 종합 진단 점수 슬라이드 (D등급 빨간색)
3. 손익분기 계산 슬라이드 (숫자로 설득)

## 5. 4차 메시지 -- 마지막 기회

### FOMO 1: 경쟁사 움직임 (사실일 때만)

```
사장님, 참고로 말씀드리면
{지역} {업종} 중에 최근에 플레이스 관리 시작한 업체가 있어요.
리뷰가 한 달 만에 {N}개 늘었더라고요.
격차가 더 벌어지기 전에 시작하시는 게 좋을 것 같아요.
```

### FOMO 2: 지역 독점 (실제일 때만 -- 거짓이면 신뢰 무너짐)

```
저희가 같은 지역 같은 업종을 동시에 진행하지 않거든요.
{지역} {업종}은 아직 비어있는데,
다른 업체에서 먼저 시작하시면 사장님은 대기해야 해요.
```

### FOMO 3: 마감 기한

```
이번 주 금요일까지 결정해주시면
다음 주부터 바로 세팅 들어갈 수 있어요.
그 이후엔 다음 달로 넘어가야 해서요.
```

### 4차 사용 규칙

- 경쟁사 움직임: 실제 데이터 있을 때만
- 지역 독점: 실제로 같은 지역 다른 업체 상담 중일 때만. 남발 금지
- 마감 기한: 언제든 쓸 수 있지만 말한 기한은 지켜라
- 3개 중 상황에 맞는 1개만 보내라. 몰아서 보내지 마라

## 6. 클로징 -- "할게요" 이후

**"할게요" 오면 5분 이내에 바로 보내라. 시간 끌면 마음 변한다.**

```
좋아요 사장님!

절차 간단해요.

1. 계약서 서명 (핸드폰으로 3분)
   {글로싸인 링크}

2. 첫 달 입금
   {금액}원
   {은행} {계좌번호} ({예금주})

서명이랑 입금 확인되면
바로 세팅 시작할게요.
```

### 입금 안 하면

| 시점 | 멘트 |
|------|------|
| 서명 완료 + 입금 안 됨 (다음날 오전) | "사장님, 입금 확인이 안 됐는데 혹시 불편하신 게 있으셨나요?" |
| 그래도 미입금 (2일 후) | "이번 달 세팅 마감이 {N}일인데, 혹시 일정 맞추실 수 있으세요?" |
| 그 이후 미입금 | 포기. 더 연락하면 스팸이 된다 |

### 계약 기본사항

- 3개월 이상 계약 (월 단위 없음)
- 29/49/89가 이미 최저 할인가 (추가 할인 없음)
- 결제: 카카오페이 송금 링크(가장 빠름) > 계좌이체 > 토스페이

## 7. 4차 이후 -- 장기 nurturing

4차까지 안 됐으면 포기가 아니라 nurturing으로 전환.
영업 냄새 제거하고 정보성 메시지만 보낸다.

### 2주 후

사장님, 요즘 네이버 플레이스 알고리즘이 바뀌어서
새소식 업데이트 주기가 순위에 영향 커졌어요.
참고하세요!
```

### 1개월 후

```
사장님, 지난번 진단해드린 거 기억하시나요?
그때 {순위}위였는데 지금 확인해보니 {현재순위}위예요.

(올랐으면 "좋아지셨네요!" / 내렸으면 "좀 더 내려갔네요." 추가)

이후 완전 종료. 2개월 후에 역으로 연락 오는 비율이 5~8%고, 그때는 이미 Warm 상태라 클로징률 50% 이상이다.

## 8. 방문 미팅 -- 이럴 때만 가라

### 방문 트리거 4가지 (하나라도 해당하면 가라)

1. **"직접 봐야 믿겠다"** -- 의심이 높음. 텍스트론 안 됨. PPT 보여주면서 전문성 증명
2. **시선 플랜(89만원) 관심** -- 고가 계약은 대면이 계약률 높음
3. **3차까지 긍정인데 결제 안 함** -- 마지막 한 방. 현장 서명 유도
4. **"우리 가게 한번 와보세요"** -- 사장님이 초대. 거절할 이유 없음

### 방문하지 마라

- 주목 플랜(29만원) 관심 -- 왕복 시간 대비 ROI 낮음
- 1~2차 반응 미지근 -- 가봐야 안 됨
- 편도 40분 이상 -- 비대면으로 끝내거나 포기

### 방문 시 추가로 보여줄 것

| 자료 | 왜 |
|------|-----|
| PPT 전체 슬라이드 | 한 장씩 넘기며 설명하는 게 임팩트 있다 |
| 경쟁사 플레이스 실제 화면 | 노트북으로 실시간 비교 |
| 개선 전/후 시뮬레이션 | "사진 50장 올리면 이렇게 바뀝니다" |
| 기존 고객 사례 (있으면) | "이 미용실이 3개월 전에 시작해서 지금 이렇게 됐어요" |

## 11. 첫 대면 영업 — 15분 구조

방문 미팅이 진행되면 반드시 이 구조로. 15분 넘으면 약해진다.

| 시간 | 액션 | 핵심 원칙 |
|------|------|---------|
| 0~2분 | "사장님 가게 네이버에서 미리 봤는데, 한 가지만 보여드려도 될까요?" | 심리의 벽 열기 |
| 2~5분 | 핸드폰으로 경쟁사 비교 (PPT X) | 핸드폰이 핵심. PPT = "영업이구나" 벽 생김 |
| 5~8분 | 원인 1개만 설명 | 설명 과다는 역효과 |
| 8~10분 | 사장님 반응 듣기 | 여기서 80%의 정보가 나온다 |
| 10~12분 | 29만원 패키지만 제안 | 3패키지 동시 X → 89만원으로 눈 간다 → 거부 |
| 12~15분 | 클로징 or 연락처 확보 | 계약 확률 5~10%, 진짜 목표는 2차 약속 |

### 방문 미팅 중 금지 사항

- **PPT 보여주기** — 핸드폰 1개, 종이 진단 1장만. PPT는 답장 후 비대면에서만
- **모든 문제점 설명** — 1개만. "이 부분이 가장 아쉬워요" 수준
- **가격 3개 동시 제시** — 절대 금지. 29만원만. 2차 이상에서 3개 제시
- **노트북 펼치기** — 이미 신뢰가 없다는 신호

### 방문 미팅 준비물

| 준비물 | 쓰임새 |
|-------|--------|
| A4 진단 리포트 (인쇄) | 사장님 손에 쥐어주기. "이거 보관하세요" |
| 핸드폰 1개 (경쟁사 화면) | 실시간 비교. 경쟁사 1위/2위/3위 미리 저장 |
| 계약서 1장 | "만약 진행하신다면" 보여주기만 (서명 안 함) |
| 사례 프린트 (있으면) | "비슷한 가게가 이렇게 바뀌었어요" |

## 12. 거절 처리 — 5가지 현장형

방문 미팅 또는 비대면 대화 중 거절이 나올 때, 절대 금지 멘트와 정답 대응:

### ① "괜찮아요, 안 해도 돼요"

**❌ 금지**: "요즘 네이버 플레이스가 중요합니다", "투자라고 생각하세요"

**✅ 대응**: 
```
"네, 맞아요. 그냥 이것만 보시고 판단하세요."
→ 핸드폰으로 경쟁사 비교 1장 30초
→ "보셨죠? 사장님은 {내리뷰}개, 경쟁사는 {경쟁사리뷰}개예요."
→ 반응 대기
```

**반응에 따라:**
- 조용함/관심 없음 → "그럼 이거 두고 갈게요" (진단 리포트 놓고 나감)
- "흠..." → "생각해보세요. 좋으면 연락 주세요"

---

### ② "돈이 없어요"

**❌ 금지**: "투자라고 생각하면", "수익이 훨씬 크니까"

**✅ 대응**:
```
"제가 지금 바로 할 수 있는 거 하나 알려드릴게요.
{취약항목} 이 부분, 무료로 개선하는 방법 하나만."
```

**그리고 무료 팁 1개만 알려주기:**
- "사진이 부족하면, 근처 가게 사진들을 참고하고, 스마트폰으로 이 각도로..."
- 30초 팁 끝 → "이렇게라도 해보세요" → 나감
- **2주 후 재방문** (돈 모았을 가능성)

---

### ③ "전에 해봤는데 효과 없었어요"

**❌ 금지**: "저희는 달라요", "그건 제대로 하신 게 아니에요"

**✅ 대응**:
```
"어떤 방식이었어요?"
→ (들어라)
"그건 솔직히 그 방식이면 효과 없는 게 맞아요. 
네이버 알고리즘이 그렇거든요."
```

**신뢰 구축:**
```
"저도 처음엔 비슷한 방식했는데 안 되더라고요.
근데 {구체적인 다른 포인트}를 바꾸니까 달랐어요.
전 이 방식으로 해요."
```

**핵심: 전 업체를 욕하지 않되, 사장님이 느낀 실패를 인정해주기**

---

### ④ "생각해볼게요"

**❌ 금지**: "언제쯤 연락 주실 수 있을까요?", "내일까지 결정해주시면..."

**✅ 대응** (비대면 답장 기준):
```
"당연하죠. 진단 리포트 여기 두고 갈게요.
혹시 궁금한 게 생기면 카톡이나 문자 주세요.
{전화번호}
```

**진짜 목표: 진단 리포트를 사장님이 5분이라도 보게 하기**

방문 미팅에서 나왔다면:
```
"당연히 생각해보셔야죠.
궁금한 점 있으면 전화주세요.
혹시 가격 때문에 고민이세요, 아니면 효과가 확실한지 궁금하신 건가요?"
```

---

### ⑤ (아예 바쁘고 말 안 들을 때)

**❌ 금지**: 계속 설명하기, 고집 피우기

**✅ 유일한 대응**:
```
"사장님, 이거 하나만 두고 갈게요."
→ A4 진단 리포트 1장 놓고 나온다
→ 여기서 끝
```

**이게 진짜 고급 스킬이다.**
- 강하게 나가지 않아서 거부감 X
- 리포트를 남겨서 2차 기회 생성
- 다른 사장님 영업 시간 절약

## 13. 가격 프레이밍 원칙

같은 가격이어도 표현 방식에 따라 구매 의지 60% 차이.

### 절대 금지 멘트

| 잘못된 표현 | 왜 안 되는가 |
|----------|-----------|
| "월 29만원입니다" | 부담스러움 → 즉시 거부 |
| 3패키지 동시 소개 | 선택 과부하 → 가장 비싼 것에서 눈 감음 → 중간 것도 비싸 보임 |
| "투자라고 생각하세요" | 너무 뻔함. 전형적 영업 톤. 신뢰 깨짐 |
| "효과 없으면 환불해드려요" | 자신감 없어 보임 → 오히려 회의 증가 |

### 정답 표현

| 올바른 표현 | 왜 먹히는가 |
|----------|-----------|
| "하루 만원도 안 돼요" | 월 개념 → 일 개념으로 축소. 심리적 부담 1/30 |
| 첫 제안은 29만원만 | 선택지 1개 → 고민할 거 없음. 다음 단계에서 업셀 |
| "지금도 경쟁자가 손님 가져가고 있어요" | 행동 촉구. 기회비용 강조. 투자 아니라 손실 회피 |
| "2주 안에 노출 변화 없으면 말씀하세요" | 자신감 + 사장님 판단 존중. 신뢰 구축 |

### 패키지별 프레이밍

#### 주목 플랜 (29만원)

**❌**: "기본 패키지입니다"
**✅**: "처음 시작하실 때 가장 많이 선택하는 플랜이에요"
(이미 많은 사람이 했다 = 검증됨 = 안전)

---

#### 집중 플랜 (49만원) — 추천 패키지

**❌**: "풀 서비스 패키지입니다"
**✅**: "사진과 리뷰, 이 두 가지만 잡아도 순위가 많이 올라가거든요.
그래서 대부분 집중 플랜 선택하세요"
(이미 많은 사람이 했다 + 구체적 이유)

---

#### 시선 플랜 (89만원)

**❌**: "프리미엄 패키지입니다", "최고급 서비스입니다"
**✅**: "이 지역 상위 3개 업체가 다 집중 이상으로 하고 있어요.
그 수준으로 따라잡으려면 이 플랜이 필요해요"
(경쟁 상황 강조 + 현실적 필요성)

---

### 손익분기 계산 표현

**❌**: "집중 플랜이 월 49만원이고요, 고객 9명 정도면 본전이에요"
(추상적. 사장님이 "9명이 쉬워?"라고 생각)

**✅**: "지금 매달 {손실고객}명이 경쟁사로 가고 있는데,
이 중 {손익분기}명만 돌아와도 본전이에요.
{손익분기}명은 진짜 가능한 수준이거든요"
(구체성 + 가능성 강조 + 현실감)

---

### 기간 프레이밍

**❌**: "월별로도 가능합니다"
**✅**: "기본 3개월 계약이에요. 
첫 달에 기본 세팅 끝내드리고,
2~3개월 차에 순위 변화가 나타나기 시작해요"
(기간 필요성 설명 + 기대값 관리)
