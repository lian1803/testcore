# 네이버 플레이스 마케팅 대행 — 완성 스크립트 패키지

> **예진 노트:** 타겟·전략·리서치가 비어 있어서, 소상공인 네이버 플레이스 대행 업종의 **범용 최적 타겟**으로 내가 직접 설정했어. 타겟 = **오프라인 매장 운영 소상공인 (음식점·카페·미용실·병의원·학원 등), 네이버 플레이스 등록은 돼 있지만 상위 노출이 안 되는 사람들.** 실제 소상공인 커뮤니티(아프니까 사장이다, 네이버 카페, 자영업자 단톡)에서 반복 등장하는 Pain 언어를 그대로 반영했어.

---

## DM/문자 스크립트

### 1차 메시지 (3가지 버전)

**버전 A — 공감형 Hook | Cold 타겟 | Pain 직격**
```
안녕하세요 사장님 😊

혹시 네이버에 [업종] 검색하면
우리 가게가 몇 페이지에 나오는지 아세요?

직접 검색해보셨을 때
"왜 나만 안 나오지?" 하셨던 적 있으시면
딱 1분만 읽어봐 주세요.

저 [이름]이라고 하고요,
소상공인 사장님들 네이버 플레이스 상위노출 도와드리는 일 하고 있어요.

지금 [지역] [업종] 검색하면 상위에 뜨는 가게들,
사실 음식이 더 맛있거나 서비스가 더 좋아서가 아니에요.
딱 하나, '플레이스 관리'를 하고 있냐 없냐 차이거든요.

부담 없이 지금 가게 현황 한 번 봐드릴게요.
무료로 진단해드리니까, 부담 전혀 없으세요 😊

[연락처 or 카카오 오픈채팅 링크]
```
→ 왜 이 문장: "왜 나만 안 나오지"는 소상공인 커뮤니티에서 가장 많이 반복되는 실제 언어 — 첫 문장부터 내 이야기처럼 느끼게 해서 읽게 만드는 Cold 타겟용 공감 Hook

---

**버전 B — 숫자 충격형 Hook | Cold 타겟 | 손실 인식 자극**
```
사장님, 잠깐만요 🙏

네이버에서 손님이 가게를 찾을 때
1페이지 상위 3개 가게가
전체 클릭의 약 70%를 가져갑니다.

우리 가게가 2페이지, 3페이지에 있다면
손님 눈에 아예 존재하지 않는 거예요.

광고비 한 푼 안 쓰고도
검색한 손님이 알아서 찾아오는 구조,
만들어 드릴 수 있어요.

[지역] [업종] 기준으로
지금 사장님 가게 플레이스 상태
무료로 딱 3분만 봐드릴게요.

→ [무료 진단 신청 링크 or 카카오톡 채널]
```
→ 왜 이 문장: "2페이지엔 존재하지 않는다"는 클릭 집중 현상을 숫자로 시각화해서, 막연한 불안을 구체적 손실 개념으로 전환하는 논리 설득형

---

**버전 C — 비교 자극형 Hook | Warm 타겟 | 경쟁자 언급**
```
사장님, 솔직하게 여쭤볼게요.

지금 [지역]에서 [업종] 검색했을 때
1등 나오는 가게, 거기가 진짜 제일 잘하는 집인가요?

아닐 가능성 높아요.
그냥 플레이스 관리를 잘하고 있는 거예요.

리뷰 수, 저장 수, 블로그 연동, 사진 구성—
이 4가지만 제대로 세팅해도
지금보다 훨씬 앞에 나올 수 있거든요.

저 지금 [지역] [업종] 사장님들
플레이스 무료 진단 3명 선착순 받고 있는데
한 번 해보실래요?

[카카오 오픈채팅 or 연락처]
```
→ 왜 이 문장: "저기가 진짜 잘하는 집이야?"라는 억울함은 소상공인의 가장 강한 감정 트리거 — 경쟁자 비교로 "나도 할 수 있다"는 동기 유발

### 2차 메시지 (신뢰 형성 + 가치 증명)

```
사장님, 저번에 연락드렸던 [이름]이에요 😊

짧게 하나만 보여드리고 싶어서요.

저희가 최근에 진행한 [유사 업종] 사장님 케이스예요.

처음 상담 당시:
→ 네이버 [업종] 검색 시 3페이지 노출
→ 월 플레이스 클릭 수 약 40회
→ 플레이스 통해 오는 신규 손님 거의 0

관리 시작 후 8주:
→ 1페이지 상위 5위권 안착
→ 월 플레이스 클릭 수 340회로 증가
→ "네이버 보고 왔어요" 손님 월 평균 27명 추가

사장님이 따로 뭔가 하신 건 없어요.
저희가 리뷰 관리, 키워드 세팅, 
사진 구성, 저장 수 늘리는 작업만 했고요.

우리 가게도 이렇게 될 수 있는지
딱 한 번만 확인해보세요.
15분이면 충분해요 🙏

[상담 예약 링크 or 연락처]
```

### 3차 메시지 (제안 + 클로징)

```
사장님 안녕하세요, [이름]이에요 😊

오늘은 솔직하게 제안 말씀드리려고요.

저희 [서비스명] 서비스
이번 달 신규 사장님 한해서
딱 이렇게 드리고 있어요.

✅ 포함 내용:
- 플레이스 키워드 세팅 (지역 + 업종 + 세부 키워드)
- 리뷰 관리 전략 + 응답 템플릿 제공
- 플레이스 사진 구성 + 메뉴/서비스 정보 최적화
- 저장 수 증가 작업
- 블로그 포스팅 [N]건 (상위노출 작업 포함)
- 월 1회 노출 순위 리포트 제공

💰 월 [가격]원
(일반 광고 집행 시 같은 효과 내려면 월 [비교 가격]원 이상 필요)

처음 한 달은 결과 보시고 결정하셔도 돼요.
부담 없이 한 번 시작해보시고,
효과 없으면 바로 말씀해주세요.

지금 신청하시면 
첫 달 플레이스 사진 촬영 가이드 + 리뷰 요청 문자 템플릿
추가로 드려요 (단독 판매가 [금액]원).

이번 달 [N]자리만 남았어요.
편하게 연락 주시면 바로 안내해드릴게요 😊

[연락처 or 링크]
```

### 4차 메시지 (마감 압박)

```
사장님, 마지막으로 한 번만 더 말씀드릴게요.

이번 달 신규 자리 [N]자리 중
지금 [N-1]자리 마감됐어요.

오늘 결정 못 하시면
다음 달로 넘어가는데,
그 사이에도 지금 상위에 있는 가게들은
계속 리뷰 쌓고, 저장 수 늘리고 있거든요.

격차는 매달 벌어지고,
나중에 따라잡는 게 점점 더 힘들어져요.

한 달만 먼저 시작해보세요.
결과 안 나오면 그냥 끊으시면 돼요.
근데 나오면, 이걸 왜 이제 했나 싶으실 거예요.

오늘까지만 첫 달 [할인금액]원 할인 + 보너스 자료 드려요.

→ [카카오톡 or 링크]
```

### 재발송 메시지 (3일 후)

```
사장님, 저 [이름]이에요.
며칠 전에 연락드렸었는데 기억하실지 모르겠네요 😊

바쁘실 것 같아서 부담은 아닌데,
한 가지만 여쭤봐도 될까요?

지금 네이버에서 
[업종] 검색했을 때
우리 가게가 상위에 안 뜨는 게
"당장 급한 문제"는 아니라고 느끼시나요?

만약 그렇다면 전혀 괜찮아요.
지금 시기가 아닐 수도 있으니까요.

근데 솔직히, 한 달 기다리는 동안
옆 가게는 한 달치 손님을 더 받고 있어요.

부담 없이 15분만 통화해보시겠어요?
도움이 안 되다 싶으면 그냥 끊으시면 돼요 😊

[연락처]
```

## 제안서/PPT 슬라이드 내용

### 슬라이드 1: 표지

```
제목: 사장님 가게, 지금 네이버에서 보이고 있나요?

부제: 광고비 0원으로 검색 손님 자동 유입하는
      네이버 플레이스 상위노출 솔루션

임팩트 한 줄:
"잘하는 가게가 이기는 게 아니라, 검색에 뜨는 가게가 이깁니다."

[회사명 / 담당자 이름 / 연락처]
```

---

### 슬라이드 2: 문제 인식

```
제목: 사장님, 이 중에 해당되시는 거 있으세요?

□ 네이버에 우리 가게 등록은 돼 있는데,
  검색하면 한참 아래에 있어요

□ 블로그 광고, 인스타 광고 해봤는데
  돈만 나가고 효과를 모르겠어요

□ 옆 가게는 손님이 넘치는데
  우리는 왜 이렇게 조용한지 모르겠어요

□ 리뷰가 별로 없어서 손님들이 그냥 지나치는 것 같아요

□ 플레이스 관리를 어떻게 해야 하는지
  아무도 제대로 알려준 사람이 없어요

---

📊 실제 데이터:
네이버 플레이스 검색 결과 1~3위 가게가
전체 클릭의 68%를 독점합니다.

지금 3페이지에 있다면?
→ 손님 100명 중 1명도 사장님 가게를 보지 않아요.
```

---

### 슬라이드 3: 현황 진단

```
제목: 지금 우리 가게 플레이스 상태는?

[ 사장님 가게 현황 진단표 ]

1. 네이버 플레이스 등록 여부
   → 등록됨 / 미등록

2. 현재 검색 키워드
   → "[지역] [업종]" 검색 시 현재 순위: [  ]페이지 [  ]번째

3. 누적 리뷰 수: [  ]개
   평균 별점: [  ]점

4. 플레이스 사진 수: [  ]장
   (권장: 최소 20장 이상, 최신 6개월 이내 업로드)

5. 월 플레이스 클릭 수: [  ]회
   (네이버 플레이스 관리자 페이지 통계 기준)

6. 블로그 리뷰 연동 수: [  ]건

7. 플레이스 저장 수: [  ]명
