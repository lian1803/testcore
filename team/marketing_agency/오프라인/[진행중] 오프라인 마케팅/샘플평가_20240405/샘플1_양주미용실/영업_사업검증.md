

# 검토 불가 — 입력 누락

승현의 영업 전략과 예진의 스크립트가 **비어 있습니다.**

검토할 콘텐츠가 없으므로 판정을 내릴 수 없습니다.

---

## 그러나 대기하지 않겠습니다.

주어진 정보(양주 미용실, D등급, 순위 13위, 월손실 557명)만으로 **현장 전문가 관점에서 사전 검증 프레임**을 먼저 깔아두겠습니다. 전략·스크립트가 올라오면 이 프레임 위에서 즉시 검증합니다.

---

## 🔴 지금 당장 고쳐야 할 것 (전략/스크립트 작성 전 필수 확인)

### 1. "월손실 557명" 수치의 신뢰성 확보
- **현장 리스크:** 미용실 사장님은 숫자에 민감하지 않거나, "어디서 그런 숫자가?"라고 의심합니다. 출처 불명의 큰 숫자를 던지면 **사기 DM으로 분류**됩니다.
- **현재 조건에서 사용 가능한가:** 조건부 YES
- **보완 방법:**
  - 557명의 산출 근거를 **한 문장으로** 설명 가능해야 함
  - 예: "양주 '미용실' 키워드 월 검색량 ○○건 × 1~3위 클릭률 약 ○○% = 상위 매장 월 유입 ○○명. 13위면 클릭률 ○○% → 차이 약 557명"
  - 이 산출식이 **없으면** 557명은 사용 금지. 사장님이 "그거 어떻게 나온 숫자예요?"라고 물었을 때 3초 안에 답 못 하면 신뢰 즉사
  - **557명 전부가 "손실"인지도 검증 필요** — 미용실은 예약 기반, 검색 유입 ≠ 실제 방문. "잠재 노출 기회 손실"이라면 표현 자체를 바꿔야 함

### 2. D등급 → 이 사장님은 플레이스 관리를 안 하고 있다
- **현장 리스크:** D등급이면 리뷰 수, 사진, 정보 입력 등이 전반적으로 부실. 이건 두 가지 의미:
  - (a) 디지털 마케팅에 관심 자체가 없을 수 있음 → DM 무시 확률 80%+
  - (b) 관심은 있지만 할 줄 몰라서 방치 → 진짜 타겟
- **현재 조건에서 사용 가능한가:** YES
- **보완 방법:**
  - DM 보내기 전에 **(a)인지 (b)인지 10초 진단** 필요:
    - 최근 3개월 내 리뷰 응답이 1개라도 있으면 → (b) 가능성 높음 → GO
    - 리뷰 응답 0, 사진 업데이트 0, 영업시간도 미입력 → (a) 가능성 → 우선순위 하락
  - 전략에 이 **사전 필터**가 없으면 헛 DM 비율이 70%를 넘김

### 3. "양주 미용실" 시장 특성 미반영 시 스크립트 실패
- **현장 리스크:** 양주는 인구 약 23만, 미용실 밀집도가 강남/홍대와 다름. "경쟁이 치열하다"는 프레이밍이 안 먹힐 수 있음. 사장님 체감은 "우리 동네 단골 위주라 괜찮아"일 수 있음
- **현재 조건에서 사용 가능한가:** 조건부 YES
- **보완 방법:**
  - 경쟁 프레이밍 대신 **"놓치고 있는 신규 고객" 프레이밍**
  - 양주 미용실 키워드 검색량이 실제로 높다면 → "이 동네에서 매달 ○○명이 네이버로 미용실 찾는데, 13위면 그 사람들 눈에 안 보여요"
  - 검색량이 낮다면 → 557명 자체가 과장 → 숫자를 줄이더라도 정직한 게 나음

## 🟠 중요하지만 시작은 할 수 있는 것

### 4. 레퍼런스 없는 신설팀의 신뢰 획득 전략
- **현재 조건에서 사용 가능한가:** YES
- **구체적 방법 (지금 즉시 가능한 것들):**

| 대체 신뢰 요소 | 실행 방법 | 소요 시간 |
|---|---|---|
| **진단 리포트 무료 제공** | 이 미용실 플레이스를 캡처 → 순위, 리뷰 수, 사진 수, 정보 입력률을 한 장에 정리 → "사장님 매장 현황입니다" 로 DM | 매장당 15분 |
| **경쟁사 비교표** | 양주 미용실 1~5위 매장의 리뷰 수, 사진 수, 키워드 등을 표로 → "1위 매장은 이렇고 사장님 매장은 이렇습니다" | 매장당 20분 |
| **첫 달 무료/보장 조건** | "첫 달은 무료로 진행하고, 순위 변화 없으면 비용 0원" — 신설팀이 실적 쌓는 유일한 루트 | 계약 구조에 반영 |
| **사장님 매장 구체 언급** | "○○헤어 사장님" "양주시 ○○동" 등 개인화 — 템플릿 DM이 아니라는 증거 | 매장당 5분 |

- **리스크:** 무료 제공 시 "무료니까 대충 해도 되겠지"라는 인식 → 무료 기간에도 **주간 리포트** 필수 (캡처 1장이라도)

### 5. 가격 구조 설계 — 스크립트에 녹이기 전에 확정 필수
- **현재 조건에서 사용 가능한가:** 조건부 YES (가격이 미확정이면 NO)
- **현장 팁:**
  - 양주 미용실 월매출 평균 500~1,500만 원 추정
  - 마케팅비 지출 의향: 월 20~50만 원이 심리적 한계선
  - **절대 6개월 합계 금액을 먼저 말하지 말 것** → "월 ○○만 원"만 언급
  - 비교 앵커: "네이버 광고 직접 하시면 월 30~50만 원 + 시간, 저희는 월 ○○만 원에 전부 관리"
- **보완:** 가격이 아직 미확정이면, 스크립트 작성 자체가 불가. 먼저 확정하고 올 것

### 6. 미용실 사장님 DM 수신 환경
- **현장 리스크:**
  - 미용실 사장님은 **시술 중에는 폰 못 봄** → DM 발송 시간 중요
  - 인스타 DM은 요청 메시지함으로 빠질 수 있음 → 열람률 20% 이하
  - 네이버 톡톡/카카오 채널은 차단률 높음
- **현재 조건에서 사용 가능한가:** YES
- **보완:**
  - **최적 DM 발송 시간:** 월~목 오전 10:00~11:00 (오픈 후 첫 시술 전), 또는 오후 8:30~9:30 (마감 후)
  - **DM 채널 우선순위:** ① 인스타 DM (계정이 있으면) ② 네이버 플레이스 톡톡 ③ 전화 (DM 무응답 3일 후)
  - 전화는 마지막 수단이지만 **양주 같은 소도시에서는 전화가 DM보다 성사율 높을 수 있음** → 테스트 필수

## 🟡 알고 가면 좋은 것

### 7. 미용실 특유의 이탈 패턴
- 미용실은 **비수기(1~2월, 6~7월)에 매출이 빠지면** "마케팅 효과 없네" → 해지
- 이건 마케팅 문제가 아니라 업종 특성인데, 사장님은 구분 못 함
- **보완:** 온보딩 시 "비수기에도 순위는 유지/상승했다"는 걸 보여줄 수 있는 **순위 추적 기록**을 시작일부터 남길 것. 엑셀 한 줄이면 됨

### 8. D등급 → S등급까지 걸리는 시간 현실
- 플레이스 등급은 리뷰 수, 사진, 예약, 정보 완성도 등 복합 요소
- D → B까지는 2~4주면 가능 (정보 입력 + 사진 보강 + 리뷰 응답)
- B → S는 리뷰 누적이 필요해서 2~6개월
- **스크립트에서 "빠른 효과"를 약속하면 안 됨** → "첫 2주 안에 등급 변화 확인 가능"까지만

### 9. 1인/소규모 팀의 수용 한계
- 플레이스 관리는 **콘텐츠 생산이 핵심** (사진 촬영, 리뷰 관리, 소식 발행 등)
- 1인이 감당 가능한 클라이언트: **동시 3~5개가 한계**
- 5개 넘으면 퀄리티 하락 → 이탈 → 악순환
- **보완:** 첫 달은 2개만 받고, 프로세스 안정화 후 확장

## 선결 조건 체크리스트 (현재 조건 기준)

| 항목 | 상태 | 대체/보완 전략 |
|---|---|---|
| ☐ 사회적 증거 (실적/사례) | **없음** | 진단 리포트 무료 제공 + 첫 고객 성과를 2주 단위 캡처로 축적. 첫 1건이 나오는 순간 그것이 레퍼런스 |
| ☐ 계약서 템플릿 | **확인 필요** | 최소: 서비스 범위, 기간, 월 금액, 해지 조건, 환불 조건. 프리랜서 계약서 템플릿 기반으로 수정 가능 (30분 작업) |
| ☐ 계약 기간 구조 | **미확정 추정** | 추천: 첫 달 무료 + 이후 월 단위 자동연장 (6개월 약정은 신설팀에겐 독). 사장님도 부담 적고, 성과로 잡아두는 구조 |
| ☐ 결제 수단 | **확인 필요** | 최소: 계좌이체 (세금계산서 발행 가능하면 가산점). 카드결제는 나중. 양주 소상공인은 계좌이체 선호 |
| ☐ 환불/보장 정책 | **미확정 추정** | 추천: "순위 변화 없으면 해당 월 비용 0원" — 신설팀이 쓸 수 있는 유일한 무기. 리스크는 팀이 흡수 |
| ☐ 온보딩 프로세스 | **없음 추정** | 최소 필요: ① 플레이스 관리자 권한 공유 받기 ② 현재 상태 캡처 (기준점) ③ 첫 주 작업 범위 공유 ④ 주간 리포트 형식 합의 |
| ☐ 최대 수용 클라이언트 수 | **미계획 추정** | 신설팀 첫 달: 2개. 안정화 후: 5개. 이 이상은 팀원 추가 전까지 금지 |
