# 🎯 리안 컴퍼니 소상공인 콜드 DM 교과서 — 검증자 피드백 반영 최종본

## 📋 사전 체크리스트 (발송 전 필수 5분)

DM 발송 전에 반드시 이 4가지를 확인하고 변수를 채워야 한다. 이 작업 없이 보낸 DM은 "일괄 스팸"으로 처리된다.

```
✅ 1. 업체명 정확히 (네이버 플레이스 등록명)
✅ 2. 경쟁사 실명 2~3곳 + 각각 현재 리뷰 수 (직접 검색해서 확인)
✅ 3. 타겟 업체 현재 플레이스 등록 상태 (사진 수, 리뷰 수, 소식 개수)
✅ 4. 사장님 추정 성별 (인스타/블로그 있으면 먼저 확인)

예시:
- 업체명: "강남역 뷔티큐 미용실"
- 경쟁사: "강남역 헤어몬스터" (리뷰 237개) / "뷰라벨" (리뷰 189개)
- 현재 상태: 플레이스 사진 2장 / 리뷰 18개 / 소식 0개
- 사장님 성별 추정: 여성 (블로그 글톤으로 판단)
```

> ⚠️ 미용실 원장 대상 추가 절차:
> 인스타 팔로우 → 게시물 좋아요 1~2개 → 3일 후 DM 발송
> "갑자기 모르는 사람"이 아닌 "팔로워가 연락한 것"으로 인식 전환

# 🚀 1차 DM — Hook (위기감 + 패턴 인터럽트)

**목표**: 답장 유도 (상담 신청 아님 — 단순 "응답" 확률 70% 이상)
**심리 기제**: 패턴 인터럽트 + 손실 프레임 + "나를 조사한 사람" 인식
**성공 지표**: 24시간 이내 답장 30% 이상
**금지 사항**: 회사명 첫줄 노출, 출처 없는 수치, 가짜 사회적 증거, 단정형 알고리즘 설명
**핵심**: 첫 줄이 모든 것을 결정한다. 사장님의 구체적인 현실을 짚어야 "나 얘기네"라고 느낀다.

---

## 버전 A — 직접 비교 + 확인 유도 프레임 (Pain 실체화)

**전략**: 경쟁사 리뷰 수를 직접 비교해서 타겟이 스스로 검색해 확인하게 만든다. 수치는 타겟이 직접 눈으로 확인할 수 있는 것만 사용.

> ✅ 수정 근거: "47개", "23%" 등 출처 없는 수치 전부 제거. "알려 드립니다" 정보 제공 구조를 질문으로 교체. 타겟이 직접 검색해서 확인하게 만들어 위기감이 자연 발생하도록 설계.

```
안녕하세요 원장님.

[경쟁사 실명] 리뷰 [경쟁사 수]개,
[업체명] 리뷰 [타겟 수]개 —

지금 "[지역명] [업종]" 검색하면
이 차이가 그대로 순위에 반영되는 거 혹시 아세요?

요즘 검색으로 들어오는 신규 손님이
예전보다 줄었다는 느낌 받으신 적 있으세요?

궁금한 게 있어서 연락드렸어요.
→ 카카오톡 오픈채팅: [링크]
```

**→ 왜 이 멘트인가**:
- ✅ 경쟁사 리뷰 수는 타겟이 직접 검색해서 확인 가능한 실수치
- ✅ "순위에 반영"이라는 사실 → 직접 검색 행동 유도
- ✅ "신규 손님 줄었다는 느낌" → 정보 제공 아닌 공감 질문으로 교체
- ✅ CTA 포함 (오픈채팅 링크)

---

## 버전 B — 플레이스 관리 상태 직접 지적 (현재 상황 명확화)

**전략**: "관리 안 된 상태"를 객관적으로 바라보게 만들기. 방어기제 최소화.

```
안녕하세요 원장님.

혹시 네이버 플레이스 [업체명]을
요즘 들어 자주 확인하세요?

요즘 자영업자분들 보면:
- 사진은 등록해두고 안 바꿈
- 영업시간·휴무 정보는 예전 그대로
- 리뷰는 삭제·답글 관리 안 함

이렇게 3개월 지나면 검색 노출도 떨어지는 거
직접 검색해서 느끼시는 분들이 꽤 많더라고요.

혹시 지금 플레이스 들어가보신 지 좀 된 건가요?

→ 카카오톡 오픈채팅: [링크]
```

**→ 왜 이 멘트인가**:
- ✅ 구체적 관리 흔적 열거 → "아, 맞다" 공감
- ✅ "직접 검색해서 느끼시는 분들이 많다" → 검증 가능한 표현으로 교체
- ✅ 부드러운 질문 종료 → 강압적이지 않으면서도 확인하고 싶은 심리 유도
- ✅ CTA 포함

---

## 버전 C — 리뷰 격차 직접 비교 (확인 유도)

**전략**: 경쟁사와의 명시적 격차를 보여주되, 타겟이 직접 검색해서 확인하게 만든다.

> ✅ 수정 근거: "실제로 저희가 본 데이터로는" 제거 (신설팀 자체 데이터 없음). 타겟이 직접 검색하면 확인 가능한 공개 정보만 사용.

```
안녕하세요 원장님.

네이버 검색으로 보니까 [경쟁사 A] 리뷰 [수]개,
[경쟁사 B] 리뷰 [수]개인데, [업체명]은 [수]개네요.

같은 동네, 같은 업종인데 리뷰 차이가 이만큼 나면
검색 결과에서 어떻게 보이는지
지금 "[지역명] [업종]" 직접 한번 검색해 보시겠어요?

혹시 요즘 핸드폰으로 직접 검색해 보신 적 있으신가요?

→ 카카오톡 오픈채팅: [링크]
```

**→ 왜 이 멘트인가**:
- ✅ 경쟁사 리뷰 수는 누구나 검색하면 확인되는 공개 수치만 사용
- ✅ "직접 검색해 보세요" → 수치 없이도 위기감이 실체화
- ✅ "저희 데이터" 표현 완전 제거
- ✅ CTA 포함

---

## 버전 D — 신규 가게 노출 패턴 공유 (관찰형 Challenger)

**전략**: "네이버 알고리즘 구조"를 단정형이 아닌 관찰형으로 제시. 타겟이 스스로 "맞아"라고 반응하게 만든다.

> ✅ 수정 근거: "네이버가 신규 가게에 초기 노출 가중치를 주기 때문이에요" → 공식 발표된 정책 아님. 단정형을 관찰형으로 교체. "어디서 확인해요?" 질문 차단.

```
안녕하세요 원장님.

요즘 검색해보면 새로 오픈한 가게들이
상위에 자주 보이는 패턴이 있거든요.

혹시 이런 거 눈에 띄신 적 있으세요?

기존에 자리 잡은 가게라도
플레이스 업데이트가 한동안 없으면
새 가게들 사이에서 묻히는 경우가 생기더라고요.

[업체명] 플레이스 보니까
최근에 업데이트된 게 많지 않은 것 같아서요.

혹시 이런 부분 신경 쓰고 계신 편인가요?

→ 카카오톡 오픈채팅: [링크]
```

**→ 왜 이 멘트인가**:
- ✅ "가중치를 주기 때문" 단정형 → "자주 보이는 패턴" 관찰형으로 교체
- ✅ 타겟이 "맞아, 나도 봤어" 반응 유도 → 신뢰도 상승
- ✅ "어디서 확인해요?" 반문 차단 구조
- ✅ CTA 포함

---

## 버전 E — 고객 선택 심리 (검색에서 결정까지)

**전략**: "고객이 어떻게 선택하는가"라는 행동 패턴을 설명. 사장님이 "나의 문제가 고객 문제구나" 깨닫게 만들기.

```
안녕하세요 원장님.

혹시 최근에 네이버 플레이스 조회 수가
예전보다 줄었다고 느껴지나요?

요즘 고객들이 가게 선택하는 흐름을 보면:

1. "[지역명] [업종]" 검색
2. 상위 가게 리뷰·사진 빠르게 비교
3. 신뢰 안 가면 → 바로 다음 가게로 넘어감

이 흐름에서 플레이스 관리가 안 되면:
- 사진이 낡거나 적음
- 리뷰 답글이 없음
- 최신 정보가 없음

이 세 가지가 겹치면 고객 눈에는
"관리 안 하는 가게"로 보이거든요.

혹시 이런 경험 있으신가요?

→ 카카오톡 오픈채팅: [링크]
```

**→ 왜 이 멘트인가**:
- ✅ 고객 관점 제시 → 판매자 아닌 조언자 포지셔닝
- ✅ "3초" 같은 미검증 수치 제거 → "빠르게 비교"로 대체
- ✅ 공감 질문으로 종료
- ✅ CTA 포함

## ⚡ 1차 DM 답장 유형별 즉시 대응 스크립트

> ✅ 추가 근거: 검증자 지적 — "답장 왔을 때 즉흥 대응하면 영업자처럼 보인다." 발송 전 이 스크립트 숙지 필수.

### [유형 1] "네 요즘 좀 그런 것 같아요"

```
혹시 언제부터 그런 느낌이셨어요?

플레이스 지금 어떤 상태인지 잠깐 봐드릴게요.
[업체명] 기준으로 지금 순위랑 리뷰 현황
정리해서 공유해드려도 괜찮을까요?
```

### [유형 2] "누구세요? 뭐 파는 거예요?"

```
판매 목적은 아니에요.

[업체명] 검색하다가 궁금한 게 생겨서 연락드렸어요.
지금 "[지역명] [업종]" 검색하면 몇 위로 나오는지
혹시 직접 확인해보신 적 있으세요?
```

### [유형 3] "괜찮아요 / 알아서 해요"

```
그러시군요.

혹시 직접 관리하고 계신 거면
지금 어떤 방식으로 하고 계신지 여쭤봐도 될까요?

저도 참고가 될 것 같아서요.
```

> 유형 3은 이탈 방지 + 현황 파악(SPIN S단계) 동시 진입. "알아서 해요" 반응을 대화 지속 기회로 전환.

## 1차 DM 발송 가이드

| 항목 | 실행 방법 |
|------|---------|
| **발송 시간** | 평일 화~목, 오후 2~3시 (점심 후 낮시간대) |
| **발송 채널** | 네이버 톡톡 또는 카카오 오픈채팅 |
| **메시지 길이** | 최대 250자 (스크롤 없이 한 번에 읽혀야 함) |
| **연속 발송 금지** | 같은 지역/업종 3개 가게씩 끊어서 발송 |
| **응답 없을 경우** | 48시간 대기 후 2차 메시지 발송 |
| **미용실 전용** | 인스타 팔로우 → 좋아요 1~2개 → 3일 후 DM |

# 📊 2차 DM — Story (신뢰 형성 + 진단 결과 공유)

**목표**: 1차 답장 또는 비응답자 대상. 진단 데이터 공유로 "이 사람이 정말 아는 사람"이라 확인시키기.
**심리 기제**: 권위(공개 데이터 기반 분석) + 무료 정보의 호혜성
**타이밍**: D+1 또는 D+2 (1차 발송 24~48시간 후)
**SPIN 단계**: P (문제 확인) — 1차의 S(상황) 이후, 문제를 타겟이 직접 말하게 유도
**핵심**: 구체적 수치로 현재 상황을 객관화하기. "느낌"에서 "데이터"로 전환.

---

## 버전 A — 진단 보고서 미리보기 (권위 + 호혜)

**전략**: "당신을 분석했고, 결과가 있다"는 사실만 전달. 수치는 직접 확인 가능한 공개 데이터만 사용.

> ✅ 수정 근거: "월 평균 X명 손실" 같은 추정치는 근거 없음. 직접 검색으로 확인되는 수치만 사용. 사회적 증거는 네이버 공식 자료로 대체.

```
안녕하세요 원장님,

[업체명] 네이버 플레이스 현황 정리해봤어요.

🔴 현재 검색 순위: "[지역명] [업종]" 기준 [X]페이지 [Y]위
   (직접 검색해서 확인하실 수 있어요)

🔴 리뷰 현황: [경쟁사 A] [X]개 / [경쟁사 B] [Y]개 / [업체명] [Z]개
   (네이버 플레이스에서 지금 바로 비교 가능)

🟡 소식 업로드: 최근 3개월 0개
   (상위 노출 가게들은 꾸준히 올리는 패턴)

🟡 리뷰 답글: 미확인 or 없음
   (네이버는 사업자 활동도 노출 지표로 반영)

지금 상태에서 어떤 부분부터 보완하면
달라질 수 있는지 한 번 같이 봐드릴게요.

→ 카카오톡 오픈채팅: [링크]
혹시 한 번 얘기 나눠보실래요?
```

**→ 왜 이 멘트인가**:
- ✅ 모든 수치는 타겟이 직접 검색해서 확인 가능한 공개 데이터
- ✅ "직접 검색해서 확인하실 수 있어요" 명시 → 수치 신뢰도 보강
- ✅ 자체 데이터 주장 제거
- ✅ CTA 포함

## 버전 B — 관찰 기반 스토리 (Challenger 인사이트)

**전략**: 비슷한 상황에서 관리를 시작한 가게의 변화 패턴을 전달. 단, 가짜 사례 금지 — 실제 작업 전이라면 패턴 설명으로 대체.

> ✅ 수정 근거: 신설팀은 실제 사례가 없다. "저희 고객 사례"라는 거짓 사회적 증거 사용 불가. 공개 관찰 패턴으로 대체.

```
안녕하세요 원장님,

네이버 플레이스 상위 노출 가게들 보면
공통적인 패턴이 있거든요.

검색 1~3위에 있는 가게들 거의 대부분:
- 최근 3개월 소식 10개 이상 꾸준히 업로드
- 리뷰 답글 80% 이상 달려 있음
- 사진 20장 이상, 최근 사진 포함

반면 10위 아래로 밀려 있는 가게들은
이 세 가지 중 하나라도 빠져 있는 경우가 많아요.

[업체명] 지금 상태랑 비교해보면
어떤 부분이 다른지 느껴지시나요?
```
