# 리안 컴퍼니 — 콜드 DM 교과서 완성본
## 네이버 플레이스 진단 기반 소상공인 영업 멘트 라이브러리

---

# 핵심 인사이트 (자료 + 리서치 분석)

## 인사이트 1: 타겟은 "몰라서 안 하는" 게 아니라 "해봤는데 당했다" 상태다
(출처: 소상공인 거절 이유 Top 5 리서치)

마케팅 대행사에 대한 한국 소상공인의 기본 감정은 **불신**이다. "광고비만 날렸다", "순위 올렸다고 했는데 3개월 후 원상복귀", "연락만 씩씩하고 결과는 없음" — 이게 디폴트 심리 상태다. 즉, 우리의 첫 번째 적은 경쟁사가 아니라 **타겟의 과거 트라우마**다.

**전략적 함의**: 첫 DM에서 "우리 서비스 좋아요"가 아니라 "당신이 당한 그 패턴, 우리가 알고 있어요"로 시작해야 공감을 산다.

---

## 인사이트 2: 손실 프레임이 이익 프레임보다 2~3배 강하다
(출처: 손실회피 편향 리서치, Kahneman 기반)

"매출 올려드립니다"보다 "지금 이 상태면 매달 XX명이 경쟁사로 넘어가고 있습니다"가 행동을 유발한다. 소상공인은 이익보다 손실에 2.5배 민감하게 반응한다. 단, **수치는 타겟 업종 실제 매출 규모에 맞게** — 동네 미용실 월매출 1,000~3,000만원 기준으로 설계해야 신뢰가 살아있다.

**전략적 함의**: 진단 보고서의 모든 수치는 "지금 잃고 있는 것" 언어로 포장한다.

---

## 인사이트 3: 네이버 플레이스 순위는 3개월 사이클로 무너진다
(출처: 네이버 플레이스 Pain Point 리서치)

한 번 세팅해두면 된다고 생각하는 사장님이 대부분이다. 실제로는 신규 업체 가중치, 정보 완성도, 리뷰 최신성에 따라 3개월 단위로 순위가 재편된다. 이 구조를 사장님이 모른다는 것이 우리의 진입점이다.

**전략적 함의**: 진단 보고서에 "현재 순위 + 3개월 전 추정 순위"를 대비해서 보여주면 위기감이 실제화된다.

---

## 인사이트 4: 디지털 역량 자평 37.7점 — 사장님은 스스로 "나는 모른다"는 걸 안다
(출처: 경기도 소상공인 디지털 인식 실태 2024)

디지털 마케팅에 대한 자신감이 없는 사장님일수록 "누군가 대신해줬으면" 심리가 강하다. 단, 이걸 건드리면 방어기제("직접 할게요")가 올라온다. 대신 "복잡한 거 다 저희가 하고, 사장님은 결과만 보시면 됩니다"로 접근해야 한다.

**전략적 함의**: 온보딩 설계에서 고객이 해야 할 것을 최소화하고, DM에서도 "쉽다"가 아니라 "사장님은 아무것도 안 하셔도 됩니다"로 표현한다.

---

## 인사이트 5: SPIN Selling의 핵심은 "문제를 말하게 하는 것"이 아니라 "문제의 결과를 느끼게 하는 것"이다
(출처: SPIN Selling — Neil Rackham, Challenger Sale — Dixon/Adamson)

Situation → Problem → Implication → Need-Payoff 구조에서 소상공인 콜드 DM에 가장 중요한 것은 **Implication(파급 효과)** 단계다. "플레이스 관리가 안 되고 있네요"가 아니라 "플레이스 3페이지에 있으면 검색 고객의 93%가 첫 페이지에서 결정하고 끝내버립니다" — 이게 Implication이다.

Challenger Sale 관점에서는 사장님이 "몰랐던 인사이트"를 제공하는 것이 신뢰의 핵심이다. 우리가 먼저 진단 데이터를 가져가는 구조 자체가 Challenger 포지셔닝이다.

**전략적 함의**: DM 멘트의 뼈대는 항상 S → P → I → N-P 순서를 따른다. 특히 I 단계(지금 이 상황이 매달 어떤 손실을 만들고 있는지)에 가장 많은 에너지를 쓴다.

# 영업 전략 설계

## 타겟 선별 기준

### 1순위 타겟 조건 (이 조건 3개 이상이면 즉시 접촉)

```
✅ 네이버 플레이스 등록은 되어 있으나 관리 흔적 없음
   (사진 3장 이하 / 영업시간 미입력 / 소식 0개)

✅ 경쟁 업체 대비 리뷰 수 하위 30% 이하
   (같은 동네, 같은 업종, 나보다 리뷰 3배 이상 많은 경쟁사 존재)

✅ 플레이스 노출 순위 3페이지 이하
   (키워드 "지역명+업종" 검색 시 직접 확인)

✅ 업력 1년 이상 (창업 초기는 마케팅 예산 없음)

✅ 오프라인 매장 보유 (배달 전용 X)
```

### 2순위 타겟 (1순위 소진 후)

```
✅ 플레이스 등록 자체가 안 된 업체
   (검색 시 사업자정보만 나오고 스마트플레이스 없음)

✅ 경쟁사 리뷰가 100개 이상인데 본인은 20개 이하
```

### 우선 업종 순서 (Pain 강도 + 의사결정 속도 기준)

```
1위: 미용실 (경쟁 치열, 리뷰 의존도 높음, 원장 혼자 결정)
2위: 음식점/카페 (플레이스 영향력 가장 큼, 단 감도 낮을 수 있음)
3위: 학원 (학부모 검색 의존, 결정 느리지만 계약단가 높음)
4위: 네일/피부/마사지 (SNS+플레이스 복합, 원장 1인 결정 구조)
```

## 접촉 전략

### 전체 구조: Hook → Story → Offer (콜드 DM 완결)

```
D+0: 진단 결과 → Hook (위기감)
D+1: 원인 설명 → Story (신뢰)
D+3: 해결 방향 제시 → Offer (의존성)
D+5: 사회적 증거 → 마감 없는 압박
D+7: 최종 1회 → 침묵하거나 클로징
```

---

### STEP 0: 발송 전 사전 리서치 (DM 발송 전 5분 작업)

**확인할 것 4가지:**

```
1. 업체명 (정확히)
2. 경쟁사 1~2개 실명 + 리뷰 수
3. 해당 업체 현재 리뷰 수 + 플레이스 등록 상태
4. 사장님 추정 성별 (가게 인스타/블로그 있으면 확인)
```

이 4가지 없이 DM 보내면 "일괄 발송 스팸"으로 인식된다. 4가지 넣으면 "나를 조사한 사람"이 된다.

## 1차~4차 DM 멘트 라이브러리

---

### ■ 1차 DM — Hook (위기감 유발)

**목표**: "이 사람이 우리 상황을 알고 있네" → 답장 유도
**심리 기제**: 패턴 인터럽트 + 손실 프레임
**금지**: 자기소개, 회사명, 서비스 설명

---

**[1차 템플릿 A — 미용실 버전]**

```
원장님, 잠깐만요.

오늘 네이버에서 [동네명] 미용실 검색해봤는데
[경쟁업체명]이 리뷰 [XX]개로 상단에 고정되어 있더라고요.

[원장님 업체명]은 지금 검색하면 몇 번째 나오는지
확인해보신 적 있으세요?

진단 결과 하나 보내드려도 될까요?
(1분이면 됩니다)
```

---

**[1차 템플릿 B — 음식점/카페 버전]**

```
사장님, 실례가 안 된다면 여쭤봐도 될까요.

[동네명] [업종] 검색하면 상단 3개 자리에
[경쟁업체A], [경쟁업체B]가 고정으로 올라와 있는데

[업체명]은 지금 검색 몇 페이지에 있는지
혹시 알고 계세요?

짧게 확인해드릴 수 있어요.
```

---

**[1차 템플릿 C — 리뷰 격차 강조 버전]**

```
사장님 안녕하세요.

다름이 아니라 오늘 [업체명] 네이버 플레이스 보다가
연락드리게 됐어요.

지금 [경쟁업체명]이 리뷰 [XX]개인데
[업체명]은 [XX]개더라고요.

이 차이가 매달 검색 유입에서
얼마나 차이나는지 아시면 조금 놀라실 수도 있어요.

진단 결과 공유드려도 괜찮을까요?
```

---

**[1차 템플릿 D — 플레이스 미등록 버전]**

```
사장님, 오늘 [동네명] [업종] 찾아보다가
[업체명] 검색해봤는데

네이버 플레이스에 등록이 안 되어 있더라고요.

[동네명]에서 [업종] 검색하는 분들이 하루에
몇 명인지 보여드려도 될까요?

숫자 보시면 아마 좀 당황하실 수 있어요.
```

---

**1차 DM 작성 원칙 요약:**

```
✅ 첫 줄: 자기소개 X, 상대방 이름/업체명으로 시작
✅ 두 번째 줄: 경쟁사 실명 + 실제 수치
✅ 세 번째 줄: 상대방의 현재 상태 (모르고 있는 것)
✅ 마지막 줄: 질문 or 허락 구하기 (강요 X)
✅ 길이: 6줄 이내
✅ "저는 ~입니다" 없음
✅ "마케팅", "광고", "대행" 단어 1차에서 절대 금지
```

---

### ■ 2차 DM — Story (진단 보고서 + 원인 설명)

**목표**: "왜 이렇게 됐는지" 이해시키기 → "이 사람들이 제대로 아네" 신뢰 구축
**트리거**: 1차에 답장이 오거나, 답장 없이 24시간 경과 시 발송
**심리 기제**: Challenger Sale의 "Reframe" — 사장님이 몰랐던 구조 설명

---

**[2차 템플릿 A — 답장이 온 경우 (허락 받은 후 보고서 발송)]**

```
감사합니다, [이름/원장님].

[업체명] 플레이스 진단 결과예요.

─────────────────
📍 현재 상태
• 검색 순위: [키워드] 기준 [N]페이지 [N]번째
• 플레이스 정보 완성도: 100점 중 [N]점 (사진/시간/소식 기준)
• 최근 리뷰 등록일: [N]일 전
• 경쟁사 [업체명] 리뷰: [N]개 / [업체명] 리뷰: [N]개
─────────────────

네이버 플레이스는 정보 완성도랑
리뷰 최신성으로 순위가 결정되는데

지금 이 상태로는 [키워드] 검색하는 분들이
[업체명]을 발견하기가 어려운 구조예요.

그리고 이 순위는 3개월 단위로 계속 재편되기 때문에
지금 관리가 안 되면 이후에 더 내려갈 가능성이 높아요.

한 가지만 여쭤봐도 될까요?
지금 신규 고객이 주로 어디서 유입되는 편이에요?
```

---

**[2차 템플릿 B — 무응답 24시간 후 발송]**

```
[업체명] 관련해서 짧게 추가로 보내드려도 될까요.

네이버에서 [동네명] [업종] 검색하면
상단 1~3위 업체가 월 검색 유입의 약 70%를 가져가요.

지금 [업체명]이 [N]페이지에 있다는 건
검색하는 분들 10명 중 7~8명은
[업체명]을 보지도 못하고 결정한다는 거예요.

잘못하신 게 아니에요.
네이버 플레이스 구조 자체가
계속 관리 안 하면 점점 밀리게 되어 있어요.

진단 자료 공유드려도 될까요?
```

---

**[2차 템플릿 C — 리뷰 관련 원인 집중 버전]**

```
[이름]님, 왜 [경쟁업체명]이랑 이렇게 차이가 나는지
구조로 설명해드릴게요.

네이버 플레이스 순위는
① 정보 완성도
② 리뷰 수 + 최신성
③ 사진 품질
이 세 가지가 핵심이에요.

[업체명]이 지금 [N]페이지인 이유는
실력이 없어서가 아니에요.
플레이스 관리 방식의 문제예요.

[경쟁업체명]이 리뷰 [XX]개인 건
그 사장님이 특별히 잘해서가 아니라
이 구조에 맞게 움직이고 있어서예요.

이 부분 어떻게 바꿀 수 있는지
한번 이야기해볼 수 있을까요?
```

---

**2차 DM 작성 원칙 요약:**

```
✅ 진단 보고서는 실제 수치를 넣어서 (추정도 OK, 단 출처 근거 있게)
✅ "당신 잘못이 아니에요" → 방어기제 해제
✅ 원인을 "방법의 문제"로 프레이밍 (노력 탓 X)
✅ 마지막은 반드시 질문으로 끝 (닫힌 질문 X, 열린 질문)
✅ 해결책 구체적 언급 금지 ("리뷰를 이렇게 하세요" 절대 X)
```

---

### ■ 3차 DM — Offer (의존성 + 미래 그리기)

**목표**: "여기 말고는 없다" 의존성 형성 → 상담/계약 유도
**트리거**: 2차 발송 후 48시간 이내 (대화가 이어진 경우 즉시)
**심리 기제**: Need-Payoff (SPIN) + 앵커링 + 손실 회피

---

**[3차 템플릿 A — 변화된 미래 제시]**

```
[이름]님, 구체적으로 말씀드릴게요.

비슷한 상황이었던 [업종] 사장님들이
플레이스 정비 후 [N]개월 시점에서

"검색해보니까 이제 첫 페이지에 나와요"
"전화가 예전보다 늘었어요"
"처음 오신 분들이 '검색하고 왔어요' 하더라고요"

이런 피드백을 주시더라고요.

[업체명]도 지금 구조만 바꾸면
충분히 가능한 흐름이에요.

저희가 하는 건 단순해요.
사장님이 신경 안 쓰셔도 되게
플레이스 전반을 관리해드리는 거예요.

한번 자세히 이야기해보실래요?
부담스러우면 진단 결과 먼저 더 풀어서 보내드릴게요.
```

---

**[3차 템플릿 B — 앵커링 가격 제시 포함]**

```
[이름]님, 서비스 구성 간단히 보내드릴게요.

─────────────────
플레이스 관리 서비스

[프리미엄] 월 89만원
→ 플레이스 전체 운영 + 리뷰 관리 + 콘텐츠 제작 + 경쟁사 모니터링

[스탠다드] 월 49만원
