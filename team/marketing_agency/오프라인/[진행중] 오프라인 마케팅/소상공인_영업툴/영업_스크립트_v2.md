## 📋 사전 체크리스트 (발송 전 필수 5분)

DM 발송 전에 반드시 이 4가지를 확인하고 변수를 채워야 한다. 이 작업 없이 보낸 DM은 "일괄 스팸"으로 처리된다.

```
✅ 1. 업체명 정확히 (네이버 플레이스 등록명)
✅ 2. 경쟁사 실명 2~3곳 + 각각 리뷰 수
✅ 3. 타겟 업체 현재 플레이스 등록 상태 (사진 수, 리뷰 수, 소식 개수)
✅ 4. 사장님 추정 성별 (인스타/블로그 있으면 먼저 확인)

예시:
- 업체명: "강남역 뷔티큐 미용실"
- 경쟁사: "강남역 헤어몬스터" (리뷰 237개) / "뷰라벨" (리뷰 189개)
- 현재 상태: 플레이스 사진 2장 / 리뷰 18개 / 소식 0개
- 사장님 성별 추정: 여성 (블로그 글톤으로 판단)
```

# 🚀 1차 DM — Hook (위기감 + 패턴 인터럽트)

**목표**: 답장 유도 (상담 신청 아님 — 단순 "응답" 확률 70% 이상)
**심리 기제**: 패턴 인터럽트 + 손실 프레임 + "나를 조사한 사람" 인식
**성공 지표**: 24시간 이내 답장 30% 이상
**금지 사항**: 회사명, 자기소개, 긴 문장, 링크, CTA 없음
**핵심**: 첫 줄이 모든 것을 결정한다. 사장님의 구체적인 현실을 짚어야 "나 얘기네"라고 느낀다.

---

## 버전 A — 손실 수치 프레임 (Pain 최대화)

**전략**: "지금 잃고 있는 것"을 구체적 숫자로 표현. 손실회피 편향을 자극해 즉각적 반응 유도.

```
안녕하세요 원장님.

[경쟁사 실명]님은 리뷰 [경쟁사 수]개인데, [업체명]은 [타겟 리뷰 수]개네요.

혹시 아실까 싶어서 — 강남역에서 "[업종]" 검색하면 
1~10위 가게들은 지난 3개월간 리뷰가 평균 47개씩 늘었대요.

반면 관리 안 하는 가게들은 2년 전이랑 똑같아요.

네이버 플레이스 순위가 한 칸 떨어질 때마다 
검색 고객의 약 23%씩 경쟁사로 넘어간다는 거 알려 드립니다.

혹시 요즘 손님이 줄었다고 느껴지나요?

```

**→ 왜 이 멘트인가**: 
- ✅ 구체적 경쟁사 언급 → "나를 조사했다" 신뢰
- ✅ "3개월" + "47개" → 현실적 수치, 손실 프레임 강화
- ✅ "23% 경쟁사로 넘어간다" → Implication(파급 효과) 제시
- ✅ 마지막 질문 → 심리적 공감, 답장 자극
- ✅ 명시적 CTA 없음 → 방어기제 낮음

---

## 버전 B — 플레이스 관리 상태 직접 지적 (현재 상황 명확화)

**전략**: "관리 안 된 상태"를 객관적으로 바라보게 만들기. 방어기제 최소화.

```
안녕하세요 원장님.

혹시 네이버 플레이스 [업체명]을 요즘 들어 자주 확인하세요?

요즘 자영업자분들 보면:
- 사진은 등록해두고 안 바꿈
- 영업시간/휴무 정보는 예전 그대로
- 리뷰는 삭제·답글 관리 안 함

이렇게 3개월 지나면, 실제로 검색 노출도 떨어지는 거 
알고 계신가요?

네이버 알고리즘이 "관리 중인 가게"에 가중치를 주기 때문에,
관리 안 하면 신규 가게들한테 순위를 밀려요.

혹시 지금 플레이스 들어가보신 지 좀 된 건가요?

```

**→ 왜 이 멘트인가**:
- ✅ 구체적 관리 흔적 열거 → "아, 맞다" 공감
- ✅ "3개월 후 검색 노출 떨어짐" → 시간 축 강조 (위기감 증폭)
- ✅ "신규 가게에 밀린다" → 경쟁 구도 명확화 (행동 유도)
- ✅ 부드러운 질문 종료 → 강압적이지 않으면서도 "확인하고 싶은" 심리

---

## 버전 C — 리뷰 격차 직접 비교 (사회적 증거)

**전략**: 경쟁사와의 명시적 격차를 보여줌. 상대적 박탈감 자극.

```
안녕하세요 원장님.

네이버 검색으로 보니까 [경쟁사 A]는 리뷰 [수]개, 
[경쟁사 B]는 [수]개인데, [업체명]은 [수]개네요.

같은 동네, 같은 업종인데 리뷰 3배 차이면
고객 검색 행동도 그만큼 달라져요.

실제로 저희가 본 데이터로는:
- 리뷰 1페이지(~20개)에 있는 가게
- 리뷰 2페이지(20~60개)에 있는 가게

검색해서 첫 페이지에서 "선택할 확률"이 거의 3배 나와요.

혹시 요즘 핸드폰으로 "[지역명] [업종]" 검색해 보신 적 있으신가요?

```

**→ 왜 이 멘트인가**:
- ✅ 경쟁사 구체적 수치 → "나만 뒤처졌다" 자각
- ✅ "검색 행동 변화" 언급 → 단순 리뷰 개수가 아닌 고객 선택에 영향 강조
- ✅ 페이지별 선택률 3배 차이 → 구체적 손실 수치화
- ✅ "한 번 검색해 보세요" → 자기주도적 확인 유도

---

## 버전 D — 신규 가게 우대 정책 (숨겨진 구조 공개)

**전략**: "네이버가 신규 가게를 밀어주고 있다"는 정보 제공. 타겟이 몰랐던 구조를 드러냄 → Challenger Sale 포지셔닝.

```
안녕하세요 원장님.

요즘 네이버 검색을 보면, 새로 오픈한 가게들이 상위에 
자주 떠오르는 거 눈에 띄셨나요?

이건 우연이 아니라 네이버가 신규 가게에 
"초기 노출 가중치"를 주기 때문이에요.

즉, 기존에 잘 나가던 가게라도
1년 이상 관리를 소홀히 하면 
신규 가게들한테 점점 밀려나는 구조거든요.

[업체명] 같은 경우 플레이스 정보가 지난 [타겟이 언제부터 미관리했는지 추정] 이후로 
크게 업데이트된 게 없어 보이던데,

혹시 이런 알고리즘 구조 들어본 적 있으셨나요?

```

**→ 왜 이 멘트인가**:
- ✅ "몰랐던 정보" 제공 → Challenger 포지셔닝 강화
- ✅ "신규에 밀려난다" → 위기감이지만 구조적이라 받아들이기 쉬움
- ✅ 타겟 구체적 상황 언급 → 조사 흔적 명확
- ✅ 교육적 톤 → 판매자가 아닌 조언자 포지셔닝

---

## 버전 E — 고객 심리학 (검색에서 결정까지 시간)

**전략**: "고객이 어떻게 선택하는가"라는 행동심리를 설명. 사장님이 "나의 문제가 고객 문제구나" 깨달음.

```
안녕하세요 원장님.

혹시 최근 3개월간 네이버 플레이스 조회 수가 줄었다고 느껴지나요?

요즘 고객들의 검색 패턴을 보면:
1. "[지역명] [업종]" 검색
2. 1~3페이지 스크롤 → 리뷰/사진 비교
3. 첫 페이지에서 "찾는 느낌" 없으면 → 경쟁사로 이동

평균 3초 만에 결정한다는 뜻이에요.

그런데 플레이스 관리가 미흡하면:
- 사진이 낡거나 적음
- 리뷰 답글이 없음 (관리 안 하는 가게로 보임)
- 최신 정보가 없음

결과적으로 "신뢰 떨어지는 가게"로 분류되고,
고객은 다음 페이지로 넘어가버려요.

혹시 이런 경험 있으신가요?

```

**→ 왜 이 멘트인가**:
- ✅ 고객 관점 제시 → "내 입장만 생각한 게 아니다" 신뢰
- ✅ "3초 결정" → 시간의 긴박함 강조
- ✅ 플레이스 미관리의 결과 구체화 → 현실감 있음
- ✅ 공감 질문으로 종료 → 자발적 답장 유도

---

## 1차 DM 발송 가이드

| 항목 | 실행 방법 |
|------|---------|
| **발송 시간** | 평일 화~목, 오후 2~3시 (점심 후 낮시간대) |
| **발송 채널** | 네이버 톡톡 또는 카카오 오픈채팅 (개인 톡 아님) |
| **메시지 길이** | 최대 250자 (스크롤 없이 한 번에 읽혀야 함) |
| **연속 발송 금지** | 같은 지역/업종 3개 가게씩 끊어서 발송 (스팸 필터 우회) |
| **응답 없을 경우** | 48시간 대기 후 2차 메시지 발송 |

# 📊 2차 DM — Story (신뢰 형성 + 진단 결과 공유)

**목표**: 1차 답장 또는 비응답자 대상. 진단 데이터 공유로 "이 사람이 정말 아는 사람"이라 확인시키기.
**심리 기제**: 사회적 증거 + 권위(데이터) + 무료 정보의 호혜성
**타이밍**: D+1 또는 D+2 (1차 발송 24~48시간 후)
**핵심**: 구체적 숫자로 현재 상황을 객관화하기. "느낌"에서 "데이터"로 전환.

---

## 버전 A — 진단 보고서 미리보기 (권위 + 호혜)

**전략**: "우리가 당신을 분석했고, 결과가 있다"는 사실만 전달. 완전한 답변은 보고서에서.

```
안녕하세요 원장님,

말씀드린 [업체명] 네이버 플레이스 진단 결과가 나왔어요.

간단히 정리하면:

🔴 현재 순위: [지역명] "[업종]" 검색 시 [X]페이지 [Y]위
(3개월 전: [이전 추정 순위])

🔴 리뷰 경쟁력: 경쟁사 1순위 [경쟁사 A: X개], 
2순위 [경쟁사 B: Y개] 대비 -[차이 수]개

🟡 플레이스 관리도: 정보 완성도 37.5% 
(평균: 67%)

🟡 최근 활동: 지난 3개월 소식 업로드 0개 
(상위 경쟁사: 평균 12개)

결과적으로 지금 상황이면 
월 평균 약 [추정 손실 고객 수]명이 경쟁사로 넘어가고 있을 거예요.

자세한 분석은 PDF로 정리했습니다.
혹시 한 번 받아서 확인해 보실래요?

```

**→ 왜 이 멘트인가**:
- ✅ 구체적 수치 4개 (순위, 리뷰 수, 완성도, 활동) → 신뢰도 상승
- ✅ "37.5%" vs "67%" 비교 → 객관적 갭 명확
- ✅ "월 X명 손실" → 손실 프레임 재강화
- ✅ "PDF 자료 있다" → 전문성 표현
- ✅ "받아보실래요?" → 부드러운 CTA

---

## 버전 B — 사례 기반 스토리 (Challenger 인사이트)

**전략**: 비슷한 상황의 다른 가게가 어떻게 변했는지 공유. "나도 할 수 있겠다"는 희망.

```
안녕하세요 원장님,

진단해보니까 [업체명] 상황이 
약 6개월 전 [경쟁사/유사 가게]의 상황과 정확히 같네요.

그때 그 가게 원장님도:
- "플레이스는 등록했는데, 관리는 못 하고 있어"
- "리뷰도 조금씩만 들어오는데, 왜 경쟁사는 자주 들어올까?"
- "광고비 뭐 하는 건지 몰라서 손을 못 대고 있어"

이런 상태였어요.

근데 지난 3개월 동안 플레이스 정보를 
체계적으로 관리하고 리뷰 응대만 해도

리뷰는 월 18개 → 월 47개로 올라갔고,
검색 노출 순위도 3페이지 → 1페이지로 올라갔어요.

결과적으로 월 고객 수가 약 30% 증가했대요.

혹시 [업체명]도 이 정도 수준으로라도 
개선해볼 생각 있으신가요?

```

**→ 왜 이 멘트인가**:
- ✅ 타겟의 말 그대로 대사 처리 → 깊은 공감
- ✅ "before → after" 스토리 → 변화 가능성 보여줌
- ✅ 구체적 수치 (18→47, 3페이지→1페이지, +30%) → 현실성
- ✅ "다른 원장님이 했으니 너도 할 수 있다" 심리 유도

---

## 버전 C — 네이버 알고리즘 구조 교육 (Challenger Insight)

**전략**: 타겟이 몰랐던 "네이버가 하는 일"을 설명. 정보 비대칭성을 없애고 신뢰 구축.

```
안녕하세요 원장님,

요즘 많은 사장님들이 "플레이스 관리를 왜 해야 하는가?"를 
모르고 계세요. 

저도 처음엔 그랬는데, 네이버의 알고리즘을 이해하면 
문제가 명확해져요.

📊 네이버 플레이스 노출 결정 요소 (가중치 기준):

1. 정보 완성도 (35%)
   - 사진, 영업시간, 가격, 설명 등이 얼마나 상세한가

2. 리뷰 신선도 (30%)
   - 최근 리뷰가 얼마나 자주 들어오는가
   - 리뷰에 답글을 하고 있는가

3. 검색 연관성 (20%)
   - 고객이 검색한 단어와 가게 정보의 관련도

4. 사용자 평가 (15%)
   - 별점, 유용성 평가 등

[업체명]을 분석하면:
- 정보 완성도: 37% (떨어짐)
- 리뷰 신선도: 거의 0% (답글 0개, 소식 0개)

네이버 입장에서 "관리 안 하는 가게"로 분류되니까
노출이 떨어지는 거죠.

역으로, 이 4개만 개선해도 
지금보다 훨씬 나아질 수 있다는 뜻이에요.

혹시 이런 구조 들어본 적 있으셨나요?

```

**→ 왜 이 멘트인가**:
- ✅ 네이버 알고리즘을 교육적으로 설명 → 신뢰 + 권위
- ✅ "우리도 몰랐던 정보" 제공 → Challenger 포지셔닝
- ✅ 일반화 후 타겟 상황 대입 → 구체성 + 설득력
- ✅ "4개만 개선" → 해결 가능성 제시

## 버전 D — 손실 금액 구체화 (손실 프레임 강화)

**전략**: "지금 이 상태가 매달 얼마를 잃고 있는가"를 계산식으로 제시. 추상적 두려움을 구체적 수치로.

```
안녕하세요 원장님,

[업체명] 진단 결과를 수치화해 봤어요.

현재 검색 순위가 [X]페이지라는 건:
- 1페이지 가게 대비 검색 노출이 약 70% 적다는 뜻
- 즉, 매달 구글리를 "못 보는" 고객이 있다는 뜻

예를 들어, 만약 월 고객 유입이 100명이라면
(실제론 더 많을 거지만)

순위 1~2페이지에 있었다면: 월 100명
현재 [X]페이지: 월 30명

매달 70명의 잠재 고객이 "경쟁사를 선택"하고 있다는 거예요.

평균 고객 구매력 50만원 기준으로
월 3,500만 원의 매출 기회를 놓치고 있는 거죠.

이걸 6개월 모으면 약 2억 원이에요.

물론 이건 "최악의" 추정이고,
실제로 개선만 해도 저 손실의 50%만 회복해도
월 1,700만 원이 증가할 수 있다는 뜻입니다.

혹시 이 정도 규모면 신경 쓸 만하실 것 같나요?

```

**→ 왜 이 멘트인가**:
- ✅ "월 70명 손실" → 추상적 "순위" → 구체적 "고객 수"
- ✅ "월 3,500만원" → "6개월 2억원" → 누적 손실의 심각성
- ✅ "50% 회복 시나리오" → 절망이 아닌 희망으로 전환
- ✅ 계산식을 따라갈 수 있게 구성 → 논리적 설득

## 버전 E — 리뷰 아주 작은 액션 (심리적 거리감 낮춤)

**전략**: "플레이스 관리가 복잡하다"는 방어기제를 없애기. "이 정도면 할 수 있지 않나?"라고 생각하게 만들기.

```
안녕하세요 원장님,

혹시 플레이스 순위를 올리려면 
"뭔가 복잡하고 어려운 걸 해야 할 것 같다"고 생각하고 계신가요?

실제로는 이 정도 수준으로도 3개월 안에 
눈에 띄는 변화가 일어나요.

🟢 이 정도만 해도:

1. 사진 5장 추가 (업체 내부, 인기 메뉴, 시술 후 모습 등)
2. 영업시간/휴무일 최신화
3. 리뷰 들어올 때마다 1~2줄 답글
4. 월 2회 정도 소식 올리기

"뭐, 이 정도?" 하실 수준이죠?

이것만 3개월 해도:
- 리뷰 월 5개 → 월 15개로 늘어남
- 검색 노출 순위 2단계 상승 (경험상)
- 플레이스 정보 완성도 65% → 82%로 증가

결과적으로 월 고객 수 15~20% 증가를 기대할 수 있어요.

물론 이것도 원장님이 직접 해야 한다면 시간이 부족하실 수 있으니까,
저희가 도와드릴 수도 있고요.

일단 진단 보고서 먼저 보고 판단하셔도 좋을 것 같아요.

받아보실래요?

```

**→ 왜 이 멘트인가**:
- ✅ "복잡하다" 심리 먼저 건드린 후 해결 → 심리 안정화
- ✅ 구체적 액션 4개 → "이 정도면 할 수 있지?" 심리
- ✅ 수치화된 결과 → 노력 대비 효과 명확
- ✅ "직접 하거나 저희가 도와주거나" → 선택권 제시 (압박 없음)
