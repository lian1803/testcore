# 데이터 분석 / 트래킹 서비스 패키지 구성표

> 작성일: 2026-04-15 | 기준: 단가 데이터(step4v2) + 벤더 리스트 + B2B 에이전시 마진 원칙

---

## 개요

데이터 분석 / 트래킹 서비스는 클라이언트의 마케팅 성과 측정 인프라를 구축하고 운영하는 서비스입니다. GA4, 픽셀, CRM 등 다채널 데이터를 통합 관리하여 의사결정을 지원합니다.

---

## 패키지 구성

### 1. 스타터 (월 50~80만원)
**대상**: 소규모 사업자, 신규 온라인 비즈니스, 월 광고비 500만원 이하

**포함 서비스**:
- GA4 기본 설치 및 계정 세팅
- 웹 데이터 스트림 생성 (1개 웹사이트)
- 기본 커스텀 이벤트 설계 및 세팅 (3~5개 이벤트: 버튼클릭, 폼 제출, 스크롤 등)
- 메타 픽셀(Meta Pixel) 설치 및 표준 이벤트 세팅
- 구글 애즈 전환 태그 설치
- Looker Studio 기본 대시보드 1개 (GA4 데이터 기반)
- 월별 성과 리포트 (수동 작성, 기본형)
- 분기별 데이터 품질 감사 (1회)
- 기술 지원 (이메일 기반, 평일 업무시간)

**가격대**:
- 최저가: 월 50만원 (GA4 + 메타 픽셀 기본)
- 평균가: 월 65만원
- 최고가: 월 80만원 (고급 커스텀 이벤트 포함)

**기대 성과**:
- 트래킹 데이터 품질 90% 이상
- 월별 성과 리포트 제공 (의사결정 근거 마련)
- 광고 효율 20~30% 개선 (측정 정확도 향상)

**적합 대상**:
- 신규 온라인 쇼핑몰/D2C
- 소규모 스타트업 (직원 5인 이하)
- 초기 디지털 마케팅 인프라 구축 필요 기업

**벤더 배정**:
- 크몽 프리랜서 풀 (GA4 기본 세팅 전문가)
- 커넥트리(Connectree) — 소형 클라이언트 담당

### 2. 스탠다드 (월 120~180만원)
**대상**: 성장기 브랜드, 멀티채널 트래킹 필요, 월 광고비 1,000~3,000만원

**포함 서비스**:
- GA4 고급 설치 (다중 도메인/사브도메인 추적)
- 커스텀 이벤트 설계 및 세팅 (10~15개: 페이지뷰, 클릭, 폼 제출, 스크롤, 비디오 재생 등)
- GTM(구글 태그 매니저) 계정 구축 및 기본 태그 세팅
- 메타 픽셀 + Meta Conversions API(CAPI) 구축 (서버사이드 추적)
- 구글 애즈 전환 태그 + 고급 설정
- 네이버 광고 전환 추적 세팅
- 카카오 픽셀 / 카카오 모먼트 세팅
- Looker Studio 고급 대시보드 2~3개 (GA4 + 광고 채널 통합)
- 월별 성과 리포트 (자동 + 수동 인사이트 추가, 표준형)
- 분기별 데이터 품질 감사 (2회)
- 월 1회 데이터 분석 컨설팅 (30분)
- 기술 지원 (이메일 + 전화, 평일 업무시간)

**가격대**:
- 최저가: 월 120만원 (표준 멀티채널 세팅)
- 평균가: 월 150만원
- 최고가: 월 180만원 (CAPI + 고급 대시보드)

**기대 성과**:
- 멀티채널 통합 트래킹 (메타, 구글, 네이버, 카카오)
- 트래킹 데이터 품질 95% 이상
- 광고 효율 30~50% 개선
- 월별 인사이트 제공 (최적화 근거 도출)

**적합 대상**:
- 이커머스 성장기 업체 (월 매출 5,000만~1억원)
- 온라인 서비스 스케일링 단계
- 멀티 채널 광고 집행 기업

**벤더 배정**:
- 오픈소스마케팅(OSOMA) — GA4 + GTM 전문
- 비즈스프링(BizSpring) — CAPI + 대시보드
- 크몽 프리랜서 풀 (중급자, 30~50만원대)

### 3. 프리미엄 (월 250~350만원)
**대상**: 중견 기업 이상, 풀스택 데이터 인프라, 월 광고비 5,000만원 이상

**포함 서비스**:
- GA4 엔터프라이즈급 설치 (다중 하위도메인, 앱 + 웹 통합, 크로스 디바이스)
- 고급 이벤트 설계 (20~30개: 모든 비즈니스 액션 추적)
- GTM 고급 태깅 (DataLayer 설계, 동적 변수 자동화)
- 메타 CAPI + 픽셀 이중 추적 (쿠키리스 환경 대응)
- 구글 애즈 + 구글 마케팅 플랫폼(GMP) 연동
- 네이버 광고 고급 추적 + 성과형 DA 통합
- 카카오, 틱톡 등 추가 채널 통합
- 서버사이드 태깅(Server-Side GTM) 구축 (선택)
- BigQuery 연동 및 원시 데이터 추출 환경 구축 (선택)
- Looker Studio 커스텀 대시보드 5개 이상 (역할/부서별)
- 월별 성과 리포트 (자동 생성 + 전략적 인사이트 + 최적화 제안, 고급형)
- 월 2회 데이터 분석 컨설팅 (1시간)
- 유입 경로(Attribution) 분석 및 멀티터치 모델링
- 데이터 품질 감사 (월 1회)
- 기술 지원 (이메일 + 전화 + Slack, 업무시간 +비상)
- 분기별 데이터 전략 회의

**가격대**:
- 최저가: 월 250만원 (CAPI + 기본 빅쿼리)
- 평균가: 월 300만원
- 최고가: 월 350만원 (풀스택 데이터 인프라 + BigQuery + Attribution)

**기대 성과**:
- 풀스택 데이터 인프라 구축 (쿠키리스 대응)
- 트래킹 데이터 품질 98% 이상
- 광고 효율 50~80% 개선
- 어트리뷰션 분석을 통한 예산 배분 최적화 (20~30% ROI 증대)
- 실시간 성과 모니터링 + 자동화된 인사이트 제공

**적합 대상**:
- 중견 커머스 (월 매출 1억원+)
- 멀티 브랜드 운영 기업
- 리쿠르팅, 금융, 교육 등 D2C 기업
- 데이터 기반 의사결정 문화 정착 기업

**벤더 배정**:
- 오픈소스마케팅(OSOMA) — GA4 풀 패키지 + 교육
- 비즈스프링(BizSpring) — GA4 + BigQuery + CDP
- 그로스플랫폼(Growth Platform) — BigQuery + 대시보드
- AB180(에어브릿지) — 어트리뷰션 + CDP 통합
- 위브랩(WIV Labs) — 서버사이드 태깅 + CAPI 최적화

## 계약 조건

### 기본 조건
- **최소 계약 기간**: 3개월 (스타터), 6개월 (스탠다드/프리미엄)
- **선금 규모**: 첫 2개월 선금 (신규), 1개월 선금 (계약갱신)
- **가격 조정**: 연 1회 (물가상승률 반영, 최대 +5%)
- **수정 횟수 제한**: 월 3회 (대시보드 수정) / 추가 수정은 건당 청구

### 해약 조건
- **조기 종료**: 30일 전 통보
- **해약 수수료**: 없음 (계약 기간 내 결제한 비용은 환불 불가)
- **데이터 이관**: 계약 종료 후 30일 내 데이터 이관 제공

### 갱신 조건
- **갱신 알림**: 계약 만료 90일 전 → 60일 전 → 30일 전
- **갱신 조건**: 기존 조건 유지 (단, 신규 기능/서비스 추가 시 협의)
- **중단 기간**: 0일 (자동 연속 갱신, 7일 전 통보 시 중단 가능)

## 프리랜서 단가 vs B2B 에이전시 마진 원칙

### 단가 구성
```
프리랜서 원가(크몽): 50만원
  ↓
에이전시 마진 (+30%): 15만원
  ↓
판매가(스탠다드): 65만원
```

### 마진율 기준
- **스타터**: 프리랜서 원가 + 25% = 판매가
- **스탠다드**: 프리랜서 원가 + 30% = 판매가
- **프리미엄**: 프리랜서 원가 + 40% = 판매가 (고도화/운영 비용 추가)

## 예상 고객 사례

### 사례 1: 신규 온라인 쇼핑몰 (월 광고비 300만원)
- **추천**: 데이터 분석 스타터 (65만원)
- **월 투자**: 65만원 + 광고비 300만원 = 365만원
- **기대 효과**: GA4 기본 추적 완성, 월별 성과 리포트로 광고 최적화 근거 마련
- **3개월 후 예상 매출**: 월 800만~1,000만원

### 사례 2: 성장 중인 이커머스 (월 광고비 1,500만원)
- **추천**: 데이터 분석 스탠다드 (150만원)
- **월 투자**: 150만원 + 광고비 1,500만원 = 1,650만원
- **기대 효과**: 멀티채널 통합 트래킹, CAPI 기반 쿠키리스 대응, 실시간 대시보드
- **3개월 후 예상 매출**: 월 4,000~5,000만원 (30% ROAS 개선)

### 사례 3: 중견 D2C 브랜드 (월 광고비 5,000만원)
- **추천**: 데이터 분석 프리미엄 (300만원)
- **월 투자**: 300만원 + 광고비 5,000만원 = 5,300만원
- **기대 효과**: 풀스택 데이터 인프라, 어트리뷰션 분석, BigQuery 기반 심화 분석
- **3개월 후 예상 매출**: 월 3억~4억원 (50% ROAS 개선, 예산 배분 최적화)

## 기존 온라인 마케팅 패키지와의 결합

데이터 분석 서비스는 기존 퍼포먼스 마케팅, SNS 마케팅 등과 결합하여 시너지를 창출합니다.

### 통합 패키지 예시
| 조합 | 월 투자 | 대상 | 기대 효과 |
|------|--------|------|---------|
| 퍼포먼스 스타터 + 데이터 분석 스타터 | 115만원 | 신규 스타트업 | 광고 운영 + 성과 측정 |
| 퍼포먼스 스탠다드 + 데이터 분석 스탠다드 + SNS 스타터 | 290만원 | 성장 중인 이커머스 | 멀티채널 광고 + 통합 분석 + SNS 운영 |
| 퍼포먼스 프리미엄 + 데이터 분석 프리미엠 + 콘텐츠/바이럴 스탠다드 | 630만원 | 중견 브랜드 | 풀 채널 전략 + 데이터 기반 의사결정 |
