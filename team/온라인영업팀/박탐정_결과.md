## ⚠️ 박탐정 → 리안 긴급 보고

**업무 지시와 팀 미션 간 심각한 불일치 발견**

### 📋 지시받은 업무
```
양주 미용실 대상 온라인 마케팅 영업 
— 네이버플레이스 진단 후 컨설팅 제안
```

### 🎯 온라인영업팀 실제 타겟
```
쿠팡/스마트스토어 셀러 
→ 잠재고객 발굴 → 진단 → 아웃리치 → 계약
```

### ❌ **왜 문제인가?**
1. **타겟 업종 불일치**: 미용실(로컬 서비스업) ≠ 온라인 셀러(이커머스)
2. **마케팅 채널 불일치**: 네이버플레이스 ≠ 쿠팡/스마트스토어
3. **고객 Pain 불일치**: 미용실은 예약/내방 고객 필요 vs 셀러는 온라인 매출 필요
4. **진단 기준 불일치**: 네이버리뷰/블로그 vs 상품페이지/광고 성과

## 🔍 리안, 이거 확인 필요해

### 질문 1: **실제 타겟이 누구야?**
- A안) 양주 미용실 (+ 음식점/카페 등 로컬 업종)  
  → 그럼 팀 미션을 "로컬 소상공인 온라인 마케팅 대행"으로 **재설정** 필요

- B안) 쿠팡/스마트스토어 셀러 (현재 팀 미션대로)  
  → 그럼 이번 업무 지시가 **잘못됨**. 셀러 타겟으로 다시 지시 필요

### 질문 2: **영업 상품이 뭐야?**
- 네이버플레이스 컨설팅 (리뷰 관리/블로그 작업)?
- 온라인몰 마케팅 대행 (상품페이지/광고)?
- 둘 다?

## 🛠️ 박탐정이 할 수 있는 2가지 옵션

### 옵션 1: **양주 미용실 타겟 작업 진행** (지시대로)
→ 단, 팀 미션과 맞지 않으니 **일회성 프로젝트**로 처리  
→ 산출물: "양주 미용실 잠재고객 리스트 + 네이버플레이스 진단 체크리스트"

**예상 결과물:**
- 네이버 "양주 미용실" 검색 → 리뷰 30개 미만 업체 리스트
- 인스타그램 팔로워 500명 미만 업체 리스트
- 블로그 없거나 3개월 이상 미운영 업체 리스트
- 네이버플레이스 진단 체크리스트 (리뷰수/사진수/예약기능/블로그연동 점수표)

### 옵션 2: **팀 미션에 맞게 쿠팡/스마트스토어 셀러 타겟 작업** (미션대로)
→ 업무 지시 무시하고 **본래 팀 목표**에 집중  
→ 산출물: "온라인 셀러 잠재고객 발굴 가이드 + 매출/광고비 기준 체크리스트"

**예상 결과물:**
- 스마트스토어 "월매출 500만원 이상" 찾는 법 (네이버 스토어 검색어 리스트)
- 쿠팡 "로켓배송 셀러" 식별 기준 (리뷰 100개 이상 + 판매자 정보 공개)
- 광고 집행 여부 판단 기준 (상품페이지에 "프로모션" 배너 있음 = 광고비 쓰는 중)
- 잠재고객 우선순위 점수표 (매출 추정치/광고 집행 여부/콘텐츠 품질 점수)

## 💬 박탐정의 솔직한 조언

**리안, 너 지금 혼란스러운 거 맞지?**

내가 보기엔 두 가지 중 하나야:
1. **업무 지시를 잘못 받았거나** (미용실이 아니라 셀러 타겟이어야 하는데 실수로 미용실 넣었거나)
2. **팀 미션이 아직 안 굳어진 거야** (로컬 업종도 하고 온라인 셀러도 하고 싶은데 방향이 안 정해짐)

**내 추천:**
- 지금 당장 **"우리 팀이 누구한테 돈 받을 건지"** 결정해
- 미용실이면 → 팀 미션 바꿔 (로컬 소상공인 전문)
- 셀러면 → 이번 업무 취소하고 셀러 타겟 작업해

**혼재하면 절대 안 되는 이유:**
- 진단 기준이 완전히 달라 (네이버플레이스 vs 쇼핑몰 상품페이지)
- DM 스크립트가 완전히 달라 (예약 늘려드려요 vs 매출 올려드려요)
- 제안서 구성이 완전히 달라 (리뷰 관리 vs 상품페이지 최적화)
