# PRD: 1인 사업자 AI 세금신고 도우미

**문서 버전**: v1.0
**작성자**: 지훈 (실행팀 PRD)
**작성일**: 2025-01-27
**상태**: CONDITIONAL_GO — 컨시어지 MVP 단계

---

## 제품 개요

1인 사업자·프리랜서가 영수증 사진을 찍으면 AI가 자동 분류하고 종합소득세 신고서 초안을 생성해주는 **"신고 준비 도구"** (세무 대리 행위 아님).

---

## Must Have

1. **영수증 OCR 업로드 + AI 자동 분류** — 핵심 가치 제안. 사용자가 사진 한 장으로 경비 처리 시작. 없으면 제품이 아님
2. **경비 항목 수동 수정 UI** — OCR 오류 보정 필수. 정확도 100% 불가능하므로 사용자 최종 확인 레이어 필수 (법적 책임 분산)
3. **종합소득세 신고서 초안 자동 생성 (PDF/Excel)** — 핵심 아웃풋. 사용자가 국세청 홈택스에 직접 입력할 수 있는 수준의 초안
4. **연간 경비 대시보드** — 월별/카테고리별 지출 현황 시각화. 사용자 재방문 유도 + 절세 포인트 인지
5. **회원가입/로그인 + 사업자 프로필 설정** — 업종코드·과세유형(일반/간이/면세)에 따라 공제 항목이 달라지므로 필수
6. **신고서 다운로드 + 홈택스 입력 가이드** — "신고 대행"이 아닌 "준비 도구"임을 명확히 하는 UX. 법적 포지셔닝 핵심
7. **구독 결제 (월/연간)** — 수익 모델 검증. 무료 체험 후 유료 전환 플로우

---

## Must NOT (범위 외)

| 제외 기능 | 이유 |
|-----------|------|
| 홈택스 자동 제출 (API 연동) | 세무 대리 행위 해당 가능성 → 세무사법 위반 리스크. 법률 검토 완료 전 절대 불가 |
| 세무사 매칭/연결 서비스 | 이번 MVP 범위 외. 추후 업셀링 옵션으로 검토 |
| 부가가치세(VAT) 신고 | 종합소득세 단일 집중으로 MVP 완성도 확보. 분산 금지 |
| 법인 사업자 지원 | 세법 구조 완전히 다름. 1인 사업자/프리랜서만 타겟 |
| 실시간 세무 상담 챗봇 | 세무사법상 유권해석 필요. 법률 검토 전 제외 |
| 카드사/은행 자동 연동 (스크래핑) | 금융정보 수집 규제 복잡. 2차 개발에서 검토 |
| 다국어 지원 | 초기 한국 시장 집중 |

## User Flow

### 시나리오 1: 신규 사용자 — 첫 영수증 등록부터 신고서 생성까지

```
1단계: 앱 진입 → 랜딩 페이지 (가치 제안 + CTA "무료로 시작하기")
2단계: 회원가입 → 이메일/소셜 로그인 → 이메일 인증
3단계: 사업자 프로필 설정
       → 업종 선택 (드롭다운: 프리랜서/소매업/서비스업 등)
       → 사업자등록번호 입력 (선택, 미입력 시 프리랜서로 처리)
       → 과세유형 선택 (일반/간이/면세/소득세만 해당)
       → 신고 대상 연도 선택 (기본값: 직전 연도)
4단계: 대시보드 진입 → "영수증 추가" CTA 클릭
5단계: 영수증 업로드
       → 카메라 촬영 또는 갤러리 선택
       → 이미지 업로드 → OCR 처리 (로딩 스피너, 최대 10초)
       → AI 분류 결과 표시: [날짜 / 금액 / 상호명 / 분류 카테고리]
6단계: 분류 결과 확인 및 수정
       → 카테고리 오분류 시 드롭다운으로 수동 변경
       → "사업 관련 경비 맞음" 토글 확인
       → 저장
7단계: 경비 누적 후 신고서 생성 요청
       → "신고서 만들기" 버튼 클릭
       → 소득 정보 입력 (연간 매출액, 기타 소득)
       → AI 신고서 초안 생성 (처리 시간 최대 30초)
8단계: 신고서 초안 검토
       → 항목별 금액 확인
       → 수정 필요 시 경비 목록으로 돌아가 수정
9단계: 다운로드
       → PDF 또는 Excel 선택 다운로드
       → 홈택스 입력 가이드 팝업 표시 (단계별 스크린샷 포함)
10단계: 유료 전환 프롬프트
        → 무료 플랜: 연 1회 신고서 생성, 영수증 20건 제한
        → "더 쓰려면 구독하세요" → 결제 페이지
```

### 시나리오 2: 기존 사용자 — 영수증 추가 등록 (일상적 사용)

```
1단계: 앱 로그인 → 대시보드 바로 진입
2단계: 하단 FAB(+) 버튼 → 영수증 촬영
3단계: OCR 결과 확인 → 저장 (10초 이내 완료 목표)
4단계: 대시보드에서 월별 경비 누적 현황 확인
```

### 시나리오 3: 구독 결제

```
1단계: 무료 한도 초과 시 페이월 표시
2단계: 플랜 선택 (월간 9,900원 / 연간 79,200원)
3단계: 카드 정보 입력 (Stripe/토스페이먼츠)
4단계: 결제 완료 → 프리미엄 기능 즉시 활성화
```
