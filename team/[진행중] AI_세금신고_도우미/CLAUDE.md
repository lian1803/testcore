# AI 세금신고 도우미 — Claude Code 구현 지시서 v1.0

---

## 시장 리서치 요약 (이사팀 산출물)

### 타겟 사용자
- 25~45세 1인 사업자·프리랜서 (디자이너, 개발자, 유튜버, 강사, 배달라이더 등)
- 세무사 비용(연 40만원) 부담스럽거나 직접 신고 원하는 사람
- 종합소득세 신고 시즌(5월)에 가장 수요 집중

### 핵심 Pain Point
- 영수증 수동 정리 → 하루 1~2시간 낭비
- 세무사 없이 신고 시 어디서 시작할지 모름
- 삼쩜삼은 환급 특화, 자비스는 기장 서비스 — 영수증 OCR + 신고서 생성 공백 존재

### 경쟁사 분석
| 경쟁사 | 약점 |
|--------|------|
| 삼쩜삼 | 환급 특화, 영수증 관리 미흡 |
| 자비스 | 기장 서비스 중심, 고가 |
| 캐시노트 | 신고 기능 없음, 소상공인 특화 |

### 핵심 차별점
- 영수증 OCR + AI 자동분류 → 신고서 초안 one-stop
- "신고 준비 도구" 포지셔닝 (세무 대리 아님 → 세무사법 안전)
- 5월 종소세 시즌 집중 타겟팅

## 구현 주의사항 (준혁 조건부 GO 조건 반영)

### 🔴 MVP 범위 (절대 준수)
- **1차 MVP**: 영수증 OCR + AI 분류 + 종합소득세 신고서 초안 생성
- **홈택스 자동 제출**: 절대 금지 (세무사법 위반 리스크)
- **부가세 신고, 법인 지원**: 2차 릴리즈
- **카드사·은행 스크래핑**: 금융정보 수집 규제 복잡 → 제외

### 🟡 법적 포지셔닝 (전체 UX에 반드시 반영)
- 모든 UI 문구: "신고 준비 도구", "참고용", "최종 책임은 사용자에게"
- 홈택스 직접 입력 가이드 제공 (대행 아님 명시)
- 개인정보: 영수증 원본 처리 후 삭제 옵션 필수

### 🟡 수익 모델
- 무료: 영수증 20건/월, 신고서 1회/년
- 유료(월 9,900원 / 연 79,200원): 무제한
