# 💰 AI 세금신고 도우미 MVP - 수익 모델 설계 및 유료 전환 전략

## 🎯 수익 모델 핵심 원칙 (리안 타겟 분석 기반)

**타겟:** 프리랜서/디자이너/개발자/유튜버 (연소득 3,000만~7,000만원)
**기존 대안:** 
- 홈택스 직접 신고 (무료, 평균 8시간 소요, 공제 누락률 40%)
- 세무사 위임 (연 40만원, 환급액 평균 120만원)

**우리의 가치 제안:** 환급액 극대화 + 시간 절약 + 압도적 가성비

---

## 📊 수익 모델 비교 분석서 (3가지 시나리오)

### **시나리오 A: 구독료 단일 모델 (SaaS 정통)**

| 요금제 | 월 가격 | 연 가격 (할인) | 제공 기능 | 타겟 |
|--------|---------|----------------|-----------|------|
| **무료** | ₩0 | ₩0 | • 예상 환급액 조회<br>• 영수증 스캔 월 10건<br>• 공제 항목 체크리스트 | 유입 + 데이터 축적 |
| **베이직** | ₩9,900 | **₩99,000 (₩19,800 절약)** | • 무제한 영수증 스캔<br>• 종소세 신고서 자동생성<br>• 1회 세무사 검토 | **메인 전환 타겟** (연 1회 신고) |
| **프로** | ₩16,500 | **₩198,000 (₩0 추가)** | • 베이직 전체<br>• 분기별 세무 컨설팅 3회<br>• 장부 자동 기장 (간편장부) | N잡러/고소득자 |

**예상 재무 지표 (MAU 1만명 기준):**
```
무료→유료 전환율: 12% (벤치마크: 세무 SaaS 10-15%)
유료 사용자 1,200명 × ₩99,000 = 연 매출 ₩118,800,000
- CAC ₩25,000 (네이버 키워드 광고) → LTV/CAC = 3.96 ✅
- 월간 이탈률 3% (연간 구독 → 계절성 낮음)
- ARPU: ₩9,900 (월 기준)
```

**장점:**
- 예측 가능한 매출 (MRR)
- 단순한 가격 구조 → 의사결정 빠름
- 삼쩜삼 대비 "1/4 가격" 앵커링 효과

**단점:**
- 환급액 적은 유저는 ROI 체감 어려움
- 신고 시즌(5월) 외 이탈 위험

### **시나리오 B: 성과 수수료 모델 (삼쩜삼 벤치마크)**

| 구조 | 수수료율 | 최소 수수료 | 최대 수수료 |
|------|----------|-------------|-------------|
| **환급액 기반** | 환급액의 **15%** | ₩50,000 | ₩300,000 |

**예시:**
- 환급액 80만원 → 수수료 ₩120,000 (세무사 40만원 대비 70% 저렴)
- 환급액 200만원 → 수수료 ₩300,000 (상한 적용)

**예상 재무 지표 (MAU 1만명):**
```
신고 완료율: 8% (무료 사용자 중 실제 신고)
평균 환급액: ₩1,200,000 (벤치마크: 삼쩜삼 120만원)
평균 수수료: ₩180,000 (15% 적용)

800명 × ₩180,000 = 연 매출 ₩144,000,000
- CAC ₩30,000 (환급 보장 메시지 효과) → LTV/CAC = 4.8 ✅
- 이탈률 60% (다음 해 재사용 40%)
```

**장점:**
- "성공하면 과금" 심리적 저항 낮음
- 환급액 큰 유저에서 고수익
- 바이럴 효과 ("나는 150만원 환급받고 15만원만 냈어")

**단점:**
- 환급 없으면 매출 0 → 수익 변동성 큼
- 세무사법 저촉 가능성 (수임 행위 논란)
- 경쟁사(삼쩜삼 10-20%) 대비 차별화 필요

### **시나리오 C: 하이브리드 (추천 ⭐)**

**구조:** 기본 구독료 + 환급 성과 보너스

| 요금제 | 연 구독료 | 환급 보너스 | 총 예상 비용 |
|--------|-----------|-------------|--------------|
| **스탠다드** | **₩99,000** | 환급액 ₩100만원 초과분의 **5%** | 환급 150만원 → 총 ₩124,000 |
| **프리미엄** | **₩0** | 환급액의 **12%** | 환급 150만원 → 총 ₩180,000 |

**예상 재무 지표 (MAU 1만명):**
```
스탠다드 선택: 70% (840명) → ₩99,000 + 평균 보너스 ₩25,000 = ₩104,160,000
프리미엄 선택: 30% (360명) → 평균 수수료 ₩180,000 = ₩64,800,000
총 매출: ₩168,960,000

- CAC ₩28,000 → LTV/CAC = 4.2 ✅
- 이탈률 45% (구독 유지 인센티브 + 성과 연동)
- ARPU: ₩14,080 (혼합)
```

**장점:**
- **리스크 분산**: 구독료로 기본 매출 확보 + 성과로 업사이드
- **유저 선택권**: 환급 자신 있으면 프리미엄 (수수료만), 안정 원하면 스탠다드
- **경쟁 차별화**: 삼쩜삼(수수료만) vs 세무사(고정비) 중간 포지셔닝

**단점:**
- 가격 구조 복잡 → 설명 필요 (랜딩 페이지 섹션 추가)
- 두 모델 동시 운영 복잡도

## 🎯 프라이싱 전략서 (심리학 + 데이터 기반)

### **1. 가격 앵커링 (Price Anchoring)**

**랜딩 페이지 메시지:**
```
❌ 세무사 기장료 연 40만원
❌ 홈택스 직접 신고 평균 8시간 소요
✅ AI 세금신고 도우미 연 9.9만원
   → 세무사 대비 75% 절약 + 시간 95% 단축
```

**가격 표시 전략:**
- ~~₩16,500/월~~ → **₩9,900/월** (연 구독 시)
- "하루 커피 한 잔 (₩330) 가격"
- "환급액 평균 120만원 → 투자 대비 **12배 리턴**"

### **2. 가격 민감도 테스트 (Van Westendorp PSM 적용)**

**설문 질문 (n=200 타겟 유저):**
1. "너무 저렴해서 품질이 의심되는 가격은?" → 하한선
2. "가성비 좋다고 느끼는 가격은?" → 최적 하단
3. "비싸지만 납득 가능한 가격은?" → 최적 상단
4. "너무 비싸서 절대 안 사는 가격은?" → 상한선

**예상 결과 (세무 SaaS 벤치마크):**
```
하한선: ₩50,000 (이하 = 불신)
최적 하단: ₩80,000 (가성비 체감 시작)
최적 상단: ₩150,000 (프리미엄 한계)
상한선: ₩250,000 (이상 = 이탈)

→ 최적 가격대: ₩99,000 (중앙값)
```
