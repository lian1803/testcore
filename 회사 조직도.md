## 조직 원칙

**중앙 기능팀 없음.** 모든 역할(디자이너, 마케터, 개발자, 영업 등)이 프로젝트 팀에 직접 배치.
프로젝트마다 팀이 따로 구성되고, 프로젝트가 끝나면 팀도 해체.

**고정팀 (회사 공용 인프라):**
- 이사팀 — 분석 + 팀 설계 (항상 유지)
- 교육팀 — 지식 관리 (항상 유지)
- UltraProduct팀 — 코드 개발 전용 고정 실행 엔진 (항상 유지, 소프트웨어 프로젝트 시 실행)

**프로젝트팀 (필요 시 신설):**
- 이사팀이 설계 → 교육팀이 커리큘럼 제공 → 실행
