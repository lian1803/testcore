# /theme-factory — 컬러 팔레트 + 폰트 시스템 확정

이 명령어가 실행되면 아래를 즉시 수행해라. 질문하지 마라.

---

## 입력 파악

현재 프로젝트 폴더에서 다음을 읽어라:
1. `CLAUDE.md` 또는 `wave1_채널사전판단.md` — 디자인 방향 섹션
2. `wave1_cdo.md` (있으면) — CDO 초기 방향

읽은 내용에서 다음을 추출해라:
- 브랜드 톤/무드 키워드
- 타겟 사용자 특성
- 마케팅 채널 (인스타/블로그/B2B 등)
- 경쟁사 UI 특징

---

## 컬러 팔레트 (oklch 색공간)

| 역할 | 이름 | oklch 값 | HEX 근사 | 사용처 |
|------|------|----------|----------|--------|
| Primary | [이름] | oklch(...) | #... | CTA 버튼, 링크, 강조 |
| Secondary | [이름] | oklch(...) | #... | 보조 버튼, 배지 |
| Accent | [이름] | oklch(...) | #... | 하이라이트, 아이콘 |
| Background | [이름] | oklch(...) | #... | 페이지 배경 |
| Surface | [이름] | oklch(...) | #... | 카드, 모달 배경 |
| Border | [이름] | oklch(...) | #... | 구분선, 인풋 테두리 |
| Text Primary | [이름] | oklch(...) | #... | 본문, 제목 |
| Text Secondary | [이름] | oklch(...) | #... | 설명, 라벨 |
| Success | - | oklch(...) | #... | 완료, 긍정 |
| Warning | - | oklch(...) | #... | 주의 |
| Error | - | oklch(...) | #... | 에러 |

다크모드: [지원 여부 + 다크 팔레트 별도 표기]
