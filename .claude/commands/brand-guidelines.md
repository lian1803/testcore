이 명령어가 실행되면 아래를 즉시 수행해라. 질문하지 마라.

---

## 입력 파악

현재 프로젝트 폴더에서 다음을 읽어라:
1. `wave1_theme.md` — /theme-factory 결과 (컬러 + 폰트)
2. `CLAUDE.md` — 프로젝트 포지셔닝, 타겟, 디자인 방향

---

```
## 브랜드 포지셔닝

- **핵심 가치**: [3단어]
- **브랜드 퍼스널리티**: [예: 신뢰할 수 있는 전문가 / 친근한 조력자 / 혁신적인 선구자]
- **한 줄 포지셔닝**: [타겟에게 [가치]를 주는 [카테고리]]

## 보이스 & 톤

| 상황 | 톤 | 예시 문구 |
|------|-----|----------|
| 버튼 CTA | 능동적, 간결 | "지금 시작하기" / "무료로 해보기" |
| 에러 메시지 | 친근, 해결책 제시 | "잠깐, 뭔가 잘못됐어요. 다시 시도해볼까요?" |
| 빈 상태 | 격려 | "아직 데이터가 없어요. 첫 번째를 만들어볼까요?" |
| 성공 | 축하, 긍정 | "완료됐어요!" |
| 로딩 | 안심 | "준비 중이에요..." |

## 컴포넌트 사용 규칙

### 버튼
- Primary: [컬러] + [폰트 굵기] + [라운드] + hover 효과
- Secondary: [컬러] + border + hover 효과
- Ghost: 투명 배경 + hover시 [컬러] tint
- 크기: sm(32px) / md(40px) / lg(48px)
- 금지: 그림자 과다, 네온 효과 (브랜드 톤에 안 맞으면)

### 카드
- 배경: Surface 컬러
- 테두리: Border 컬러 1px
- 라운드: [px]
- 그림자: [shadow-sm / shadow-md / 없음]
- hover: [효과 명시]

### 폼/인풋
- 배경: Background
- 테두리: Border, focus시 Primary
- 라운드: [px]
- 레이블 위치: 위 (floating label 금지 — 모바일 UX 불편)

### 아이콘
- 라이브러리: lucide-react (shadcn/ui 기본)
- 크기: 16px(인라인) / 20px(버튼) / 24px(섹션)
- 컬러: 텍스트 컬러 계열 (아이콘 단독 강조 색 금지)

## 간격 시스템 (8px 그리드)

| 용도 | 값 |
|------|----|
| 컴포넌트 내부 패딩 | 8px / 12px / 16px |
| 섹션 간격 | 48px / 64px / 80px |
| 페이지 수평 패딩 | 16px(모바일) / 24px(태블릿) / 32px+(데스크탑) |
| 카드 내부 패딩 | 16px / 24px |

## 금지 사항

- 지정된 팔레트 외 색상 임의 추가 금지
- 폰트 굵기 900 이상 금지 (너무 무거움)
- 섹션 배경에 그라디언트 남발 금지 (1~2군데만)
- 아이콘 없이 버튼만 있는 빈 상태 화면 금지
- 모바일에서 hover-only 인터랙션 금지

```
