## ── 세계 최고 카피라이터 워크샵 — 10년차 카피 마스터클래스 ──

### ✍️ 핵심 카피 공식 완전 정복

---

#### PAS (Problem → Agitate → Solution)

**구조:** 문제 제시 → 고통 증폭 → 해결책 제시

**왜 효과적인가:** 사람은 이익을 얻는 것보다 손해를 피하는 것에 2배 더 민감하다 (손실 회피 편향). PAS는 그 심리를 정확히 건드린다.

**마케터 SaaS 타겟 실제 예시:**

```
[Problem]
인스타 광고 돌리는데 왜 매달 성과가 다른지 모르시겠죠?

[Agitate]
이번 달 ROAS 3.2인데 저번 달은 왜 1.8이었는지.
어떤 광고가 진짜 돈 버는 건지, 어떤 게 그냥 돈 태우는 건지.
매번 감으로 판단하다 보면 틀릴 때마다 그게 다 내 돈이에요.

[Solution]
Lian Dash는 광고 채널별 성과를 한 화면에 모아서
"이 캠페인이 문제야"를 5초 만에 보여줍니다.
```

**언제 쓰나:** 콜드 이메일 첫 단락, 인스타 광고 본문, 랜딩페이지 문제 섹션

---

#### AIDA (Attention → Interest → Desire → Action)

**구조:** 시선 잡기 → 흥미 유발 → 욕망 자극 → 행동 유도

**마케터 SaaS 타겟 실제 예시:**

```
[Attention — 헤드라인]
광고비 낭비 원인, 5초 만에 찾아드립니다

[Interest — 서브헤드]
구글·메타·카카오 데이터를 하나로 연결.
채널별 ROAS, 클릭당 비용, 전환율을 실시간으로 비교합니다.

[Desire — 사회적 증거 + 구체 수치]
이미 인하우스 마케터 200명이 쓰고 있어요.
"첫 달에 광고비 30% 절약했습니다" — 이커머스 마케터 김○○

[Action — CTA]
무료로 2주 써보기 → (신용카드 불필요)
```

**언제 쓰나:** 랜딩페이지 Hero 섹션, 이메일 전체 구조, 광고 카피 풀 버전

---

#### Hook-Story-Offer (Russell Brunson 공식)

**구조:** 첫 줄에 모든 것을 걸어라 → 공감 스토리 → 제안

**왜 DM/이메일에서 최강인가:** 첫 문장을 안 읽으면 나머지는 존재하지 않는다. Hook이 전부다.

**마케터 타겟 DM 실제 예시:**

```
[Hook — 첫 줄]
혜경님 인스타 보다가 궁금한 게 생겨서요.

[Story — 2~3문장]
저도 예전에 광고 채널 6개 쓰면서 엑셀로 수동 집계했는데,
매주 리포트 만드는 데만 3시간씩 날렸거든요.
그때 만든 게 지금 Lian Dash예요.

[Offer — 제안]
무료로 2주 써보실래요?
설치 10분, 설정 5분이에요.
```

**언제 쓰나:** 인스타 DM, 콜드 이메일, 짧은 광고 카피

### 🎣 훅(Hook) 작성 기법 20가지

훅의 법칙: **첫 문장이 다음 문장을 읽게 만들어야 한다.** 그게 전부다.

**패턴 1 — 구체적 숫자**
> "광고비 월 300만 원 쓰는 마케터가 놓치는 것 3가지"
> → 숫자는 뇌가 구체성을 인식하게 만든다. "몇 가지"보다 3배 클릭 높음.

**패턴 2 — 역설/반직관**
> "ROAS가 높을수록 오히려 손해인 이유"
> → 예상을 깨는 문장. "이게 무슨 말이야?"가 클릭을 만든다.

**패턴 3 — 질문형**
> "마케팅 툴 5개 구독하고 있는데, 진짜 쓰는 건 몇 개예요?"
> → 독자가 자신에게 대입하게 만든다. 공감이 생기면 읽는다.

**패턴 4 — 공감형**
> "저만 매달 리포트 쓰는 게 제일 싫었나요?"
> → "나도!"의 감정. 광고가 아닌 대화처럼 느끼게 한다.

**패턴 5 — 두려움/손실**
> "지금 이 광고가 돈 버는지 태우는지, 모르고 있을 수도 있어요"
> → 손실 회피 본능. 모른다는 불안이 클릭을 만든다.

**패턴 6 — 사회적 증거 훅**
> "인하우스 마케터 200명이 엑셀 버린 이유"
> → '나만 뒤처지는 건 아닐까' 심리 자극.

**패턴 7 — 비밀/내부 정보**
> "광고 대행사가 클라이언트에게 안 알려주는 것"
> → 독점 정보에 대한 호기심.

**패턴 8 — 시간 절약**
> "주간 리포트 3시간 → 5분으로 줄인 방법"
> → 마케터의 현실적 고통 직격.

**패턴 9 — 개인화**
> "혜경님 인스타 보다가 생각났는데요"
> → 이름 + 구체적 참조. 일반 DM과 즉시 차별화.

**패턴 10 — 결과 먼저**
> "첫 달에 광고비 28% 절약한 마케터 이야기"
> → 결론을 먼저 주면 '어떻게?'가 궁금해진다.

**패턴 11 — 시나리오**
> "만약 광고 채널이 10개라면, 지금 어디서 돈 새는지 알 수 있나요?"
> → 독자의 상황을 가정해서 자기 이야기로 만든다.

**패턴 12 — 불편한 진실**
> "솔직히 말하면, 마케터 대부분이 감으로 광고 집행해요"
> → 업계의 불편한 현실을 인정하면 신뢰가 생긴다.

**패턴 13 — 비교**
> "경쟁사는 데이터 보면서 결정하는데, 우리 팀은?"
> → 경쟁 의식과 뒤처짐의 두려움.

**패턴 14 — 단순화**
> "광고 성과 분석, 사실 이것만 보면 됩니다"
> → 복잡한 것을 쉽게 만들어주겠다는 약속.

**패턴 15 — 경험 공유**
> "저도 광고 대행사에서 5년 일했는데요"
> → 전문성을 자연스럽게 드러내는 방식.

**패턴 16 — 도발**
> "광고 툴 더 많이 쓴다고 성과 좋아지지 않습니다"
> → 반대 의견. 논쟁을 일으키면 읽힌다.

**패턴 17 — 시의성**
> "2025년 마케터가 가장 많이 버리는 것"
> → 연도/현재성. 지금 읽지 않으면 뒤처진다는 느낌.

**패턴 18 — 스토리 시작**
> "3년 전에 광고비 600만 원을 하루에 날린 적이 있어요"
> → 구체적 사건으로 시작. 다음이 궁금해진다.

**패턴 19 — 원인 분석**
> "ROAS가 갑자기 떨어지는 진짜 이유 (대행사가 말 안 해주는 것)"
> → 숨겨진 원인 + 내부 정보 조합.

**패턴 20 — 약속**
> "이 글 읽고 나면 광고 리포트 보는 방식이 바뀔 거예요"
> → 독서의 결과를 미리 제시. 시간 투자의 이유를 만든다.

### 📧 콜드 이메일/DM 변환율 높이는 공식

#### 제목줄 (Subject Line) — 오픈율 결정

**최적 길이:** 30~50자 (모바일 기준)
**핵심 원칙:** 클릭베이트 금지. 본문과 일치해야 신뢰가 생긴다.

| 패턴 | 예시 | 효과 |
|------|------|------|
| 개인화 + 역할 | "[혜경님] 인하우스 마케터분들께 드리는 질문" | 오픈율 +40% |
| 구체 수치 | "광고비 28% 절약한 마케터 사례" | 클릭률 2배 |
| 짧고 직접적 | "Lian Dash 2주 무료 제안" | 명확한 의도 |
| 호기심 유발 | "마케터들이 엑셀 버리는 이유" | 궁금증 자극 |
| 공감형 | "주간 리포트 쓰는 거 저만 힘들었나요?" | 공감 → 오픈 |

**피해야 할 것:** "안녕하세요 저는~", "무료 제공!", 대문자 과다, 느낌표 3개 이상

---

#### 첫 문장 — 답장률 결정

**규칙:** 나 이야기 금지. 상대방 이야기로 시작.
**최적 길이:** 1~2문장. 끊어라.

```
❌ 나쁜 예:
"안녕하세요. 저는 Lian Dash의 마케팅 담당자 예진입니다.
저희는 마케터를 위한 데이터 분석 솔루션을 제공하고 있습니다."

✅ 좋은 예:
"혜경님 인스타 보다가 광고 여러 채널 쓰시는 것 같아서요."
"최근 메타 광고 성과 급락 겪으신 분들 많더라고요."
"인하우스 마케터로 일하다 보면 리포트가 제일 귀찮죠."
```

---

#### 가치 제안 본문 — 1~3문장

**공식:** [타겟 고통] + [구체적 결과] + [왜 우리인지]

```
✅ 예시:
"광고 채널 5개면 데이터 취합만 2시간이잖아요.
Lian Dash 쓰면 그게 5분으로 줄어요.
이미 인하우스 마케터 200명이 쓰고 있고요."
```

---

#### CTA (Call to Action) — 다음 행동 결정

**원칙:** CTA는 하나만. 여러 개면 아무것도 안 한다.
**마찰 최소화:** "구매" 말고 "써보기"나 "보기"로 낮춰라.

| 상황 | 강한 CTA | 약한 CTA (권장) |
|------|---------|----------------|
| 첫 DM | "지금 구매하세요" ❌ | "2주 무료로 써보실래요?" ✅ |
| 이메일 | "데모 신청하기" ❌ | "15분 통화 괜찮으세요?" ✅ |
| 광고 | "지금 가입" ❌ | "무료 시작하기 (카드 불필요)" ✅ |

**마찰 제거 문구 (CTA 옆에 붙이기):**
- "신용카드 불필요"
- "10분이면 셋업 완료"
- "언제든 해지 가능"
- "설치 없음, 바로 시작"

```
그래서 만든 게 Lian Dash예요.

2주 무료로 써보실 의향 있으세요?

— 예진 드림
```

### 🇰🇷 한국어 카피 특수 기법

#### 1. 존댓말 레벨 최적화

| 타겟 | 권장 레벨 | 예시 |
|------|---------|------|
| 20~30대 인하우스 마케터 | 해요체 (친근 + 존중) | "~해요", "~거든요", "~잖아요" |
| 40대 이상 마케터/대표 | 합쇼체 | "~합니다", "~십시오" |
| SNS 캐주얼 콘텐츠 | 반말체 혼합 | "~임", "~ㅋㅋ", "~다고?" |

**인하우스 마케터 최적 톤:** 해요체 + 구어체 혼합
> "이거 솔직히 귀찮으셨죠? 저도 그랬거든요."

---

#### 2. 한국어 리듬감 — 짧게 끊어라

```
❌ 나쁜 예 (너무 길어서 읽기 싫다):
"광고 채널이 많아질수록 각 채널별 데이터를 취합하고 분석하는 데
소요되는 시간이 증가하여 마케터의 업무 효율성이 저하됩니다."

✅ 좋은 예 (리듬감 + 읽히는 카피):
"광고 채널 늘수록 데이터도 늘어요.
그 데이터 취합하다 보면 정작 분석할 시간이 없어요.
Lian Dash가 그 부분을 해결해요."
```

**원칙:** 한 문장 = 한 생각. 끊을 수 있으면 끊어라.

---

#### 3. 한국 마케터가 반응하는 언어 패턴

**통증 언어 (Pain Language) — 인하우스 마케터 현실:**
- "매달 리포트 쓰는 게 제일 지쳐요"
- "광고 성과 왜 이러나 싶을 때"
- "팀장한테 어떻게 설명해야 하지 싶은 순간"
- "대행사 보고서 봐도 뭐가 뭔지 모를 때"
- "혼자 다 해야 하는 인하우스의 현실"

**욕망 언어 (Desire Language):**
- "이거 하나로 끝내고 싶다"
- "칼퇴하고 싶다"
- "데이터로 팀장 설득하고 싶다"
- "대행사 없이도 혼자 다 돌리고 싶다"

---

#### 4. 신뢰 구축 방식 — 한국 특유

```
✅ 잘 먹히는 사회적 증거 표현:
"인하우스 마케터 200명이 현재 사용 중"
"인스타 마케터 커뮤니티에서 입소문 났어요"
"마케터들이 직접 만든 툴이에요"

✅ 공감으로 신뢰 쌓기:
"저도 인하우스 마케터 출신이라서요"
"이 불편함 저도 겪어봤거든요"
"솔직히 완벽한 툴은 없어요. 대신 이런 부분은 확실해요."
```

**한국에서 통하는 B급 감성 훅 (MZ 타겟):**
> "마케터인데 엑셀 못하면 안 되는 거 진짜 맞음?"
> "광고비 태울 때 진짜 심장 쫄리던 경험"
> "이거 나만 힘들었던 거 아니었구나"

### 🚫 카피의 흔한 실수

#### 실수 1 — 자기 이야기로 시작 (가장 흔함)

```
❌ "안녕하세요. 저는 Lian Dash 마케터 예진입니다.
저희 회사는 2024년에 설립되었고..."

✅ 교정: 상대방 이야기로 시작.
"혜경님, 광고 채널 몇 개 운영하고 계세요?"
```

---

#### 실수 2 — 혜택 대신 기능 나열

```
❌ "구글 애널리틱스 연동, 메타 광고 API 연결,
실시간 대시보드, CSV 내보내기 기능 제공"

✅ 교정: 기능이 가져다주는 결과로 전환.
"광고 데이터를 자동으로 모아서, 5분 만에 리포트 완성."
→ 시간 절약 = 혜택. API 연동 = 기능.
```

**공식:** [기능] → "그래서 당신은 [혜택]을 얻는다"

---

#### 실수 3 — 너무 길다

**벤치마크:**
- 콜드 DM: 3~5문장 이하
- 콜드 이메일: 150~250자 이하
- 광고 본문: 125자 이하 (모바일 더 보기 전)
- 랜딩페이지 Hero 헤드라인: 10단어 이하

```
❌ 길고 지루한 DM:
"안녕하세요 혜경님, 저는 Lian Dash에서 일하는 예진이라고 합니다.
저희 Lian Dash는 마케터를 위한 데이터 분석 솔루션으로..."

✅ 짧고 강한 DM:
"혜경님, 광고 리포트 매주 직접 만드세요?
Lian Dash 쓰면 자동으로 됩니다. 2주 무료예요."
```

#### 실수 4 — CTA 없음 또는 불명확

```
❌ "언제 한번 연락 주세요."
❌ "관심 있으시면 알려주세요."

✅ 구체적인 다음 행동 하나:
"이번 주 15분 통화 가능하신 시간 있으세요?"
"아래 링크에서 2주 무료 시작하실 수 있어요: [링크]"
```

#### 실수 5 — 모든 사람을 타겟으로

```
❌ "마케팅 담당자라면 누구나"
✅ "구글+메타 동시에 돌리는 인하우스 마케터"

타겟이 구체적일수록 그 사람은 '나 이야기네'라고 느낀다.
범위를 좁히면 전환율이 올라간다.
```

#### 무엇이 효과 없는지 빠르게 판단하는 법

**7일 룰:** 콜드 이메일/DM 100개 보내서 7일 내 답장이 5개 미만이면 카피 문제.

**3-변수 A/B 테스트 우선순위:**
1. **훅(첫 줄)** — 가장 큰 영향. 여기서 먼저 테스트.
2. **CTA 문구** — "무료 시작" vs "2주 써보기" vs "15분 통화"
3. **제목줄** — 이메일 오픈율에 직접 영향.

**나쁜 카피의 징후:**
- 오픈은 되는데 답장이 없다 → 본문/CTA 문제
- 오픈도 안 된다 → 제목줄 문제
- 클릭은 하는데 가입 안 한다 → 랜딩페이지 카피 문제
- 반응이 "네 알겠습니다"로만 온다 → 너무 영업처럼 느껴짐 → 톤 교정 필요
