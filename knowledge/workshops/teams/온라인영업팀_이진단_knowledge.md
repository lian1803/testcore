### 네이버 플레이스 최적화 진단 체크리스트 2024 (점수화 기준 포함)

**네이버 플레이스 상위 노출은 기본정보 완성도 90%+, 사진 20장+, 리뷰 50개+/평점 4.5+, 키워드 밀도 5%+, 영업시간 100% 정확, 메뉴/가격 80% 등록으로 점수 85점 이상 목표.** 이 체크리스트는 검색 결과[1][2][4][5][8]에서 추출한 실전 항목을 **완성도(%) + 사진수 + 리뷰수/평점 + 키워드 + 영업시간 + 메뉴/가격**으로 분류, 각 항목별 **점수화 기준(0-100점)** 적용. 총점 = 합산/6, 80점+ 시 1페이지 노출 사례 80%[1][4].

#### 2. 사진 수 (가중치 15%, 목표 20장+)
- **최소 20장 (상품/매장/메뉴 각 5장+, 최근 6개월 내 70%+)**: 클릭률 2배↑, 첫인상 핵심[1][5].
- **점수화**: 0-10장=20점, 11-20장=60점, 21장+=100점. 흐름 담긴 사진(입구→인테리어→상품) 우선[1].
- **실전 사례**: 상위 매장 평균 30장, 체류시간 +20%[1][3].

#### 3. 리뷰 수 & 평점 (가중치 30%, 목표 50개+/4.5점)
- **리뷰 수**: 영수증/포토리뷰 50개+ (이벤트 유도: 저장+리뷰 시 쿠폰)[2][4][5].
- **평점**: 4.5+ (신뢰도+최신+답글 반영 알고리즘)[5].
- **점수화**:
  | 리뷰수 | 평점 | 점수 |
  |--------|------|------|
  | <20 | <4.0 | 20 |
  | 20-50 | 4.0-4.5 | 70 |
  | 50+ | 4.5+ (답글 80%+) | 100 |
- **실전**: 리뷰 이벤트+답글로 2주 1페이지 도달 사례[4], 목표치 달성 시 순위 +30%[2].

#### 4. 키워드 최적화 (가중치 15%, 목표 밀도 5%+)

- **평가 수치**: 인사이트 '프로필 방문' 10%↑ 시 10점. 누락 시 -2점씩 차감.

- **평가 수치**: 하이라이트 조회 20%↑ 프로필 체류 시 10점. 없음 0점.
- **개선 팁**: 방문자 1차 랜딩 페이지 역할 → '자주 묻는 질문' 등록으로 이탈 -15%.[2]

- **개선 팁**: 비팔로워 도달 40% 목표 → 추천 알고리즘 우선.[5]

# 소상공인 블로그 마케팅 진단 기준

## 1. 네이버 블로그 검색 노출 여부 진단

**검색 노출의 핵심은 "최적화"된 콘텐츠**이다[2]. 단순히 글을 많이 올리는 것이 아니라 네이버가 신뢰할 수 있고 꾸준히 관리된 블로그를 우선 노출한다[2].

진단 기준:
- 블로그가 검색창에 노출되는가
- 업종별 주요 키워드로 상위 노출되는가
- 포스팅 후 검색반영 시간(일반적으로 24~72시간)

## 2. 포스팅 품질 평가 지표

효과적인 포스팅 기준[2]:

| 요소 | 세부 사항 |
|------|---------|
| **시각 자료** | 고화질 사진 4~8장 이상 첨부(제품/매장/후기 등) |
| **콘텐츠 구조** | 소개-메뉴-후기-오시는 길(간단하고 명확한 구성) |
| **실제 경험** | 직접 경험담이나 이용 후기 포함 |
| **트렌드 반영** | 계절별 신메뉴, 이벤트 등 최신 정보 |
| **커뮤니케이션** | 댓글 답변, 이웃과 상호작용 |
