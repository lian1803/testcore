### B2B 세일즈 미팅 30분 대본 구조 Best Practice

**권장 구조: 아이스브레이킹(3분) → 상황 질문(5분) → 문제 공감(5분) → 솔루션 제시(7분) → 가격 제안(5분) → 클로징(5분).** 이 구조는 BANT(Budget, Authority, Needs, Timeline) 프레임워크를 기반으로 고객 상황 파악 → 니즈 공감 → 가치 제시 → 의사결정 촉진 순으로 설계되어 전환율 20-30% 향상 사례 확인[2][3].

#### 1. 아이스브레이킹 (0:00-3:00, 3분)
**목적:** 신뢰 구축, 긴장 완화. 공통점 찾기 + 미팅 agenda 공유.  
**스크립트 사례 (SaaS 영업 예시):**  
"안녕하세요, [고객 이름]님. [공통 연결고리, e.g., '지난 LinkedIn에서 [회사] 성장 소식 봤어요'] 반갑습니다. 저는 [회사] [이름]입니다. 오늘 30분 동안 [고객 회사]의 [주요 도전, e.g., 리드 관리] 효율화 방안을 중점으로 논의하고, 다음 액션까지 정리하겠습니다. 괜찮으신가요?"[2]  
**팁:** 미팅 context(참석자, 목적) 즉시 기록. BANT 초기 탐색 시작[3].

#### 2. 상황 질문 (3:00-8:00, 5분)
**목적:** 고객 현재 상태 파악 (BANT: Budget/Authority/Needs 초기). 수치화 질문으로 데이터 수집.  
**스크립트 사례:**  
"먼저 [고객 회사]의 현재 [도메인, e.g., 리드 수집] 프로세스를 알려주세요.  
- 연간 리드 규모는? (인바운드:아웃바운드 비율은?)  
- 주요 경쟁사는? (e.g., A사, B사)  
- 팀 규모와 예산 범위는 어느 정도인가요? (상반기 도입 가능 여부)"[2]  
**예시 기록:** "리드: 인바운드 3:아웃바운드 7, 예산 상반기 남음"[2].  
**팁:** 고객 비즈니스 모델/이슈 수집. 80% 듣기, 20% 질문[3].

#### 3. 문제 공감 (8:00-13:00, 5분)
**목적:** 니즈 깊이 파악, 수치화 문제 정의. "왜 필요한가?" 재확인.  
**스크립트 사례:**  
"[상황 요약, e.g., '리드 70% 아웃바운드라 응답률 낮으시네요']. 구체적으로 어떤 문제가 가장 크신가요?  
- 현재 기능 부족으로 손실되는 매출은? (e.g., 응답률 20% 미만 → 기회 상실 30%)  
- '자동화 필요'라면, 어떤 부분이 가장 막히나요? (e.g., 이메일 발송 지연으로 전환율 15%↓)"[2][3]  
**예시 기록:** "문제: 지원자 응답률 2배 부족, 추가 개발 니즈"[1][2].  
**팁:** 고객 미인지 문제 재정의. "그게 ~라면 ROI 2배 가능" 공감[6].

#### 4. 솔루션 제시 (13:00-20:00, 7분)
**목적:** 맞춤 JTBD(Jobs-to-be-Done) 매칭. 결과 중심 데모.  
**스크립트 사례:**  
"그 문제를 [솔루션]으로 해결한 사례입니다. [고객 시나리오]에 맞춰:  
- 템플릿 파이프라인 + 자동 이메일: 응답률 2배 (10인 이하 팀 사례).  
- [데모 3분]: 여기서 [고객 데이터] 입력 시 실시간 결과 보이시죠?  
희망 기능 맞나요?"[1][6]  
**팁:** 헤드라인식: "결과(2배↑) + 방법(템플릿) + 타겟(10인 팀)"[1]. 블로커 즉시 대응[2].

#### 5. 가격 제

### 소상공인 마케팅 대행 영업 거절 유형 TOP 6 및 대응법 - '비싸요' '효과 있어요?' '혼자 할 수 있어요' '생각해볼게요' '다른 데 알아볼게요' '지금은 아니에요' 각 거절에 대한 심리학 기반 리프레이밍 화법
### 소상공인 마케팅 대행 영업 거절 TOP 6 및 심리학 기반 리프레이밍 대응

소상공인 마케팅 대행 영업에서 흔한 **거절 유형 TOP 6**은 비용·효과·자체 가능성에 대한 우려에서 비롯되며, **인지 부조화(심리학: 행동과 믿음 불일치 해소 욕구)**와 **손실 회피 편향(Prospect Theory: 손실을 이득보다 2배 크게 느낌)**을 기반으로 리프레이밍. 각 대응은 **증거 제시 + 가치 전환 + 구체 수치/사례**로 즉시 적용 가능.[1][2]

#### 1. **"비싸요"** (비용 우려: 손실 회피 편향)
   - **리프레이밍 화법**: "초기 비용이 부담스럽게 느껴지실 수 있어요. 하지만 **매출 20-30% 상승 사례**처럼 3개월 내 ROI 3배 달성 시 순이익 500만 원 초과예요. **무료 1주 테스트**로 10% 트래픽 증가 확인 후 결정 어때요?" 
   - **근거/프레임워크**: 소상공인 사례에서 초기 투자 회수율 300% 달성(평균 광고비 100만 원 → 매출 300만 원).[1] 손실 회피 극복: 작은 승리 먼저 제공.

#### 2. **"효과 있어요?"** (불확실성: 증거 부족 인지 부조화)
   - **리프레이밍 화법**: "의심되시는 건 당연해요. **유사 업종(뷰티숍) 15곳에서 키워드 상위노출로 방문자 40%↑, 매출 25% 증가** 데이터 보여드릴게요. **당신 가게 맞춤 KPI(예: 주문 20%↑)**로 1개월 보장 계약 해볼까요?"
   - **근거/프레임워크**: 공정위 사례처럼 과도 약속 피함. 실제 뷰티 업종 마케팅 성공률 70% (트래픽→전환).[3] 신뢰 구축: quantifiable 증거 우선.

#### 3. **"혼자 할 수 있어요"** (자립 편향: 노력 과대평가)
   - **리프레이밍 화법**: "혼자 하시려는 열정 멋져요. 하지만 **전문 대행 시 시간 80% 절감, 전문 툴(SEO/ADS)로 효과 2.5배**. 당신은 영업 집중, 우리는 마케팅 맡기세요. **최초 50만 원 규모**로 비교 테스트 해보는 건?"
   - **근거/프레임워크**: 소상공인 60%가 자체 마케팅 실패(시간 부족).[1][2] 시간 가치화: 전문가 ROI 프레임 (자체 1배 vs 대행 2.5배).

#### 4. **"생각해볼게요"** (결정 지연: 지위 쿼드 미스(Status Quo Bias))
   - **리프레이밍 화법**: "생각 시간 필요하시죠. 그 사이 **경쟁사 3곳 이미 15% 매출↑ 중**이에요. **24시간 내 결정 시 20% 할인 + 무료 감사 보고서** 드릴게요. 어떤 부분 더 궁금하세요?"
   - **근거/프레임워크**: 지위 쿼드 극복 위해 FOMO(Fear Of Missing Out) + urgency. 소상공인 지연 시 기회 상실 40%.[2] 즉시 가치 추가.

#### 5. **"다른 데 알아볼게요"** (비교 쇼핑: 선택 과부하)
   - **리프레이밍 화법**: "좋은 선택이에요. 저희는 **사기 업체와 달리 공정위 등록 + 환불 보장**, **유사 케이스 85% 재계약률**. **3곳 견적 비교표** 만들어드릴게요 – 우리만의 차별점(ROI 4배) 확인하세요."
   - **근거/프레임워크**: 사기 업체 피해 26명 단체소송 사례처럼 신뢰 강조.[1][2] 비교 우위: 데이터 테이

### SPIN selling 기법 한국어 적용 - Situation/Problem/Implication/Need-payoff 질문법을 소상공인 마케팅 대행 미팅에 적용한 구체적 질문 리스트
### SPIN Selling 한국어 적용: 소상공인 마케팅 대행 미팅 질문 리스트

SPIN Selling(상황/문제/함의/니즈-페이오프 질문)은 소상공인(예: 로컬 카페, 음식점, 소규모 상점) 마케팅 대행 미팅에서 고객의 **숨겨진 pain point**를 파헤쳐 **맞춤 솔루션**을 제안하는 최고 효율 기법. 실전 적용 시 **S 3-5개 → P 4-6개 → I 3-5개 → N 2-4개** 순으로 15-20분 내 질문, 80% 듣기 위주. 한국 소상공인 특성(매출 불안정, 디지털 미숙, 지역 경쟁 심화) 반영해 **구체적 질문 20개** 압축.
