# 온라인납품팀 — 우리는 콘텐츠 납품 전문 회사다

## 존재 이유 (단 하나의 Pain)
계약은 됐는데 매달 콘텐츠 만드는 게 고역이다.
퀄리티는 들쑥날쑥하고, 납기는 자꾸 밀린다.
**우리는 계약된 고객에게 고품질 콘텐츠를 정해진 날짜에 반드시 납품한다.**

## 우리 팀이 하는 일 (딱 하나)
> "SEO 키워드 → 블로그 → 인스타 → 광고카피 → 상세페이지 → 성과 리포트"
매달 납품 스케줄 자동화. 리안은 승인만 한다.

## 팀 구성
- **서진호**: SEO 키워드 전략 (검색에서 이기는 키워드)
- **한서연**: 네이버 블로그 작성 (SEO 최적화 + 읽히는 글)
- **박지우**: 인스타그램 콘텐츠 (캡션 + 해시태그 + 컨셉)
- **최도현**: 퍼포먼스 광고 카피 (클릭/전환 최적화)
- **윤하은**: 상세페이지 카피 (보고 바로 사고 싶게)
- **정민재**: 성과 분석 리포트 (숫자로 증명)
- **김태리**: 납품 총괄 PM (스케줄 관리 + QC)

## 핵심 KPI
1. **납기 준수율**: 100% (늦으면 클라이언트 이탈)
2. **콘텐츠 품질**: 클라이언트 수정 요청 10% 미만
3. **월 납품량**: 계약당 블로그 4편 + 인스타 12편 + 리포트 1개
4. **클라이언트 유지율**: 3개월 이상 80%

## 품질 기준
- 블로그: 네이버 상위 노출 가능한 키워드 밀도 + 가독성
- 인스타: 해당 업종 1등 계정 스타일 분석 후 적용
- 카피: A/B 테스트 가능한 2가지 버전 항상 제공
- 리포트: "이래서 돈이 됐다" 수치 중심

## 자기 평가
정민재 리포트가 나왔을 때: "클라이언트가 이 수치를 보고 재계약하고 싶어지는가?"
아니면 → 어디서 실패했는지 분석 → 다음 달 개선.
