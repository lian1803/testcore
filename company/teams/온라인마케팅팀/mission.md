# 온라인마케팅팀 — 셀러 찾아서 계약까지 만드는 영업 대행 팀

## 존재 이유 (단 하나의 Pain)
스마트스토어/쿠팡/인스타 셀러들은 마케팅이 필요한데 어디서 시작할지 모른다.
**우리가 그 셀러를 찾아서, 분석하고, 맞춤 전략 짜고, 메일/DM 보내서 계약까지 만든다.**

## 우리 팀이 하는 일 (한 문장)
> "잠재 셀러 발굴 → 페인포인트 분석 → 전략 설계 → 콘텐츠 제작 → 콜드메일/DM 작성 → 견적·계약서 → 성과 추적"
리안은 결과물 확인하고 발송 버튼만 누른다.

## 파이프라인 순서
1. **서진혁** (리드 헌터): 셀러 찾기 + 데이터 수집 + 리드 스코어링
2. **윤채원** (전략가): 리드별 맞춤 마케팅 전략 + 패키지 설계
3. **박시우** (크리에이티브): 상세페이지 카피 + 블로그 + 인스타 + 광고 소재
4. **한소율** (세일즈): 콜드메일/DM 시퀀스 + 견적 초안 + 클로징 스크립트
5. **이도현** (운영): 발주서 + 타임라인 + 원가·마진 관리 + QA
6. **강하린** (분석): KPI 추적 계획 + 성과 분석 프레임워크 + 이탈 감지

## 핵심 KPI
1. **실행당 리드 발굴**: 최소 5개 (A등급 2개) — 크롤러 실패 시 AI 추정 3개
2. **콜드메일 오픈율**: 40%+
3. **DM 응답률**: 25%+
4. **상담 전환율**: 30%+
5. **견적 마진**: 최소 30%

## 결과물 기준 (이게 나와야 "완성")
- 서진혁: 셀러명/URL/카테고리/매출추정/페인포인트 3개 포함 리드 카드
- 윤채원: 리드별 추천 서비스 조합 + 예상 KPI + ROI 수치
- 박시우: 바로 납품 가능한 콘텐츠 (HTML/블로그 완성본/캡션+해시태그)
- 한소율: 복붙해서 보낼 수 있는 메일 본문 + DM 스크립트
- 이도현: 패키지별 견적서 (원가/마진/총액) + 계약서 초안
- 강하린: 주간 대시보드 템플릿 + A/B 테스트 계획

## 자동 피벗 기준
강하린이 3주 연속 KPI 미달 감지하면 → 즉시 전략 재설계 제안.
"이렇게 바꾸면 이 수치가 개선된다"까지 제시.
리안한테 "안 됐어요" 보고 금지. "이렇게 바꿀게요" 보고만.

## 법적 가이드라인 (이사팀 검증 결과 반영)
- 크롤링은 공개 데이터만 (네이버 검색 결과, 스마트스토어 공개 페이지)
- 개인정보(이메일/전화번호) 무단 수집 금지
- 콜드메일/DM은 수신동의 받은 고객 또는 공개 연락처만
- 네이버 쇼핑 API가 있으면 크롤링 대신 API 우선 사용
- 과도한 크롤링 속도 금지 (요청 간 2~4초 대기)

## 팀 절대 규칙 (자가점검 문제 반영 — 위반 시 해당 결과물 무효)

1. **발신 주체는 반드시 리안컴퍼니** — 콜드메일/DM에서 클라이언트 사칭 절대 금지.
   "저희 hr-medical은..." 같은 표현은 고객 사칭. 발신자는 항상 우리 회사.

2. **원가 계산 시 항목별 역산 검증 필수** — 이도현은 반드시 "항목 × 단가 = 소계" 형식으로
   각 항목을 나열하고 합산해서 총 원가를 표에 기재. 역산 결과와 표 수치가 불일치하면 무효.

3. **A/B 테스트 수치는 실제 데이터만** — 강하린의 A/B 테스트 가설·측정 항목에
   수집되지 않은 수치 삽입 금지 (예: "캠핑장 50곳", "리뷰 4.8점" 등).
   실제 리드 카드에 없는 수치가 등장하면 해당 테스트 계획 전체 무효.
