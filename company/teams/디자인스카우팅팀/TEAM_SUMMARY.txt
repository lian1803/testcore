================================================================================

1. 박어워즈.py — Playwright Web Scraper
   └─ Captures screenshots from 4 design sites
   └─ Falls back to BeautifulSoup if Playwright unavailable
   └─ Saves to: teams/디자인스카우팅팀/screenshots/

2. 서트렌드.py — Perplexity Trend Researcher
   └─ Searches 3 trend queries via Perplexity API
   └─ Returns structured design trends
   └─ Saves to: teams/디자인스카우팅팀/trends_results.json

3. 김레포.py — Design System Repository Scanner
   └─ Scans design_system for available effects
   └─ Tracks which effects have been used recently
   └─ Recommends top 3 underused effects
   └─ Saves to: teams/디자인스카우팅팀/repo_results.json

4. 디자인정보.py — Team Leader & Report Generator
   └─ Orchestrates all 3 scouts
   └─ Generates polished briefing with Claude Haiku
   └─ Saves locally: teams/디자인스카우팅팀/briefing_YYYY-MM-DD.md
   └─ Appends to: 보고사항들.md (shared report file)
